p5の
<a href="/reference/p5/WEBGL" target="_blank">WEBGL</a>
モードには7つの基本形状が含まれています。それらの形状は、平面、ボックス、
シリンダー、円錐、トーラス、球、そして楕円体です。さらに、
<a href="/reference/p5/model" target="_blank">model()</a>
は、<a href="/reference/p5/loadModel" target="_blank">loadModel()</a>を介して読み込まれたカスタムジオメトリを表示します。
この例には、各基本形状が含まれています。また、<a href="https://nasa3d.arc.nasa.gov/models" target="_blank">NASAのコレクション</a>からのモデルも含まれています。
