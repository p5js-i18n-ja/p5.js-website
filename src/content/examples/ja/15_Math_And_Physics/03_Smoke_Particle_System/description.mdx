煙のパーティクルシステムデモ、Dan Shiffmanの元の
<a href="https://natureofcode.com/book/chapter-4-particle-systems/">例</a>
に基づいています。

このコードは
<a href="/reference/p5/p5.Vector">p5.Vector</a>
クラスを使用しており、 
<a href="/reference/p5/createVector">createVector()</a>
関数を含んでいます。パーティクルの位置と速度を更新するためのさまざまな計算は、p5.Vectorメソッドを使用して行われます。

パーティクルシステムは
<a href="/reference/p5/class">クラス</a>として実装されており、オブジェクトの配列（クラスParticleのインスタンス）を含んでいます。
