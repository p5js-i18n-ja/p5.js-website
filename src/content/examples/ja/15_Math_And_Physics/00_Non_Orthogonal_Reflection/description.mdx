この例は、傾斜面で跳ねるボールを示しており、
反射のためのベクトル計算を使用して実装されています。

このコードは、 
<a href="/reference/p5/p5.Vector">p5.Vector</a>
クラスを広範囲に使用しており、 
<a href="/reference/p5/createVector">createVector()</a> 関数を使用して新しいベクトルを作成し、
ベクトル計算のためのベクトルメソッド
<a href="/reference/p5.Vector/add">add()</a>
および
<a href="/reference/p5.Vector/dot">dot()</a>
を使用しています。
