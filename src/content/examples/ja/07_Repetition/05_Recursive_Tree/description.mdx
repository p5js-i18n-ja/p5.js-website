これは、再帰を通じてシンプルな木のような構造を描画する例です。
分岐角度は、水平なマウス位置に基づいて計算されます。マウスを左右に動かして角度を変更してください。
これは、Processingの<a href="https://processing.org/examples/tree.html">Recursive Tree Example</a>に基づいています。
