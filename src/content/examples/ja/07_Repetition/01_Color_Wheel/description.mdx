この例は、異なる色相の外観を示しています。これは
<a href="/reference/p5/for" target="_blank">forループ</a>
を使用して変換を繰り返します。ループは
angleという変数を初期化し、円の回転とその色相を変更します。ループが繰り返されるたびに、キャンバスの中心に対して円が描画されます。 
<a href="/reference/p5/push" target="_blank">push()</a>
および <a href="/reference/p5/pop" target="_blank">pop()</a>
関数は、これらの変換が個々の円にのみ影響を与えるようにします。
