<a href="https://developer.mozilla.org/en-US/docs/Glossary/Interpolation" target="_blank">補間</a>
は、2つの他の値の間の値を計算します。例えば、数字5は
0と10の間の中間点です。異なるタイプの補間は
値の間の変化率が異なります。線形補間は、
lerpと略され、一定の変化率を使用します。この
<a href="/reference/p5/lerp" target="_blank">lerp()</a>
関数は2つの数値の間を線形に補間します。この
<a href="/reference/p5/lerpColor" target="_blank">lerpColor()</a>
関数は、ここで示されているように、2つの色の間を線形に補間します。
この例では、stripeCount変数が表示される横縞の数を調整します。
値を高く設定すると、個々の縞のようには見えず、グラデーションのように見えます。
