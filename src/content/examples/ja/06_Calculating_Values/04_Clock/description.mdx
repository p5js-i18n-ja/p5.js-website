現在の時間は、<a href="/reference/p5/second" target="_blank">second()</a>、<a href="/reference/p5/minute" target="_blank">minute()</a>、および <a href="/reference/p5/hour" target="_blank">hour()</a> 関数を使用して取得できます。この例では、<a href="/reference/p5/map" target="_blank">map()</a> を使用して針の角度を計算し、その後 <a href="/reference/#Transform" target="_blank">変換</a> を使用して位置を設定しています。
