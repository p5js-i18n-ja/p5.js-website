---

<a href="https://developer.mozilla.org/en-US/docs/Glossary/Interpolation" target="_blank">補間</a>は、2つの他の値の間の値を計算します。例えば、数値5は0と10の間の中間点です。異なるタイプの補間は、値の間の変化率が異なります。線形補間は、略してlerpと呼ばれ、一定の変化率を使用します。<a href="/reference/p5/lerp" target="_blank">lerp()</a>関数は、2つの数値の間で線形補間を行います。

マウスを画面上で動かすと、シンボルがそれに従います。アニメーションの各フレームを描画する間に、楕円は現在の位置からカーソルに向かって部分的に移動します。
