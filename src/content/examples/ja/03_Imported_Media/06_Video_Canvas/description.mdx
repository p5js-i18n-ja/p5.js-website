<a href="/reference/p5/createVideo" target="_blank">createVideo()</a>
および <a href="/reference/p5/image" target="_blank">image()</a> 
関数を使用して、キャンバスに動画をアップロードできます。動画キャプチャは 
<a href="/reference/p5/image" target="_blank">image()</a> 
コンストラクタを通じて渡されるため、 
<a href="/reference/p5/filter" target="_blank">filter()</a> 
メソッドを使用して動画キャプチャにフィルターを追加できます。この例をローカルで実行するには、実行中の 
<a href="https://github.com/processing/p5.js/wiki/Local-server">ローカルサーバー</a>が必要です。
キャンバス内に埋め込まずに動画を作成するには、 
<a href="/examples/imported-media-video/">動画</a>の例を参照してください。
