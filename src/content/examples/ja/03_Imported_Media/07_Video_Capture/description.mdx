title: ビデオキャプチャ
oneLineDescription: カメラからのライブビデオフィードを表示します。
---

<a href="/reference/p5/createCapture" target="_blank">createCapture()</a>
および <a href="/reference/p5/image" target="_blank">image()</a> 
関数を使用すると、デバイスのビデオキャプチャを取得し、キャンバスに描画できます。 
ビデオキャプチャは <a href="/reference/p5/image" target="_blank">image()</a> 
コンストラクタを通じて渡されるため、 
<a href="/reference/p5/filter" target="_blank">filter()</a> 
メソッドを使用してビデオキャプチャにフィルターを追加できます。 
ビデオのアップロード、表示、スタイリングの異なる戦略については、 
<a href="/examples/imported-media-video/" target="_blank">ビデオ</a> および 
<a href="/examples/imported-media-video-canvas/" target="_blank">ビデオキャンバス</a> 
の例を訪れてください。
