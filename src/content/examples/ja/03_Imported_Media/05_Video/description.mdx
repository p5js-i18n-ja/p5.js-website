<a href="/reference/p5/noCanvas" target="_blank">noCanvas()</a>
および <a href="/reference/p5/createVideo" target="_blank">createVideo()</a> 
関数を使用することで、キャンバス内にビデオを埋め込むことなく、 
<a href="https://developer.mozilla.org/en-US/docs/Glossary/DOM" target="_blank">DOM</a> 
にビデオをアップロードできます。キャンバス要素内に埋め込まれたビデオを作成するには、
<a href="/examples/imported-media-video-canvas/" target="_blank">ビデオキャンバス</a>の例を参照してください。
