oneLineDescription: キャンバス上で画像を半透明にします。
---

このプログラムは、画像のアルファ値を変更することによって
一つの画像を別の画像の上に重ねます。
<a href="/reference/p5/tint" target="_blank">tint()</a>
関数を使用します。キャンバス上でカーソルを左右に動かすことで
画像の位置を変更できます。この例をローカルで実行するには、
画像ファイルと稼働中の
<a href="https://github.com/processing/p5.js/wiki/Local-server">ローカルサーバー</a>が必要です。
