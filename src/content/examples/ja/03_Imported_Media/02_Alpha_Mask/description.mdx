<a href="/reference/p5/mask" target="_blank">mask()</a> メソッドを使用すると、画像のマスクを作成して、画像の異なる部分の透明度を指定できます。この例をローカルで実行するには、2つの画像ファイルと実行中の 
<a href="https://github.com/processing/p5.js/wiki/Local-server">ローカルサーバー</a>が必要です。
