<a href="/reference/p5/createAudio" target="_blank">createAudio()</a>
はオーディオプレーヤーを作成します。この例では、プレーヤーのコントロールを表示し、
その再生速度を調整します。マウスがウィンドウの左端にあるときは再生速度が通常で、
マウスが右に移動するにつれて速くなります。オーディオプレーヤーなどのメディア要素の使用に関する詳細は、
<a href="/reference/p5/p5.MediaElement" target="_blank">p5.MediaElement</a>
リファレンスページにあります。オーディオファイルは
<a href="https://freesound.org/people/josefpres/sounds/711156/" target="_blank">Josef Presによるパブリックドメインのピアノループ</a>です。
