<a href="/reference/p5/deviceMoved" target="_blank">deviceMoved()</a> 関数は、スケッチを表示しているモバイルデバイスが動いたときに実行されます。
この例では、<a href="/reference/p5/accelerationX" target="_blank">accelerationX</a>、<a href="/reference/p5/accelerationY" target="_blank">accelerationY</a>、および <a href="/reference/p5/accelerationZ" target="_blank">accelerationZ</a> の値が円の位置とサイズを設定します。
これはモバイルデバイスでのみ機能します。
