if文とelse文を使用すると、特定の条件が真であるときにのみ
<a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/block" target="_blank">コードのブロック</a>
を実行できます。この例では、マウスが押されているときのみアニメーションが実行されます。これは、59行目のif文によるものです。if文とelse文についての詳細は
<a href="/reference/p5/if">p5リファレンス</a>
や<a href="https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Building_blocks/conditionals" target="_blank">MDN</a>で読むことができます。

比較演算子は、2つの値を比較することによって条件を形成するのに役立ちます。この例では、円の色相が360以上になると色相がゼロにリセットされます。これは69行目のif文によるものです。比較演算子についての詳細は
<a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators#relational_operators" target="_blank">MDN</a>で読むことができます。

論理演算子を使用すると、条件を組み合わせることができます。
<a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Logical_AND" target="_blank">&amp;&amp;</a>
は、両方の条件が真であることを確認します。この例では、円はキャンバスの水平中心に近いときに黒で塗りつぶされ、そうでないときは白で塗りつぶされます。これは、45行目のif文によるもので、円のx位置が100以上であり、かつ300以下であることを確認します。
<a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Logical_OR" target="_blank">||</a>
は、少なくとも1つの条件が真であることを確認します。円はキャンバスの左端または右端に達すると水平速度が反転します。これは75行目のif文によるものです。
