---

<a href="/reference/p5/scale">scale()</a> 関数は、指定された係数で現在の座標系をスケーリングします。

<a href="/reference/p5/push">push()</a> と <a href="/reference/p5/pop">pop()</a> 関数は、それぞれ座標系を保存および復元します。

この例では、原点にサイズ200の四角形が描画され、3つの異なるスケーリング係数が適用されています。
