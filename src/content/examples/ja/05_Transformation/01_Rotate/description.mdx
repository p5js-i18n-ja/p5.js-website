<a href="/reference/p5/rotate">rotate()</a> 関数は、現在の原点を中心に現在の座標系を回転させます。

デフォルトでは、原点はキャンバスの左上隅です。キャンバスの中心を回転させるためには、まず座標系を移動させ、その後新しい原点を中心に回転させる必要があります。

<a href="/reference/p5/push">push()</a> と <a href="/reference/p5/pop">pop()</a> 関数は、それぞれ座標系を保存および復元します。
