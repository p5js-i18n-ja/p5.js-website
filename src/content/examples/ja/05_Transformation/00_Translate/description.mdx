<a href="/reference/p5/translate">translate()</a> 関数は、座標系の原点を指定された位置に移動します。

<a href="/reference/p5/push">push()</a> と <a href="/reference/p5/pop">pop()</a> 関数は、座標系や塗りつぶしの色などのさまざまな描画設定を保存および復元します。

この例では、各回ごとに異なる座標系で図形（長方形と円）を描画していることに注意してください。
