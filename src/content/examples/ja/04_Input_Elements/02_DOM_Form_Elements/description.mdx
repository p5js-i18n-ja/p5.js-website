<a href="https://developer.mozilla.org/en-US/docs/Web/API/Document_Object_Model" target="_blank">ドキュメントオブジェクトモデル</a>（DOM）は、ウェブページの結果としての構造を表します。p5.jsのフォーム要素を使用すると、<a href="/reference/p5/createInput" target="_blank">createInput()</a>、<a href="/reference/p5/createSelect" target="_blank">createSelect()</a>、および <a href="/reference/p5/createRadio" target="_blank">createRadio()</a> のように、<a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/select" target="_blank">セレクト</a>、<a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input" target="_blank">入力</a>、または <a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/radio" target="_blank">ラジオボタン</a>を介して送信された情報を取得し、その情報に基づいてDOMを更新するさまざまな方法を構築できます。
