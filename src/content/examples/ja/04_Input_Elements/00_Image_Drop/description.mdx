<a href="/reference/p5.Element/drop" target="_blank">drop()</a>
は、ファイルが要素に読み込まれるたびにコールバックを登録するp5.jsの要素メソッドです。
アップロードされたファイルは
<a href="/reference/p5/p5.File" target="_blank">p5.File</a> 
クラスに変換されます。 
<a href="/reference/p5.Element/drop" target="_blank">drop()</a> 
コールバックを使用してファイルタイプを確認し、その後ファイルタイプに応じた条件文を書くことができます。
