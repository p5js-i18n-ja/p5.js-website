<a href="/reference/p5/createGraphics" target="_blank">createGraphics()</a>
関数を使用して新しいp5.Graphicsオブジェクトを作成できます。これはキャンバス内のオフスクリーングラフィックスバッファとして機能します。オフスクリーンバッファは、現在の表示面とは異なる寸法や特性を持つことができますが、同じ空間に存在しているように見えます。
