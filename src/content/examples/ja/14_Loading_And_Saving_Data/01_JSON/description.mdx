<a href="https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Objects/JSON" target="_blank">JavaScriptオブジェクト表記、またはJSON</a>は、
ファイルにデータを書くためのフォーマットです。構文は
JavaScriptから来ていますが、JSONは他の多くの文脈でも使用されます。この例は
Daniel ShiffmanのProcessing用に書かれたLoading JSON Dataの例に基づいています。
バブルのデータを整理するために<a href="/reference/p5/class" target="_blank">クラス</a>を使用しています。スケッチが開始されると、JSONファイルから2つのバブルのデータを読み込みます。訪問者は新しいバブルを追加し、更新されたJSONファイルをダウンロードし、JSONファイルを読み込むことができます。
