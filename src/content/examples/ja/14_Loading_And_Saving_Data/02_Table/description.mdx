カンマ区切り値、またはCSVは、ファイルにデータを書くためのフォーマットです。p5はこのフォーマットを使用して
<a href="/reference/p5/p5.Table" target="_blank">p5.Table</a>を使って作業できます。
この例は、Daniel Shiffmanの
<a href="https://processing.org/examples/loadsavetable.html" target="_blank">タブularデータの読み込み</a>
のProcessing用の例に基づいています。バブルを表すデータを整理するためにクラスを使用しています。スケッチが開始されると、CSVファイルから4つのバブルのデータを読み込みます。訪問者は新しいバブルを追加し、更新されたCSVファイルをダウンロードし、CSVファイルを読み込むことができます。
