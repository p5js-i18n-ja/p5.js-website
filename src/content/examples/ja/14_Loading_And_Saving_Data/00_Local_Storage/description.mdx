title: ローカルストレージ
oneLineDescription: デバイス上のブラウザからデータを保存します。
---

ブラウザは、ウェブサイトが訪問者のデバイスにデータを保存することを許可します。これを
<a href="https://developer.mozilla.org/en-US/docs/Web/API/Window/localStorage" target="_blank">ローカルストレージ</a>と呼びます。
<a href="/reference/p5/getItem" target="_blank">getItem()</a>、
<a href="/reference/p5/storeItem" target="_blank">storeItem()</a>、
<a href="/reference/p5/clearStorage" target="_blank">clearStorage()</a>、
および<a href="/reference/p5/removeItem" target="_blank">removeItem()</a>
関数がこれを制御します。

この例は、Daniel ShiffmanのLoading JSON DataおよびProcessingで書かれたLoading Tabular Dataの例に触発されています。バブルのデータを整理するために
<a href="/reference/p5/class" target="_blank">クラス</a>を使用しています。訪問者は新しいバブルを追加でき、そのデータはローカルストレージに保存されます。訪問者がスケッチを再訪問すると、同じバブルが再読み込みされます。
