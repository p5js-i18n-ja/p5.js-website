p5.jsへようこそ！p5.jsは初めてで、インタラクティブなスケッチを作成するための基本を学びたいですか？このチュートリアルに従って、シンプルな[インタラクティブな風景](https://editor.p5js.org/Msqcoding/sketches/SDbVilWaf)を作成しましょう。以下のことを学びます：

- キャンバスのサイズと背景色を変更する
- 図形やテキストを追加し、カスタマイズし、色を付ける
- マウスポインターの位置に反応するようにスケッチに簡単なインタラクティブ性を追加する
- コードにコメントを付ける
- [p5.jsリファレンス](/reference/)ページを使用してさらに学ぶ

## 前提条件

- [環境のセットアップ](../setting-up-your-environment)

始める前に、以下のことができるようになっているはずです：

- [p5.js Web Editor](https://editor.p5js.org/)にログインして新しいプロジェクトを保存する

**または** 

- [VS Code](https://code.visualstudio.com/)（または他のコードエディタ）で新しいp5.jsプロジェクトを作成して保存する

[p5.js Web Editor](https://editor.p5js.org/)または[VS Code](https://code.visualstudio.com/)でプロジェクトを作成して保存するためのステップバイステップガイドは、[環境のセットアップ](../setting-up-your-environment)をご覧ください。


## ステップ1：新しいp5.jsプロジェクトに名前を付けて保存する

- 新しいp5.jsプロジェクトを作成し、名前を付けて保存します。

[p5.js Web Editor](https://editor.p5js.org/)を使用する場合：

- [p5.js Web Editor](https://editor.p5js.org/)にログインします。
- *鉛筆アイコン*をクリックし、表示されるテキストボックスに「Interactive Landscape」と入力してプロジェクトに名前を付けます。
- *ファイル*をクリックし、*保存*を選択します。
- 保存したスケッチのギャラリーに移動して、プロジェクトが保存されたことを確認します：
  - *ファイル*をクリックし、*開く*を選択します。
  - アカウントに保存されたプロジェクトのリストの一番上に最新のスケッチが表示されます。

![p5.js Web Editorのユーザーが新しいプロジェクトに「Interactive Landscape」と名前を付け、保存し、保存されたスケッチのギャラリーで見つけ、プロジェクト名をクリックして開きます。](../images/introduction/rename-sketch.gif)

#### デフォルトのコード

すべてのp5.jsプロジェクトには、p5.jsライブラリと3つのファイルが含まれています：[`index.html`](https://www.classes.cs.uchicago.edu/archive/2021/spring/11111-1/happycoding/p5js/web-dev.html#:~:text=The%20index.,the%20page%20using%20HTML%20tags!)、[style.css](https://happycoding.io/tutorials/p5js/web-dev)、そして`sketch.js`です。sketch.jsファイルにコードを追加してキャンバスに変更を加えます。新しいp5.jsプロジェクトは、sketch.jsファイルに以下のコードで始まります：

```js
function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
}
```


すべてのsketch.jsファイルは、2つの主要な関数で始まります：[`setup()`](/reference/p5/setup)と[`draw()`](/reference/p5/draw)です。 

[関数](/reference/p5/function)は、特定のタスクを実行する一連の指示です。 

sketch.jsのコードが実行されると：

- [`setup()`](/reference/p5/setup)が呼び出され、1回だけ実行されます。これはプロジェクトのデフォルト値を設定するのに使用できます。
  - [`setup()`](/reference/p5/setup)の中括弧（`{}`）内で、`createCanvas(400, 400)`はプレビューウィンドウで見ることができる、幅400ピクセル、高さ400ピクセルのHTMLキャンバスを作成します。
- [`draw()`](/reference/p5/draw)は[`setup()`](/reference/p5/setup)の直後に呼び出され、プログラムが停止するか[`noLoop()`](/reference/p5/noLoop)関数が呼び出されるまで、その中括弧内のコード行を1秒間に60回実行します。
  - [`draw()`](/reference/p5/draw)の中括弧内（[`draw()`](/reference/p5/draw)のコードブロックとも呼ばれます）では、[`background()`](/reference/p5/background)関数がキャンバスの背景色を設定します。デフォルト値（220）はキャンバスの色をグレーに設定します。

[`setup()`](/reference/p5/setup)、[`draw()`](/reference/p5/draw)、[`createCanvas()`](/reference/p5/createCanvas)、[`background()`](/reference/p5/background)についてさらに学ぶには、[p5.jsリファレンス](/reference/)ページをご覧ください。

## ステップ2：キャンバスのサイズを変更する

- [`createCanvas()`](/reference/p5/createCanvas)の*引数*を変更して、キャンバスの幅を600ピクセル、高さを400ピクセルに変更します。*再生*ボタンを押してプレビューウィンドウの変更を確認します。
- *再生*ボタンを押すか、*自動更新*ボックスにチェックを入れてキャンバスを更新することを忘れないでください。

コードは以下のようになるはずです：

```js
function setup() {
  createCanvas(600, 400);
}

function draw() {
  background(220);
}
```


<Callout title="ヒント">
[p5.js Web Editor](https://editor.p5js.org/)で*再生*をクリックし、"Auto-refresh"の横にあるボックスにチェックを入れると、プロジェクトにコードを追加するたびにキャンバスが継続的に更新されます。このボックスにチェックを入れると、スケッチに変更を加えるたびに*再生*ボタンを押す必要がなくなります。
</Callout>

## ステップ3：コメントを使用してコードを説明する

- `createCanvas(600, 400)`が何をするのかを説明するコメントを書きます。

コードは以下のようになります：

```js
function setup() {
  // 幅600ピクセル、高さ400ピクセルの
  // キャンバスを作成します。
  createCanvas(600, 400);
}

function draw() {
  background(220);
}
```

### コメント

スケッチで*コメント*を使用すると、各コード行の目的を覚えやすくなります。JavaScriptはコメントを読み取ることができないため、コメントはあなたのコードを探索する他の人々にあなたのアイデアを伝えたり、自分自身にコードが何をしているかを思い出させたりするのに最適な方法です。

- 単一行のコメントを埋め込むには、スケッチのどこでも2つのフォワードスラッシュ（`//コメント`）を使用します。


### ステートメント

sketch.jsファイルはJavaScriptで書かれており、各コード行は*ステートメント*と呼ばれます。各ステートメントはセミコロン（`;`）で終わります。sketch.jsファイルで書く各ステートメントの後にセミコロンを追加するのがベストプラクティスです。関数や`if`文の中括弧の後にセミコロンを含めるべきではありません。

コメントとステートメントについての詳細は、[The Coding Trainのこのビデオ](https://www.youtube.com/watch?v=xJcrPJuem5Q)をご覧ください。


## ステップ4：背景色を変更する

- [`background()`](/reference/p5/background)関数の*引数*を`background(135, 206, 235)`に変更して、キャンバスの背景色を空色に変更します。
- このコード行を説明するコメントを追加します。
- キャンバスを自動的に更新するために*自動更新*ボックスにチェックを入れます。

コードは以下のようになります：

<EditableSketch code={`
function setup() {
  //幅600ピクセル、高さ400ピクセルの
  //キャンバスを作成します
  createCanvas(600, 400);
}

function draw() {
  //空色の背景
  background(135, 206, 235);
}
`} />

[`background()`](/reference/p5/background)や図形などの他のp5.jsコマンドも関数です。[`background()`](/reference/p5/background)関数は特にキャンバスの背景色を変更します。[`background()`](/reference/p5/background)のp5.jsリファレンスページによると、上記のコードで見られる3つの*引数*は、キャンバスに表示される青色に対応する赤、緑、青の値を表しています。

赤、緑、青に対応する数字の組み合わせは1000万以上あり、これらを使用できます。これらは[RGBカラーコード](/reference/p5/color)として知られています。上記のコードでは、Rの値は135、Gの値は206、Bの値は235です。これはキャンバスに表示されている薄い青色に対応しています。

- [`background()`](/reference/p5/background)のR、G、B値の変更がキャンバスの色にどのように影響するかを探るには、[この例](https://editor.p5js.org/Msqcoding/full/3j6stYuo2)をご覧ください。

キャンバスの背景色を変更する他の方法については、[background()](/reference/p5/background)と[color](/reference/p5/color)のp5.jsリファレンスページをご覧ください。

#### ヒント

[Googleのカラーピッカー](https://g.co/kgs/aCdbzD)を使用して、背景を任意の色に設定するためのコードを検索できます。使用したい色を見つけ、RGBとラベル付けされたボックス内の数字をコピーし、[`background()`](/reference/p5/background)に貼り付けます。


## ステップ5：キャンバスに図形を描く

- [`circle()`](/reference/p5/circle)関数を使用して、キャンバスの右上隅に太陽を描きます。
- このコード行の最後にセミコロン（`;`）を追加します。
- このコード行を説明するコメントを追加します。

<Callout title="注意">
図形を[`draw()`](/reference/p5/draw)関数の中括弧内（コードブロック内）に追加し、キャンバスを自動的に更新するために*自動更新*ボックスにチェックを入れることを忘れないでください。
</Callout>

コードは以下のようになるはずです：

<EditableSketch code={`
function setup() {
  //幅600ピクセル、高さ400ピクセルの
  //キャンバスを作成します
  createCanvas(600, 400);
}

function draw() {
  //空色の背景
  background(135, 206, 235);
  //右上隅の太陽
  circle(550, 50, 100);
}
`} />

[`circle()`](/reference/p5/circle)などの一部の図形関数は、円の水平位置（x座標）、垂直位置（y座標）、サイズ（幅と高さ）を記述する数字を使用します。xとy座標は円の中心点（`x`, `y`）を示します（以下の図を参照）。

- 上記のコード例で使用されている[`circle()`](/reference/p5/circle)関数の最初の*引数*である数字550は、中心点のx座標です。これは、中心点がキャンバスの左端から550ピクセル右に位置していることを意味します。
- 2番目の引数である数字50は、中心点のy座標です。これは、中心点がキャンバスの上端から50ピクセル下に位置していることを意味します。
- 3番目の引数である数字100は、円のサイズ（幅と高さ）です。これは、円の幅が50ピクセル、高さが50ピクセルであることを意味します。

![中心点が(x,y)にある円と、直径が"size"の図。円関数の使用構文が図の上に表示されています：「circle(x, y, size);」](../images/introduction/circle-diagram.png)

詳細については、[`circle()`](/reference/p5/circle)のp5.jsリファレンスページをご覧ください。

### p5.jsキャンバス座標

キャンバスには、左上隅の水平位置0、垂直位置0から始まる目に見えない座標系が付属しています。

![HTML &lt;canvas&gt;要素の図には、グリッドパターンのオーバーレイが表示されており、水平軸がx座標を、垂直軸がy座標を表し、グリッドの各交点が特定の(x,y)座標ペアを表しています。x軸に沿った矢印は、x座標がキャンバス上で右に移動するにつれて増加することを示しています。y軸に沿った矢印は、y座標がキャンバス上で下に移動するにつれて増加することを示しています。](../images/introduction/coordinates.png)

キャンバス上のオブジェクトがこのポイントから右に移動すると、その水平位置が増加します。キャンバス上のオブジェクトがこのポイントから下に移動すると、その垂直位置が増加します。オブジェクトがキャンバス上で左に移動すると、そのx座標は減少します。オブジェクトがキャンバス上で上に移動すると、そのy座標は減少します。以下の例は、円の位置がキャンバス上で変化するときのx座標またはy座標の値を表示しています：

- [この例](https://editor.p5js.org/Msqcoding/full/AM5ZwrmNo)は、円がキャンバス上で水平に移動するときのx座標を表示します。
- [この例](https://editor.p5js.org/Msqcoding/full/jZeTUjZfZ)は、円がキャンバス上で垂直に移動するときのy座標を表示します。

キャンバスの水平位置と垂直位置の最大値は、[`createCanvas()`](/reference/p5/createCanvas)関数によって設定されます。[`createCanvas()`](/reference/p5/createCanvas)の最初の数字は右端のx座標で、2番目の数字は下端のy座標です。

HTMLキャンバス座標系と図形についてさらに学ぶには、[このp5.jsリファレンスページ](/reference)をご覧ください。

## ステップ6：キャンバス上の図形と輪郭に色を付ける

- [`circle()`](/reference/p5/circle)の上の行に`fill("yellow")`を追加して、太陽に色を付けます。
- [`circle()`](/reference/p5/circle)の上の行に`stroke("orange")`を追加して、太陽の輪郭に色を付けます。
- [`circle()`](/reference/p5/circle)の上の行に`strokeWeight(20)`を追加して、輪郭の太さを変更します。
- コードを説明するコメントを追加します。

<Callout title="注意">
[`fill()`](/reference/p5/fill)、[`stroke()`](/reference/p5/stroke)、[`strokeWeight()`](/reference/p5/strokeWeight)を[`draw()`](/reference/p5/draw)のコードブロック内、図形が描画される行の前に必ず追加してください。
</Callout>

コードは以下のようになるはずです：

<EditableSketch code={`
function setup() {
  // 幅600ピクセル、高さ400ピクセルの
  // キャンバスを作成します。
  createCanvas(600, 400);
}

function draw() {
  //空色の背景
  background(135, 206, 235);
  //右上の太陽
  fill("yellow"); //黄色  

  stroke("orange"); //オレンジの輪郭 

  strokeWeight(20); //太い輪郭    

  circle(550, 50, 100);
}
`} />

[`fill()`](/reference/p5/fill)関数はキャンバス上のあらゆる図形の色を設定し、[`stroke()`](/reference/p5/stroke)関数は線と輪郭の色を設定します。両方とも[`background()`](/reference/p5/background)が使用するのと同じ引数を使用します。上記のコードでは、[`fill()`](/reference/p5/fill)と[`stroke()`](/reference/p5/stroke)の*引数*として[HTML色名](https://www.w3schools.com/tags/ref_colornames.asp)を使用しています。[RGBとHEXカラーコード](https://htmlcolorcodes.com/)も使用できます。[`strokeWeight()`](/reference/p5/strokeWeight)関数は、その下にある線、輪郭、点の太さを設定するために数字を使用します。デフォルトの[`strokeWeight()`](/reference/p5/strokeWeight)は1ピクセルです。上記のコードでは、[`strokeWeight(20)`](/reference/p5/strokeWeight)が円の輪郭を20ピクセルに設定しています。

<Callout title="ヒント">
[p5.js Web Editor](https://editor.p5js.org/)では、色の名前の横にある小さな色付きボックスをクリックすることで、[`fill()`](/reference/p5/fill)、[`stroke()`](/reference/p5/stroke)、[`background()`](/reference/p5/background)で使用する異なる色を検索できます！*引数*として[HTML色名](https://www.w3schools.com/tags/ref_colornames.asp)を入力すると、ボックスが表示されます。色名は[文字列](/reference/p5/String)データ型であるため、引用符（`""`）で囲まれています。

![fill("yellow")を使用して円を黄色に塗った後、p5.js Web Editorでコーディングしているユーザーが、HTML色"yellow"の横に表示される黄色の四角をクリックします。これによりカラーツールが表示され、赤色が選択されます。ユーザーがコードエディタをクリックすると、円の色が黄色から赤に変わります。](../images/introduction/editor-color-picker.gif)
</Callout>

### 塗りつぶしの順序

[`fill()`](/reference/p5/fill)のデフォルト値は白です。これは、[`draw()`](/reference/p5/draw)内に[`fill()`](/reference/p5/fill)が一度も現れない場合、すべての図形がデフォルトで白になることを意味します。キャンバス上の任意の図形の色を変更するには、図形を描画する前に[`fill()`](/reference/p5/fill)を呼び出す必要があります。[`stroke()`](/reference/p5/stroke)のデフォルト値は黒です。輪郭の色と形を変更するには、図形が描画される前に[`stroke()`](/reference/p5/stroke)と[`strokeWeight()`](/reference/p5/strokeWeight)を呼び出す必要があります。キャンバス上に複数の図形を描画する場合、図形や輪郭の色が変わるたびに[`fill()`](/reference/p5/fill)、[`stroke()`](/reference/p5/stroke)、[`strokeWeight()`](/reference/p5/strokeWeight)を呼び出してください。

- 例については[このスケッチ](https://editor.p5js.org/Msqcoding/sketches/k0zCPrquf)をご覧ください。

[`fill()`](/reference/p5/fill)、[`stroke()`](/reference/p5/stroke)、[`strokeWeight()`](/reference/p5/strokeWeight)についてさらに学ぶには、[カラーリファレンス](/reference/p5/color)ページをご覧ください。[`fill()`](/reference/p5/fill)、[`stroke()`](/reference/p5/stroke)、[`background()`](/reference/p5/background)で使用するカラーコードを検索するには、p5.js Web Editorのカラーツールまたは[Googleのカラーピッカー](https://g.co/kgs/aCdbzD)を使用してください。

## ステップ7：キャンバスにさらに図形を描いて色を付ける

- キャンバスの下半分に草を描きます：
  - 太陽のコードの下に`stroke(0)`と`strokeWeight(1)`を追加して、図形の輪郭の値をリセットします。
  - `rect(0, 200, 600, 200)`を使用して、キャンバスに長方形の図形を追加します。
  - 長方形のステートメントの上の行に`fill("green")`を追加して、長方形を緑色に塗ります。
  - 草を作成するコード行を説明するコメントを追加します。
- （オプション）図形と色を使用して、風景にさらに詳細を追加します。以下の図形関数をクリックして、スケッチに含める方法を学びましょう：
  - [`ellipse()`](/reference/p5/ellipse)
  - [`square()`](/reference/p5/square)
  - [`quad()`](/reference/p5/quad)
  - [`triangle()`](/reference/p5/triangle)
  - [`line()`](/reference/p5/line)
  - [`point()`](/reference/p5/point)
  - [`arc()`](/reference/p5/arc)

<Callout title="注意">
- 図形と色を[`draw()`](/reference/p5/draw)関数の中括弧内に必ず追加してください。
- [`fill()`](/reference/p5/fill)は、特定の図形に色を付けるために、その図形が描画される行の前の行で呼び出す必要があります。追加の図形の色を変更するには、[`fill()`](/reference/p5/fill)を再度呼び出す必要があります。
</Callout>

<EditableSketch code={`
function setup() {
  //幅600ピクセル、高さ400ピクセルの
  //キャンバスを作成します
  createCanvas(600, 400);
}

function draw() {
  //空色の背景
  background(135, 206, 235);
  //右上の太陽
  fill("yellow");//黄色  

  stroke("orange"); //オレンジの輪郭 

  strokeWeight(20); //太い輪郭    

  circle(550, 50, 100);
  //下半分の草

  stroke(0);//黒い輪郭

  strokeWeight(1);//輪郭の太さ

  fill("green");

  rect(0, 200, 600, 200);
}
`} />

[`rect()`](/reference/p5/rect)は、長方形の水平位置（x座標）、垂直位置（y座標）、幅、高さを記述する数字を使用します。xとy座標（x, y）は長方形の左上の点を示します（以下の図を参照）。

- 上記のコード例で使用されている[`rect()`](/reference/p5/rect)関数の最初の*引数*である数字0は、x座標です。これは、長方形の左上隅がキャンバスの左端にあることを意味します。
- 2番目の*引数*である数字200は、y座標です。これは、長方形の左上隅がキャンバスの上端から200ピクセル下にあることを意味します。
- 3番目の*引数*である数字600は、長方形の幅です。これは、長方形が600ピクセル幅であることを意味します。
- 4番目の*引数*である数字200は、長方形の高さです。これは、長方形が200ピクセルの高さであることを意味します。

![水平の長さが「width」とラベル付けされ、垂直の長さが「height」とラベル付けされた長方形の図。左上隅に「(x,y)」とラベル付けされた点があります。rect()関数の使用構文が図の上に表示されています：「rect(x, y, width, height);」](../images/introduction/rect-diagram.png)

詳細については、[`rect()`](/reference/p5/rect)、[単純な図形](/reference/#Shape)、[色](/reference/#Color)のp5.jsリファレンスページをご覧ください。

## ステップ8：絵文字を追加する

- 花の絵文字とてんとう虫の絵文字（または他の好きな絵文字）を風景に追加します：
  - `text("🌸", 100, 250);`を使用して、キャンバスに花を描きます
  - `text("🐞", 300, 250);`を使用して、キャンバスにてんとう虫を描きます
    - 選択した絵文字は引用符で囲む必要があります（例：`"🌸"`）。
    - キーボードを使用して絵文字を挿入する方法については、以下の説明をご覧ください：
      - [Macで絵文字を入力する](https://support.apple.com/guide/mac-help/use-emoji-and-symbols-on-mac-mchlp1560/14.0/mac/14.0#mchl9571760f)
      - [Windowsで絵文字を入力する](https://blogs.windows.com/windowsexperience/2018/02/05/windows-10-tip-get-started-emoji-keyboard-shortcut/)
      - [Chromebookで絵文字を入力する](https://support.google.com/chromebook/answer/6076237?hl=en#zippy=%2Cinsert-emoji-or-other-images)
  - 絵文字が描画される行の前の行に`textSize(75)`を使用して、花とてんとう虫のサイズを調整します。
  - 花とてんとう虫を作成するコード行を説明するコメントを追加します。
  - キャンバスを更新するために*再生*ボタンを押すか、*自動更新*ボックスにチェックを入れることを忘れないでください。

<Callout title="注意">
[`textSize()`](/reference/p5/textSize)と[`text()`](/reference/p5/text)を、草を描くコード行の後の[`draw()`](/reference/p5/draw)のコードブロック内に必ず追加してください。[`text()`](/reference/p5/text)が[`rect()`](/reference/p5/rect)の前にコードに現れ、キャンバス上で長方形が描画される場所に配置されている場合、テキストが長方形の後ろに隠れてしまう可能性があります！重なりがどのように起こり得るかについての詳細な説明は、以下の[描画順序](#drawing-order)セクションをご覧ください。
</Callout>

コードは以下のようになるはずです：

<EditableSketch code={`
function setup() {
  //幅600ピクセル、高さ400ピクセルの
  //キャンバスを作成します
  createCanvas(600, 400);
}

function draw() {
  //空色の背景
  background(135, 206, 235);
  //右上隅の太陽
  fill("yellow");
  circle(550, 50, 100);
 
  //下半分の草
  fill("green");
  rect(0, 200, 600, 200);
 
  //絵文字
  textSize(75)
  text("🌸", 100, 250) //花
  text("🐞", 300, 250) //てんとう虫
}
`} />

[`text()`](/reference/p5/text)は、*文字列*（引用符（`""`）内のテキスト）とテキストのx座標とy座標を記述する数字を使用します。xとy座標（x, y）はテキストボックスの左下の点を示します（以下の図を参照）。

- 上記のコード例で使用されている[`text()`](/reference/p5/text)関数の最初の*引数*である文字列`"🌸"`は、キャンバスに表示されるテキストです。p5.jsで使用されるすべての文字列は引用符（`""`）内にある必要があります。
- 2番目の*引数*である数字100は、x座標です。これは、テキストボックスの左下隅がキャンバスの左端から100ピクセル右にあることを意味します。
- 3番目の*引数*である数字250は、y座標です。これは、テキストボックスの左下隅がキャンバスの上端から250ピクセル下にあることを意味します。

![破線で囲まれた長方形の中に「string」という単語があり、左下隅に「(x,y)」とラベル付けされた点がある図。破線の輪郭はテキストを囲む目に見えないボックスを表し、「(x,y)」の点はHTML &lt;canvas&gt;上でのその位置を記述する座標を表しています。](../images/introduction/text-diagram.png)

[`textSize()`](/reference/p5/textSize)は、その後に現れる[`text()`](/reference/p5/text)関数のサイズをピクセル単位で記述する数字を使用します。キャンバスに表示されるテキストのサイズを変更したい場合は、[`text()`](/reference/p5/text)関数を呼び出す前に必ず[`textSize()`](/reference/p5/textSize)を使用してください。

- テキストサイズの違いについては、[この例](https://editor.p5js.org/Msqcoding/sketches/kQa37NCgT)をご覧ください。

### <a id="drawing-order"></a>描画順序

[`draw()`](/reference/p5/draw)は括弧内のコードを何度も繰り返し実行するため、図形はキャンバス上に層のように描画されます。最初に現れるコードが最初に描画され、[`draw()`](/reference/p5/draw)の中でより下に現れるコードはその後に描画されます。これが、図形やテキストをキャンバスに追加する前に[`fill()`](/reference/p5/fill)、[`stroke()`](/reference/p5/stroke)、[`textSize()`](/reference/p5/textSize)を設定する必要がある理由です。p5.jsは図形やテキストを描画する***前に***、その色やサイズを知る必要があります。これにより、図形を重ねることもでき、時には他の図形の後ろに図形を隠すこともできます！図形が他の図形の後ろに隠れていないか確認するために、図形のx座標とy座標を必ず確認してください。

重ね合わせの例については、これらのスケッチをご覧ください：[重なり合う図形（隠れている）](https://editor.p5js.org/Msqcoding/sketches/A4qkDd729)と[重なり合う図形](https://editor.p5js.org/Msqcoding/sketches/s822PY5T8)

詳細については、[`text()`](/reference/p5/text)、[`textSize()`](/reference/p5/textSize)、[`draw()`](/reference/p5/draw)のリファレンスページをご覧ください。

### データ型

p5.jsの一部の関数は*文字列*データ型の*引数*を必要とし、他の関数は*数値*データ型の引数を必要とします。

- [文字列](/reference/p5/String)：テキストで表され、常に引用符（`""`）で囲まれています
- [数値](/reference/p5/number)：数字で表されます

例えば：

- [`circle()`](/reference/p5/circle)は3つの[数値](/reference/p5/number)を使用します：x座標を表す数値、y座標を表す数値、ピクセル単位のサイズを表す数値。
- [`text()`](/reference/p5/text)は表示するテキストを表す[文字列](/reference/p5/String)と、2つの[数値](/reference/p5/number)を使用します：x座標を表す数値とy座標を表す数値。
- [`fill()`](/reference/p5/fill)、[`stroke()`](/reference/p5/stroke)、[`background()`](/reference/p5/background)は[数値](/reference/p5/number)と[文字列](/reference/p5/String)の両方を使用します：
  - グレースケールの色には1つの数値
  - [RGB色](https://www.w3schools.com/html/html_colors_rgb.asp)にはコンマで区切られた3つの数値
  - [HTML色名](https://www.w3schools.com/tags/ref_colornames.asp)または[HEX色値](https://www.w3schools.com/html/html_colors_hex.asp)を含む[文字列](/reference/p5/String)


## ステップ9：インタラクティブにする！

- てんとう虫の絵文字のx座標とy座標を`mouseX`、`mouseY`に変更します。
  - `text("🐞", mouseX, mouseY);`
- マウスポインターをキャンバス上で動かし、てんとう虫がマウスポインターの位置に従うのを確認してください！
  - キャンバスを更新するために*再生*ボタンを押すか、*自動更新*ボックスにチェックを入れることを忘れないでください。
- てんとう虫がマウスポインターと一緒に動く仕組みを説明するコメントを追加してください。

コードは以下のようになるはずです：

<EditableSketch code={`
function setup() {
  //幅600ピクセル、高さ400ピクセルの
  //キャンバスを作成します
  createCanvas(600, 400);
}

function draw() {
  //空色の背景
  background(135, 206, 235);
  //右上隅の太陽
  fill("yellow");
  circle(550, 50, 100);
 
  //下半分の草
  fill("green");
  rect(0, 200, 600, 200);
 
  //絵文字
  textSize(75)
  text("🌸", 100, 250) //花
  text("🐞", mouseX, mouseY) //てんとう虫
}
`} />

[`mouseX`](/reference/p5/mouseX)と[`mouseY`](/reference/p5/mouseY)はp5.jsライブラリに付属のビルトイン変数で、システム変数と呼ばれています。変数は後でスケッチで使用できる値を格納します。また、x座標やy座標などの値が変化することがわかっている場合に使用すると便利です。[`mouseX`](/reference/p5/mouseX)と[`mouseY`](/reference/p5/mouseY)は特に、マウスポインターがキャンバス上をドラッグされるときのマウスポインターのx座標とy座標を格納します。[数値](/reference/p5/number)を必要とする任意の引数に[`mouseX`](/reference/p5/mouseX)と[`mouseY`](/reference/p5/mouseY)を使用できます！

スケッチで[`mouseX`](/reference/p5/mouseX)と[`mouseY`](/reference/p5/mouseY)を使用する方法の例をいくつか見てみましょう：

- [マウスのx座標とy座標を表示する（mouseXとmouseY）](https://editor.p5js.org/Msqcoding/sketches/v9PYEX_vY)
- [mouseXとmouseYで色を変更する](https://editor.p5js.org/Msqcoding/sketches/VbLtYOfLc)
- [mouseXとmouseYでサイズを変更する](https://editor.p5js.org/Msqcoding/sketches/gJ27u_HEZ)

詳細については、[`mouseX`](/reference/p5/mouseX)と[`mouseY`](/reference/p5/mouseY)のp5.jsリファレンスページをご覧ください！p5.js座標系の復習は、キャンバス座標系と図形のp5.jsリファレンスページをご覧ください。[p5.jsリファレンス](https://p5js.org/reference/)に含まれる他のシステム変数を探索してみてください：

- [`width`](/reference/p5/width)：キャンバスの幅
- [`height`](/reference/p5/height)：キャンバスの高さ
- [`pmouseX`](/reference/p5/pmouseX)：マウスの前のx座標
- [`pmouseY:`](/reference/p5/pmouseY)マウスの前のy座標

### エラー

関数名のスペルを間違えたり、コンマを忘れたりするのは簡単です。構文規則はコンピュータがコードを解釈するのに役立ちます。「ルール」が破られると、コンソールにエラーメッセージが表示されます（例えば、[circle()](/reference/p5/circle)のスペルが間違っている場合）。これらのエラーは一般的に「バグ」と呼ばれます。コンソールには、あなたが犯した可能性のある間違いの詳細が表示されます。コードが正しく実行されない場合、コードにバグがある可能性があります！一般的なエラーの例とその修正方法については、[デバッグのフィールドガイド](field-guide-to-debugging)をご覧ください。

詳細については、The Coding Trainの[このビデオ](https://www.youtube.com/watch?v=LuGsp5KeJMM\&list=PLRqwX-V7Uu6Zy51Q-x9tMWIv9cueOFTFA\&index=6)をご覧ください。
