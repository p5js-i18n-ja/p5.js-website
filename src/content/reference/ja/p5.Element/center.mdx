  <p>要素を垂直方向、水平方向、または両方の方向に中央揃えします。</p>

  <p><code>center()</code>は、要素を親要素に対して中央揃えします。親要素がない場合は、
  ページのbodyに対して中央揃えします。</p>

  <p>引数が渡されない場合（<code>myElement.center()</code>のように）、要素は
  垂直方向と水平方向の両方に中央揃えされます。</p>

    function setup() {
      createCanvas(100, 100);

      // div要素を作成してスタイルを設定します。
      let div = createDiv('');
      div.size(10, 10);
      div.style('background-color', 'orange');

      // divをページのbodyに対して中央揃えします。
      div.center();

      describe('灰色の正方形とオレンジ色の長方形。長方形はページの中央にあります。');
    }
