description: >
  <p>要素の位置を設定します。</p>

  <p>最初の2つのパラメータ、<code>x</code>と<code>y</code>は、
  ウェブページの左上隅を基準にした要素の位置を設定します。</p>

  <p>3番目のパラメータ<code>positionType</code>はオプションです。これは要素の
  <a target="_blank"
  href="https://developer.mozilla.org/en-US/docs/Web/CSS/position/">配置方法</a>
  を設定します。
  <code>positionType</code>は文字列で、<code>'static'</code>、<code>'fixed'</code>、
  <code>'relative'</code>、<code>'sticky'</code>、<code>'initial'</code>、または
  <code>'inherit'</code>のいずれかを指定できます。</p>

  <p>引数が渡されない場合、例えば<code>myElement.position()</code>のように、このメソッドは
  要素の位置を<code>{ x: 0, y: 0 }</code>のようなオブジェクトで返します。</p>
