description: >
  <p>要素を親要素に付加します。</p>

  <p>例えば、<code><div></div></code>要素は、見出しと段落という2つのテキスト要素を

  保持するボックスとして使用されることがあります。この

  <code><div></div></code>は、見出しと段落の両方の親要素となります。</p>

  <p><code>parent</code>パラメータには3つのタイプがあります。
  <code>parent</code>は

  親要素のIDを示す文字列で、

  <code>myElement.parent('container')</code>のように使用できます。また、

  <a href="/reference/p5/p5.Element">p5.Element</a>オブジェクトを指定することもでき、

  <code>myElement.parent(myDiv)</code>のように使用します。最後に、<code>parent</code>は
  <code>HTMLElement</code>

  オブジェクトも指定でき、<code>myElement.parent(anotherElement)</code>のように使用します。</p>

  <p>引数なしで<code>myElement.parent()</code>を呼び出すと、その要素の

  親要素を返します。</p>
