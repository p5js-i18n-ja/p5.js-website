description: >
  <p>フレームバッファのピクセル密度を設定するか、現在の密度を返します。</p>

  <p>コンピューターディスプレイは、ピクセルと呼ばれる小さな光の格子です。ディスプレイの

  ピクセル密度は、ある領域にどれだけのピクセルが詰め込まれているかを表します。

  より小さなピクセルを持つディスプレイは、より高いピクセル密度を持ち、より鮮明な

  画像を作成します。</p>

  <p>パラメータ<code>density</code>はオプションです。数値が渡された場合、例えば

  <code>myBuffer.pixelDensity(1)</code>のように、フレームバッファのピクセル密度を

  設定します。デフォルトでは、フレームバッファのピクセル密度は作成された

  キャンバスのピクセル密度と一致します。すべてのキャンバスはデフォルトで

  ディスプレイのピクセル密度に合わせられます。</p>

  <p>引数なしで<code>myBuffer.pixelDensity()</code>を呼び出すと、現在の

  ピクセル密度を返します。</p>

      // p5.Framebufferオブジェクトに描画します。
      myBuffer.begin();
      background(200);
      circle(0, 0, 40);
      myBuffer.end();

      // p5.Framebufferオブジェクトを表示します。
      image(myBuffer, -50, -50);
    }

    // ユーザーがマウスを押したときに
    // ピクセル密度を下げます。
    function mousePressed() {
      myBuffer.pixelDensity(1);
    }

    // ユーザーがマウスを離したときに
    // ピクセル密度を上げます。
    function mouseReleased() {
      myBuffer.pixelDensity(2);
    }
