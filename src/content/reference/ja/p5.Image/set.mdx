  <p>画像の特定の位置にピクセルまたは画像を設定します。</p>
  <p>この関数は、画像の特定の位置（x, y）にピクセルの色または別の画像を設定します。
  色は、グレースケール値、[R, G, B]配列、[R, G, B, A]配列、
  p5.Colorオブジェクト、またはCSSカラー文字列として指定できます。</p>
  <p>この関数を使用した後、変更を適用するために
  <a href="#/p5.Image/updatePixels">updatePixels()</a>を呼び出す必要があります。</p>

  - name: y

  - name: a
    description: |
      <p>グレースケール値、カラー配列、p5.Color、CSS文字列、またはp5.Image。</p>

example:
  - |-

    <div>
    <code>
    let img;

    function preload() {
      img = loadImage('/assets/rockies.jpg');
    }

    function setup() {
      createCanvas(100, 100);
      background(200);

      // 画像を表示します。
      image(img, 0, 0, width, height);

      // 画像の中央に赤いピクセルを設定します。
      let x = img.width / 2;
      let y = img.height / 2;
      img.set(x, y, 'red');
      img.updatePixels();

      // 更新された画像を表示します。
      image(img, 0, 0, width, height);

      describe('山の風景の画像。中央に赤いピクセルがあります。');
    }
