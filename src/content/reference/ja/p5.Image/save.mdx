  <p>画像をファイルとして保存します。</p>
  <p>デフォルトでは、画像は現在のスケッチのフォルダに保存されます。
  オプションのパラメータを使用して、ファイル名と拡張子を指定できます。</p>
  <p>この関数は非同期で動作します。つまり、即座に完了せず、
  次の行のコードがすぐに実行されます。</p>
line: 1981
params:
  - name: filename
    description: |
      <p>保存するファイルの名前。</p>
    type: String
    optional: true
  - name: extension
    description: |
      <p>ファイルの拡張子。'png'または'jpg'を指定できます。</p>
    type: String
    optional: true
itemtype: method
example:
  - |-

    <div>
    <code>
    let img;

    function preload() {
      img = loadImage('/assets/rockies.jpg');
    }

    function setup() {
      createCanvas(100, 100);
      background(200);

      // 画像を表示します。
      image(img, 0, 0, width, height);

      // 画像を保存します。
      img.save('myImage', 'png');

      describe('山の風景の画像。"myImage.png"という名前で保存されます。');
    }
    </code>
    </div>
