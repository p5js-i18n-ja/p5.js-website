description: >
  <p>色を<code>String</code>としてフォーマットして返します。</p>

  <p><code>print(myColor.toString())</code>のように、<code>myColor.toString()</code>を呼び出すことは
  デバッグに便利です。また、p5.jsを他のライブラリと一緒に使用する際にも
  役立ちます。</p>

  <p>パラメータ<code>format</code>はオプションです。<code>myColor.toString('#rrggbb')</code>のように
  フォーマット文字列を渡すと、色文字列のフォーマット方法を決定します。
  デフォルトでは、色文字列は<code>'rgba(r, g, b, a)'</code>としてフォーマットされます。</p>

class: p5.Color
params:
  - name: format
    description: >
      <p>色文字列のフォーマット方法。
      空のままにすると、文字列はrgba(r, g, b, a)としてフォーマットされます。
      '#rgb' '#rgba' '#rrggbb' '#rrggbbaa'は16進数カラーコードとしてフォーマットします。
      'rgb' 'hsb' 'hsl'は指定された色モードでフォーマットされた色を返します。
      'rgba' 'hsba' 'hsla'は上記と同じですがアルファチャンネルが含まれます。
      'rgb%' 'hsb%' 'hsl%' 'rgba%' 'hsba%' 'hsla%'はパーセンテージとしてフォーマットします。</p>
    type: String
    optional: true

  description: フォーマットされた文字列。
