description: >
  <p>色を<code>String</code>としてフォーマットして返します。</p>

  <p><code>myColor.toString()</code>を呼び出すことは、
  <code>print(myColor.toString())</code>のようにデバッグに役立ちます。
  また、p5.jsを他のライブラリと一緒に使用する際にも便利です。</p>

  <p>パラメータ<code>format</code>はオプションです。フォーマット文字列が渡された場合
  （<code>myColor.toString('#rrggbb')</code>のように）、
  色文字列のフォーマット方法が決定されます。デフォルトでは、色文字列は
  <code>'rgba(r, g, b, a)'</code>としてフォーマットされます。</p>

isConstructor: false
itemtype: method
class: p5.Color
params:
  - name: format
    description: >
      <p>色文字列のフォーマット方法。

      空のままにすると、文字列はrgba(r, g, b, a)としてフォーマットされます。

      '#rgb' '#rgba' '#rrggbb' '#rrggbbaa'は16進数のカラーコードとしてフォーマットします。

      'rgb' 'hsb' 'hsl'は指定された色モードでフォーマットされた色を返します。

      'rgba' 'hsba' 'hsla'は上記と同じですが、アルファチャンネルが含まれます。

      'rgb%' 'hsb%' 'hsl%' 'rgba%' 'hsba%' 'hsla%'はパーセンテージとしてフォーマットします。</p>
    type: String
    optional: true

  description: |-
