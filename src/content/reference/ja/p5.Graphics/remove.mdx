description: >
  <p>グラフィックスバッファをウェブページから削除します。</p>

  <p><code>myGraphics.remove()</code>を呼び出すと、グラフィックスバッファの

  <code><canvas></code>要素がウェブページから削除されます。グラフィックスバッファは

  CPUのメモリも少し使用しており、以下のように解放できます：</p>

  <pre><code class="language-js">// グラフィックスバッファをウェブページから削除します。

  myGraphics.remove();


  // グラフィックスバッファをCPUメモリから削除します。

  myGraphics = undefined;

  </code></pre>

  <p>注意：グラフィックスバッファをCPUメモリから削除するには、グラフィックスバッファを

  参照するすべての変数に<code>undefined</code>を代入する必要があります。変数が

  まだグラフィックスバッファを参照している場合、ガベージコレクションの対象に

  なりません。</p>

      pg = createGraphics(60, 60);

      pg.background(100);

    function draw() {
      background(200);

      // p5.Graphicsオブジェクトが利用可能な場合、
      // 表示します。
      if (pg) {
        image(pg, 20, 20);
      }
    }

    // ユーザーがダブルクリックしたときに
    // p5.Graphicsオブジェクトを削除します。
    function doubleClicked() {
      // p5.Graphicsオブジェクトをウェブページから削除します。
      pg.remove();
      pg = undefined;
    }
