description: >
  <p>ジオメトリのテクスチャのv座標を反転させます。</p>

  <p><a href="/reference/p5/texture/">texture()</a>が機能するためには、ジオメトリは
  その表面上の点を長方形の画像のピクセルにマッピングする方法が
  必要です。この長方形の画像がテクスチャとして使用されます。ジオメトリの
  座標<code>(x, y, z)</code>の頂点は、テクスチャ画像の座標<code>(u, v)</code>の
  ピクセルにマッピングされます。</p>

  <p><a href="/reference/p5.Geometry/uvs/">myGeometry.uvs</a>配列は、
  ジオメトリに追加された順序で各頂点の<code>(u, v)</code>座標を
  格納します。<code>myGeometry.flipV()</code>を呼び出すと、ジオメトリの
  v座標が反転し、テクスチャが垂直方向に鏡像反転して表示されます。</p>

  <p>例えば、平面の4つの頂点は左上の角から時計回りに追加されます。
  以下は<code>myGeometry.flipV()</code>を呼び出すことで、平面のテクスチャ
  座標がどのように変化するかを示しています：</p>

  <pre><code class="language-js">// 元のテクスチャ座標を表示します。
  // 出力: [0, 0, 1, 0, 0, 1, 1, 1]
  console.log(myGeometry.uvs);

  // v座標を反転させます。
  myGeometry.flipV();

  // 反転後のテクスチャ座標を表示します。
  // 出力: [0, 1, 1, 1, 0, 0, 1, 0]
  console.log(myGeometry.uvs);

  // 以下の入れ替えに注目してください：
  // 左側の頂点: [0, 0] <--> [1, 0]
  // 右側の頂点: [1, 0] <--> [1, 1]
  </code></pre>
