description: >
  <p>ベクトルの <code>x</code>、<code>y</code>、および <code>z</code> 成分を設定します。</p>

  <p><code>set()</code> は、<code>v.set(1, 2, 3)</code> のように個別の数値、
  <code>v.set(v2)</code> のように <a href="/reference/p5/p5.Vector">p5.Vector</a> オブジェクト、
  または <code>v.set([1, 2, 3])</code> のように数値の配列を使用できます。</p>

  <p>成分に値が提供されない場合、0に設定されます。例えば、
  <code>v.set(4, 5)</code> は <code>v.x</code> を4に、<code>v.y</code> を5に、
  <code>v.z</code> を0に設定します。引数なしで <code>set()</code> を呼び出すと、
  <code>v.set()</code> のように、ベクトルのすべての成分が0に設定されます。</p>
