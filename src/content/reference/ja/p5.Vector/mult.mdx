description: >
  <p>ベクトルの <code>x</code>、<code>y</code>、および <code>z</code> 成分を乗算します。</p>

  <p><code>mult()</code> は、<code>v.mult(1, 2, 3)</code> のように個別の数値、
  <code>v.mult(v2)</code> のように別の <a href="/reference/p5/p5.Vector">p5.Vector</a> オブジェクト、
  または <code>v.mult([1, 2, 3])</code> のように数値の配列を使用できます。</p>

  <p><code>v.mult(2)</code> のように1つの値だけが提供された場合、すべての成分が2倍になります。
  成分に値が提供されない場合、その成分は変更されません。例えば、<code>v.mult(4, 5)</code> は
  <code>v.x</code> を4倍、<code>v.y</code> を5倍、<code>v.z</code> を1倍します。
  <code>v.mult()</code> のように引数なしで呼び出した場合、効果はありません。</p>

  <p><code>mult()</code> の静的バージョン（<code>p5.Vector.mult(v, 2)</code> のような）は、
  新しい <a href="/reference/p5/p5.Vector">p5.Vector</a> オブジェクトを返し、
  元のオブジェクトは変更しません。</p>
