description: >
  <p>ベクトルの <code>x</code>、<code>y</code>、<code>z</code> 成分に対して剰余（余り）除算を実行します。</p>

  <p><code>rem()</code> は、<code>v.rem(1, 2, 3)</code> のように個別の数値を使用したり、
  <code>v.rem(v2)</code> のように別の <a href="/reference/p5/p5.Vector">p5.Vector</a> オブジェクトを使用したり、
  <code>v.rem([1, 2, 3])</code> のように数値の配列を使用したりすることができます。</p>

  <p><code>v.rem(2)</code> のように1つの値だけが提供された場合、すべての成分が2で割った余りに設定されます。
  <code>v.rem(2, 3)</code> のように2つの値が提供された場合、<code>v.z</code> は変更されません。
  <code>v.rem()</code> のように引数なしで呼び出した場合は何も効果がありません。</p>

  <p><code>rem()</code> の静的バージョンである <code>p5.Vector.rem(v2, v1)</code> は、
  新しい <a href="/reference/p5/p5.Vector">p5.Vector</a> オブジェクトを返し、
  元のオブジェクトは変更しません。</p>
