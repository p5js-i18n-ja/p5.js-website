description: >
  <p>ベクトルの <code>x</code>、<code>y</code>、および <code>z</code> 成分を除算します。</p>

  <p><code>div()</code> は、<code>v.div(1, 2, 3)</code> のように個別の数値、
  <code>v.div(v2)</code> のように別の <a href="/reference/p5/p5.Vector">p5.Vector</a> オブジェクト、
  または <code>v.div([1, 2, 3])</code> のように数値の配列を使用できます。</p>

  <p><code>v.div(2)</code> のように1つの値のみが提供された場合、すべての成分が2で除算されます。
  成分に値が提供されない場合、その成分は変更されません。例えば、<code>v.div(4, 5)</code> は
  <code>v.x</code> を4で、<code>v.y</code> を5で、<code>v.z</code> を1で除算します。
  <code>v.div()</code> のように引数なしで呼び出した場合、効果はありません。</p>

  <p><code>div()</code> の静的バージョンである <code>p5.Vector.div(v, 2)</code> は、
  新しい <a href="/reference/p5/p5.Vector">p5.Vector</a> オブジェクトを返し、
  元のオブジェクトは変更しません。</p>
