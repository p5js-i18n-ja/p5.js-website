description: >
  <p>2つのベクトル間で比例的に同じ距離にある新しい<code>x</code>、<code>y</code>、<code>z</code>
  成分を計算します。</p>

  <p><code>amt</code>パラメータは、古いベクトルと新しいベクトルの間で補間する量です。
  0.0は全ての成分を古いベクトルと同じに保ち、0.5は中間点、
  1.0は全ての成分を新しいベクトルと同じにします。</p>

  <p><code>lerp()</code>の静的バージョン（<code>p5.Vector.lerp(v0, v1, 0.5)</code>のような）は、
  新しい<a href="/reference/p5/p5.Vector">p5.Vector</a>オブジェクトを返し、
  元のオブジェクトは変更しません。</p>

    function drawArrow(base, vec, myColor) {
      push();
      stroke(myColor);
      strokeWeight(3);
      fill(myColor);
      translate(base.x, base.y);
      line(0, 0, vec.x, vec.y);
      rotate(vec.heading());
      let arrowSize = 7;
      translate(vec.mag() - arrowSize, 0);
      triangle(0, arrowSize / 2, 0, -arrowSize / 2, arrowSize, 0);
      pop();
    }
