description: >
  <p>フォントをロードし、<a href="/reference/p5/p5.Font">p5.Font</a>オブジェクトを返します。</p>

  <p><code>loadFont()</code>は非同期関数です。通常、<code>preload()</code>内で呼び出されます。
  これにより、フォントが<code>setup()</code>や<code>draw()</code>で使用可能になる前に
  完全にロードされることが保証されます。</p>

  <p><code>loadFont()</code>を<code>preload()</code>の外で呼び出す場合、
  コールバック関数を渡す必要があります。コールバックは、フォントが完全にロードされた後に
  実行されます。</p>

  <p>サポートされているフォント形式は、TrueType（.ttf）、OpenType（.otf）、WOFF、およびWOFF2です。
  ブラウザのサポート状況については、
  <a href="https://caniuse.com/woff" target="_blank">WOFF</a>と
  <a href="https://caniuse.com/woff2" target="_blank">WOFF2</a>を参照してください。</p>

example:
  - |-

    <div>
    <code>
    let myFont;

    function preload() {
      // フォントをロードします。
      myFont = loadFont('/assets/inconsolata.otf');
    }

    function setup() {
      createCanvas(100, 100);

      background(200);

      // フォントを設定します。
      textFont(myFont);
      textSize(20);

      // テキストを描画します。
      text('hello', 20, 50);

      describe('灰色の背景に黒で「hello」と書かれています。');
    }
    </code>

    <div>
    <code>
    let myFont;

    function setup() {
      createCanvas(100, 100);

      background(200);

      // フォントをロードします。
      loadFont('/assets/inconsolata.otf', (font) => {
        // フォントを設定します。
        textFont(font);
        textSize(20);

        // テキストを描画します。
        text('hello', 20, 50);
      });

      describe('灰色の背景に黒で「hello」と書かれています。');
    }
    </code>
