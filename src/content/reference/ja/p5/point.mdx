description: >
  <p>空間に単一の点を描画します。</p>

  <p>点のデフォルトの幅は1ピクセルです。点の色を変更するには

  <a href="/reference/p5/stroke/">stroke()</a> 関数を使用します。幅を変更するには

  <a href="/reference/p5/strokeWeight/">strokeWeight()</a> 関数を使用します。点は

  塗りつぶすことができないため、<a href="/reference/p5/fill/">fill()</a> 関数は

  点の色に影響を与えません。</p>

  <p>2つのパラメータを持つ <code>point()</code> のバージョンでは、点の位置を

  x座標とy座標で設定できます。例: <code>point(10, 20)</code>。</p>

  <p>3つのパラメータを持つ <code>point()</code> のバージョンでは、点を

  x座標、y座標、z座標で3D空間に描画できます。例: <code>point(10, 20,

  30)</code>。これを行うには、<a href="/reference/p5/createCanvas/">createCanvas()</a> に

  <code>WEBGL</code> 引数を追加する必要があります。</p>

  <p>1つのパラメータを持つ <code>point()</code> のバージョンでは、点の位置を

  <a href="/reference/p5/p5.Vector">p5.Vector</a> オブジェクトで設定できます。</p>
