description: >
  <p>キャンバスを画像として保存します。</p>

  <p>デフォルトでは、<code>saveCanvas()</code>はキャンバスをPNGファイルとして保存し、

  ファイル名を<code>untitled.png</code>とします。</p>

  <p>最初のパラメータ<code>filename</code>は、ファイル名を設定します。

  ファイル名には拡張子を含めないでください。例えば、<code>'myCanvas'</code>と

  指定すると、<code>'myCanvas.png'</code>として保存されます。</p>

  <p>2番目のパラメータ<code>extension</code>は、ファイルタイプを設定します。

  <code>'png'</code>または<code>'jpg'</code>を使用できます。</p>

  <p>3番目のパラメータ<code>canvas</code>は、保存するキャンバスを指定します。

  デフォルトでは、現在のキャンバスが使用されます。</p>

  <p>注意：ブラウザによっては、大きなキャンバスの保存に失敗する場合があります。</p>

example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);
      background(255);
      rect(20, 20, 60, 60);
      describe('白い背景に黒い正方形が描かれています。');
    }

    // キャンバスを保存するには、's'キーを押してください。
    function keyPressed() {
      if (key === 's') {
        saveCanvas('myCanvas', 'png');
      }
    }
    </code>
    </div>

    function setup() {

      rect(20, 20, 60, 60);

    }

    // キャンバスを保存するには、's'キーを押してください。
    function keyPressed() {
      if (key === 's') {
        saveCanvas(cnv, 'myCanvas', 'jpg');
      }
    }
    </code>
