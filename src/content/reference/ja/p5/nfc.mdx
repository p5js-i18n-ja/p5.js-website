description: >
  <p><code>Number</code>を、1,000単位にカンマを付けた<code>String</code>に変換します。</p>

  <p><code>nfc()</code>は、12345のような数字を<code>'12,345'</code>のようにカンマで
  
  千の位を区切った文字列に変換します。</p>

  <p>最初のパラメータ<code>num</code>は、文字列に変換する数字です。
  
  例えば、<code>nfc(12345)</code>を呼び出すと、文字列<code>'12,345'</code>が返されます。</p>

  <p>2番目のパラメータ<code>right</code>はオプションです。数字が渡された場合、
  
  例えば<code>nfc(12345, 1)</code>のように、小数点以下に含める最小桁数を
  
  設定します。<code>right</code>が<code>num</code>の小数点以下の桁数より
  
  小さい場合、<code>num</code>は指定された小数点以下の桁数に丸められます。
  
  例えば、<code>nfc(12345.67, 1)</code>を呼び出すと、文字列
  
  <code>'12,345.7'</code>が返されます。<code>right</code>が<code>num</code>の小数点以下の
  
  桁数より大きい場合、未使用の小数点以下の桁は0に設定されます。例えば、
  
  <code>nfc(12345.67, 3)</code>を呼び出すと、文字列<code>'12,345.670'</code>が
  
  返されます。</p>

      // 数値の配列を作成します。
      let numbers = [12345, 6789];

        let y = (i + 1) * 33;

        // 元の数値とフォーマットされた数値を表示します。
        text(`${numbers[i]} : ${formatted[i]}`, 50, y);
      }

        '2行に分けて書かれたテキスト "12345 : 12,345" と "6789 : 6,789"。テキストは灰色の背景に黒で書かれています。'
      );
