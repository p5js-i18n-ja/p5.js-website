description: >
  <p><code>Number</code>型のシステム変数で、マウスの前回の垂直位置を追跡します。</p>

  <p>2Dモードでは、<code>pmouseY</code>はキャンバスの左上隅を基準にしたマウスの位置を
  追跡します。その値は前のフレームの<a href="/reference/p5/mouseY/">mouseY</a>です。
  例えば、前のフレームでマウスがキャンバスの上端から50ピクセルの位置にあった場合、
  <code>pmouseY</code>は50になります。</p>

  <p>WebGLモードでは、<code>pmouseY</code>はキャンバスの中心を基準にしたマウスの位置を
  追跡します。例えば、前のフレームでマウスがキャンバスの中心から50ピクセル下にあった
  場合、<code>pmouseY</code>は50になります。</p>

  <p>マウスの代わりにタッチが使用された場合、<code>pmouseY</code>は最後のタッチポイントの
  y座標を保持します。</p>

  <p>注意：<code>pmouseY</code>は各タッチイベントの開始時に現在の<a
  href="/reference/p5/mouseY/">mouseY</a>の値にリセットされます。</p>

      background(200);

      // WebGLモードの座標を調整する
      // 原点(0, 0)はキャンバスの中心にある

      let pmy = pmouseY - 50;
      let mx = mouseX - 50;
      let my = mouseY - 50;

      // 線を描画する
      line(pmx, pmy, mx, my);
    }
