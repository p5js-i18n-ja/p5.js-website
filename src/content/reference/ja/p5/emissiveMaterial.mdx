description: >
  <p>形状の表面材質の発光色を設定します。</p>

  <p><code>emissiveMaterial()</code>の色は、形状が照明に関係なく最大強度で表示する色を設定します。

  これにより、形状が光っているように見えることがあります。ただし、発光材質は実際には

  周囲のオブジェクトに影響を与える光を放出しません。</p>

  <p><code>emissiveMaterial()</code>は、材質の色を設定するために異なるパラメータで

  3つの方法で呼び出すことができます。</p>

  <p><code>emissiveMaterial()</code>を呼び出す最初の方法は、<code>gray</code>というパラメータを1つ持ちます。

  <code>emissiveMaterial(50)</code>のように、0から255の間のグレースケール値を

  渡して材質の色を設定できます。グレースケール値が高いほど、形状は

  より明るく見えます。</p>

  <p><code>emissiveMaterial()</code>を呼び出す2つ目の方法は、<code>color</code>というパラメータを1つ持ちます。

  <a href="/reference/p5/p5.Color">p5.Color</a>オブジェクト、色値の配列、または

  <code>emissiveMaterial('magenta')</code>のようなCSS色文字列を渡して

  材質の色を設定できます。</p>

  <p><code>emissiveMaterial()</code>を呼び出す3つ目の方法は、<code>v1</code>、<code>v2</code>、

  <code>v3</code>、<code>alpha</code>という4つのパラメータを持ちます。<code>alpha</code>はオプションです。RGBA、

  HSBA、またはHSLA値を渡して材質の色を設定できます。例えば、<code>emissiveMaterial(255, 0, 0)</code>や

  <code>emissiveMaterial(255, 0, 0, 30)</code>のようにします。色の値は

  現在の<a href="/reference/p5/colorMode/">colorMode()</a>を使用して解釈されます。</p>

  <p>注意：<code>emissiveMaterial()</code>はWebGLモードでのみ使用できます。</p>
