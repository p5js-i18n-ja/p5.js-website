description: >
  <p><a href="/reference/p5/angleMode/">angleMode()</a>を設定するために使用される<code>String</code>定数です。</p>

  <p>デフォルトでは、<a href="/reference/p5/rotate/">rotate()</a>や

  <a href="/reference/p5/sin/">sin()</a>などの関数は、ラジアン単位で測定された角度を想定しています。

  <code>angleMode(RADIANS)</code>を呼び出すことで、角度がラジアン単位で測定されることを確認します。

  これは、<a href="/reference/p5/angleMode/">angleMode()</a>が

  <a href="/reference/p5/DEGREES/">DEGREES</a>に設定されている場合に便利です。</p>

  <p>注意：<code>TWO_PI</code>ラジアンは360˚に等しいです。</p>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // 度数を使用する
      angleMode(DEGREES);

      // 0˚から90˚までの赤い弧を描く
      fill(255, 0, 0);
      arc(50, 50, 80, 80, 0, 90);

      // ラジアンを使用する
      angleMode(RADIANS);

      // HALF_PIからPIまでの青い弧を描く
      fill(0, 0, 255);
      arc(50, 50, 80, 80, HALF_PI, PI);

      describe('灰色の背景に描かれた円の下半分。右下の4分の1は赤色。左下の4分の1は青色。');
    }
