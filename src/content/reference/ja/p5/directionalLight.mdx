description: >
  <p>一方向に光を放つライトを作成します。</p>

  <p>指向性ライトは特定の点から光を放つわけではありません。画面外のどこかから照らす太陽のようなものです。ライトの方向は、-1から1の間の3つの<code>(x, y, z)</code>値を使って設定されます。例えば、ライトの方向を<code>(1, 0, 0)</code>に設定すると、ライトが右方向を向いているため、<a href="/reference/p5/p5.Geometry">p5.Geometry</a>オブジェクトは左側から照らされます。</p>

  <p><code>directionalLight()</code>を呼び出す際、ライトの色と方向を設定するパラメータには4つの方法があります。</p>

  <p><code>directionalLight()</code>を呼び出す最初の方法は6つのパラメータを使用します。最初の3つのパラメータ、<code>v1</code>、<code>v2</code>、<code>v3</code>は、現在の<a href="/reference/p5/colorMode/">colorMode()</a>を使用してライトの色を設定します。残りの3つのパラメータ、<code>x</code>、<code>y</code>、<code>z</code>は、ライトの方向を設定します。例えば、<code>directionalLight(255, 0, 0, 1, 0, 0)</code>は、赤色<code>(255, 0, 0)</code>で右方向<code>(1, 0, 0)</code>に光を放つライトを作成します。</p>

  <p><code>directionalLight()</code>を呼び出す2つ目の方法は4つのパラメータを使用します。最初の3つのパラメータ、<code>v1</code>、<code>v2</code>、<code>v3</code>は、現在の<a href="/reference/p5/colorMode/">colorMode()</a>を使用してライトの色を設定します。最後のパラメータ、<code>direction</code>は、<a href="/reference/p5/p5.Geometry">p5.Geometry</a>オブジェクトを使用してライトの方向を設定します。例えば、<code>directionalLight(255, 0, 0, lightDir)</code>は、赤色<code>(255, 0, 0)</code>で<code>lightDir</code>ベクトルが指す方向に光を放つライトを作成します。</p>

  <p><code>directionalLight()</code>を呼び出す3つ目の方法は4つのパラメータを使用します。最初のパラメータ、<code>color</code>は、<a href="/reference/p5/p5.Color">p5.Color</a>オブジェクトまたは色の値の配列を使用してライトの色を設定します。残りの3つのパラメータ、<code>x</code>、<code>y</code>、<code>z</code>は、ライトの方向を設定します。例えば、<code>directionalLight(myColor, 1, 0, 0)</code>は、<code>myColor</code>の色値で右方向<code>(1, 0, 0)</code>に光を放つライトを作成します。</p>

  <p><code>directionalLight()</code>を呼び出す4つ目の方法は2つのパラメータを使用します。最初のパラメータ、<code>color</code>は、<a href="/reference/p5/p5.Color">p5.Color</a>オブジェクトまたは色の値の配列を使用してライトの色を設定します。2つ目のパラメータ、<code>direction</code>は、<a href="/reference/p5/p5.Color">p5.Color</a>オブジェクトを使用してライトの方向を設定します。例えば、<code>directionalLight(myColor, lightDir)</code>は、<code>myColor</code>の色値で<code>lightDir</code>ベクトルが指す方向に光を放つライトを作成します。</p>
