description: >
  <p>一方向に光を放つライトを作成します。</p>

  <p>指向性ライトは特定の点から光を放つわけではありません。画面外のどこかにある太陽のように
  振る舞います。ライトの方向は-1から1の間の3つの値<code>(x, y, z)</code>で設定されます。
  例えば、ライトの方向を<code>(1, 0, 0)</code>に設定すると、ライトが右方向を直接向いているため、
  <a href="/reference/p5/p5.Geometry">p5.Geometry</a>オブジェクトは左側から照らされます。
  一度に最大5つの指向性ライトを有効にできます。</p>

  <p>ライトの色と方向を設定するパラメータを使用して、<code>directionalLight()</code>を
  呼び出す方法は4つあります。</p>

  <p>最初の方法では、<code>directionalLight()</code>に6つのパラメータを使用します。
  最初の3つのパラメータ<code>v1</code>、<code>v2</code>、<code>v3</code>は、
  現在の<a href="/reference/p5/colorMode/">colorMode()</a>を使用してライトの色を設定します。
  残りの3つのパラメータ<code>x</code>、<code>y</code>、<code>z</code>は、ライトの方向を
  設定します。例えば、<code>directionalLight(255, 0, 0, 1, 0, 0)</code>は、
  赤色<code>(255, 0, 0)</code>で右方向<code>(1, 0, 0)</code>に光を放つライトを作成します。</p>

  <p>2番目の方法では、<code>directionalLight()</code>に4つのパラメータを使用します。
  最初の3つのパラメータ<code>v1</code>、<code>v2</code>、<code>v3</code>は、
  現在の<a href="/reference/p5/colorMode/">colorMode()</a>を使用してライトの色を設定します。
  最後のパラメータ<code>direction</code>は、<a href="/reference/p5/p5.Geometry">p5.Geometry</a>
  オブジェクトを使用してライトの方向を設定します。例えば、
  <code>directionalLight(255, 0, 0, lightDir)</code>は、<code>lightDir</code>ベクトルが
  指す方向に光を放つ赤色<code>(255, 0, 0)</code>のライトを作成します。</p>

  <p>3番目の方法では、<code>directionalLight()</code>に4つのパラメータを使用します。
  最初のパラメータ<code>color</code>は、<a href="/reference/p5/p5.Color">p5.Color</a>
  オブジェクトまたは色値の配列を使用してライトの色を設定します。残りの3つのパラメータ
  <code>x</code>、<code>y</code>、<code>z</code>は、ライトの方向を設定します。
  例えば、<code>directionalLight(myColor, 1, 0, 0)</code>は、<code>myColor</code>の
  色値で右方向<code>(1, 0, 0)</code>に光を放つライトを作成します。</p>

  <p>4番目の方法では、<code>directionalLight()</code>に2つのパラメータを使用します。
  最初のパラメータ<code>color</code>は、<a href="/reference/p5/p5.Color">p5.Color</a>
  オブジェクトまたは色値の配列を使用してライトの色を設定します。2番目のパラメータ
  <code>direction</code>は、<a href="/reference/p5/p5.Color">p5.Color</a>オブジェクトを
  使用してライトの方向を設定します。例えば、<code>directionalLight(myColor, lightDir)</code>は、
  <code>lightDir</code>ベクトルが指す方向に<code>myColor</code>の色値で光を放つライトを
  作成します。</p>
