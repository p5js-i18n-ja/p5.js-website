description: >
  <p>ファイルまたはURLの内容を読み込み、その値で<a
  href="/reference/p5/p5.Table">p5.Table</a>オブジェクトを作成します。

  ファイルが指定された場合、スケッチの「data」フォルダに配置されている必要があります。

  filenameパラメータには、オンラインで見つかるファイルへのURLも指定できます。

  デフォルトでは、ファイルはカンマ区切り（CSV形式）であると想定されます。

  'header'オプションが含まれている場合のみ、Tableはヘッダー行を探します。</p>

  <p>このメソッドは非同期であり、スケッチの次の行が実行される前に

  完了しない可能性があります。<a href="/reference/p5/preload/">preload()</a>内で

  <a href="/reference/p5/loadTable/">loadTable()</a>を呼び出すことで、

  <a href="/reference/p5/setup/">setup()</a>と<a href="/reference/p5/draw/">draw()</a>が呼び出される前に

  操作が完了することが保証されます。

  <a href="/reference/p5/preload/">preload()</a>の外部では、オブジェクトを処理するために

  コールバック関数を提供することができます：</p>

  <p>読み込まれて保存されるすべてのファイルはUTF-8エンコーディングを使用します。このメソッドは64MBまでのサイズのファイルを取得するのに適しています。</p>

example:
  - |-

    <div class='norender'>
    <code>
    // プロジェクトの "assets" フォルダにある
    // 以下のCSVファイル "mammals.csv" が与えられた場合：
    //
    // id,species,name
    // 0,Capra hircus,Goat
    // 1,Panthera pardus,Leopard
    // 2,Equus zebra,Zebra

    let table;

    function preload() {
      //私のテーブルはカンマ区切り値 "csv" で
      //列ラベルを指定するヘッダーがあります
      table = loadTable('/assets/mammals.csv', 'csv', 'header');
      //ファイルはリモートでも可能です
      //table = loadTable("https://p5js.org/reference//assets/mammals.csv",
      //                  "csv", "header");
    }

    function setup() {
      //列をカウントします
      print(table.getRowCount() + ' テーブル内の総行数');
      print(table.getColumnCount() + ' テーブル内の総列数');

      print(table.getColumn('name'));
      //["Goat", "Leopard", "Zebra"]

      //テーブルを巡回します
      for (let r = 0; r < table.getRowCount(); r++)
        for (let c = 0; c < table.getColumnCount(); c++) {
          print(table.getString(r, c));
        }
      describe(`ファイルからランダムに生成されたテキスト、
        例えば "私はバターの匂いがする"`);
    }
    </code>
