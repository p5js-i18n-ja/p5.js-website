  <p><a href="#/p5/sin">sin()</a> の逆関数で、値の逆正弦を返します。</p>

  <p>/// todo ///</p>

  <p>The inverse of <a href="#/p5/sin">sin()</a>, returns the arc sine of a
  value. This function expects input values in the range of -1 to 1. By default,
  <code>asin()</code> returns values in the range -π ÷ 2 (about -1.57) to π ÷ 2
  (about 1.57). If the <a href="#/p5/angleMode">angleMode()</a> is
  <code>DEGREES</code> then values are returned in the range -90 to 90.</p>


  <hr/>

  <p>Calculates the arc sine of a number.</p>

  <p><code>asin()</code> is the inverse of <a
  href="/reference/p5/sin">sin()</a>. It expects input

  values in the range of -1 to 1. By default, <code>asin()</code> returns values
  in the

  range -π ÷ 2 (about -1.57) to π ÷ 2 (about 1.57). If

  the <a href="/reference/p5/angleMode">angleMode()</a> is <code>DEGREES</code>
  then values are

  returned in the range -90 to 90.</p>

    function setup() {
      createCanvas(100, 100);

      background(200);

      // Calculate sin() and asin() values.
      let a = PI / 3;
      let s = sin(a);
      let as = asin(s);

      // Display the values.
      text(`${round(a, 3)}`, 35, 25);
      text(`${round(s, 3)}`, 35, 50);
      text(`${round(as, 3)}`, 35, 75);

    }
