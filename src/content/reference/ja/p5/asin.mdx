description: >
  <p>数値のアークサイン（逆正弦）を計算します。</p>

  <p><code>asin()</code>は<a href="/reference/p5/sin/">sin()</a>の逆関数です。-1から1の範囲の入力値を

  期待します。デフォルトでは、<code>asin()</code>は-π ÷ 2（約-1.57）から

  π ÷ 2（約1.57）の範囲の値を返します。<a href="/reference/p5/angleMode/">angleMode()</a>が

  <code>DEGREES</code>に設定されている場合、値は-90から90の範囲で返されます。</p>

example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // sin()とasin()の値を計算します。
      let a = PI / 3;
      let s = sin(a);
      let as = asin(s);

      // 値を表示します。
      text(`${round(a, 3)}`, 35, 25);
      text(`${round(s, 3)}`, 35, 50);
      text(`${round(as, 3)}`, 35, 75);

      describe('別々の行に書かれた数字1.047、0.866、1.047。');
    }
    </code>
    </div>

      let as = asin(s);

      // 値を表示します。

      text(`${round(s, 3)}`, 35, 50);
      text(`${round(as, 3)}`, 35, 75);

    }
