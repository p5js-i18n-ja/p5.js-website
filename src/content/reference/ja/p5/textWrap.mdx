description: |
  <p>テキストの折り返し方法を設定します。</p>
  <p>デフォルトでは、テキストは WORD で折り返されます。これは、単語の途中で
  折り返されないことを意味します。CHAR を使用すると、テキストは文字単位で
  折り返されます。</p>
  <p>textWrap() の設定は、text() 関数で描画されるすべてのテキストに影響します。</p>
line: 664
isConstructor: false
itemtype: method
example:
  - |-

    <div>
    <code>
    textSize(20);
    textAlign(CENTER);
    background(200);

    // デフォルトの折り返し（WORD）
    text('This text should wrap at the word', 50, 30, 100);

    push();
    textWrap(CHAR);
    fill(0, 0, 255);
    // 文字単位で折り返し
    text('This text should wrap at the char', 50, 115, 100);
    pop();

    describe('2つのテキストブロックが表示されています。上のブロックは単語単位で折り返され、下のブロックは文字単位で折り返されています。');
    </code>
    </div>
