description: >
  <p>コードのブロックを実行するかどうかを選択する方法。</p>

  <p><code>if</code>文は、条件に基づいてコードのブロックを実行するのに役立ちます。
  例えば、<code>if</code>文を使うと、「マウスが押されたら円を描く」という
  アイデアを簡単に表現できます：</p>

  <pre><code class="language-js">if (mouseIsPressed === true) {
    circle(mouseX, mouseY, 20);
  }

  </code></pre>

  <p>文のヘッダーは<code>if</code>キーワードで始まります。括弧内の式
  <code>mouseIsPressed === true</code>は、<code>true</code>または<code>false</code>の
  <code>Boolean</code>式です。中括弧<code>{}</code>の間のコードが
  if文の本体です。本体は<code>Boolean</code>式が<code>true</code>の場合にのみ実行されます。</p>

  <p><a href="/reference/p5/mouseIsPressed/">mouseIsPressed</a>システム変数は
  常に<code>true</code>または<code>false</code>なので、上記のコードスニペットは
  次のように書くこともできます：</p>

  <pre><code class="language-js">if (mouseIsPressed) {
    circle(mouseX, mouseY, 20);
  }

  </code></pre>

  <p><code>if</code>-<code>else</code>文は、<code>Boolean</code>式が<code>false</code>の場合に
  実行されるコードのブロックを追加します。例えば、以下は
  「マウスが押されたら円を描く。そうでなければメッセージを表示する。」
  というアイデアを表現する<code>if</code>-<code>else</code>文です：</p>

  <pre><code class="language-js">if (mouseIsPressed === true) {
    // trueの場合
    circle(mouseX, mouseY, 20);
  } else {
    // falseの場合
    text('クリックしてください!', 50, 50);
  }

  </code></pre>

  <p>このコードスニペットには2つの可能なパス、つまりブランチがあります。
  プログラムは必ずどちらかのブランチを実行します。</p>

  <p><code>else</code>-<code>if</code>文を使うと、さらにブランチを追加できます。
  <code>else</code>-<code>if</code>文は、異なる条件下で異なるコードブロックを
  実行します。例えば、<code>else</code>-<code>if</code>文を使うと、
  「マウスが左側にある場合はキャンバスを白く塗る。マウスが中央にある場合は
  キャンバスをグレーに塗る。それ以外の場合はキャンバスを黒く塗る。」
  というアイデアを簡単に表現できます：</p>

  <pre><code class="language-js">if (mouseX < 33) {
    background(255);
  } else if (mouseX < 67) {
    background(200);
  } else {
    background(0);
  }

  </code></pre>

  <p><code>if</code>文には必要な数だけ<code>else</code>-<code>if</code>文を追加できます。ただし、
  <code>else</code>文は1つだけで、最後に配置する必要があります。</p>

  <p><code>if</code>文は複数の条件を同時にチェックすることもできます。
  例えば、<code>Boolean</code>演算子<span class="code">&&</span>（AND）は2つの式が
  両方とも<code>true</code>かどうかをチェックします：</p>

  <pre><code class="language-js">if (keyIsPressed === true && key === 'p') {
    text('あなたは"p"キーを押しました!', 50, 50);
  }

  </code></pre>

  <p>ユーザーがキーを押していて、かつそのキーが<code>'p'</code>である場合、
  メッセージが表示されます。</p>

  <p><code>Boolean</code>演算子<code>||</code>（OR）は、2つの式のうち
  少なくとも1つが<code>true</code>かどうかをチェックします：</p>

  <pre><code class="language-js">if (keyIsPressed === true || mouseIsPressed === true) {
    text('何かしました!', 50, 50);
  }

  </code></pre>

  <p>ユーザーがキーを押すか、マウスボタンを押すか、またはその両方を行った場合、
  メッセージが表示されます。</p>

  <p><code>if</code>文の本体には、別の<code>if</code>文を含めることができます。
  これを「ネストされた<code>if</code>文」と呼びます。例えば、ネストされた<code>if</code>文を
  使うと、「キーが押された場合、そのキーが<code>'r'</code>かどうかをチェックする。
  もしそうなら、塗りつぶし色を赤に設定する。」というアイデアを
  簡単に表現できます：</p>

  <pre><code class="language-js">if (keyIsPressed === true) {
    if (key === 'r') {
      fill('red');
    }
  }

  </code></pre>

  <p>これらのデータ型とそれらがサポートする操作について詳しく学ぶには、
  <a href="/reference/p5/Boolean/">Boolean</a>と<a href="/reference/p5/Number/">Number</a>
  を参照してください。</p>
