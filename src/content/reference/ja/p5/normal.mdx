description: >
  <p>カスタム3D形状の頂点の法線ベクトルを設定します。</p>

  <p><a href="/reference/p5/beginShape/">beginShape()</a>と
  <a href="/reference/p5/endShape/">endShape()</a>で作成される3D形状は、
  頂点と呼ばれる点の集合を接続して作られます。
  <a href="/reference/p5/vertex/">vertex()</a>で追加される各頂点には、
  その頂点から外向きに向かう法線ベクトルがあります。
  法線ベクトルは、形状に対する光の反射の仕方を制御します。</p>

  <p><code>normal()</code>は、法線ベクトルのコンポーネントを定義するために、
  異なるパラメータを持つ2つの方法で呼び出すことができます。</p>

  <p><code>normal()</code>を呼び出す最初の方法は、<code>x</code>、<code>y</code>、<code>z</code>の
  3つのパラメータを持ちます。<code>normal(1, 2, 3)</code>のように数値が渡された場合、
  それらは法線ベクトルのx、y、z成分を設定します。</p>

  <p><code>normal()</code>を呼び出す2つ目の方法は、<code>vector</code>という1つのパラメータを
  持ちます。<code>normal(myVector)</code>のように<a href="/reference/p5/p5.Vector">p5.Vector</a>
  オブジェクトが渡された場合、そのコンポーネントが法線ベクトルの設定に使用されます。</p>

  <p><code>normal()</code>は、<a href="/reference/p5/vertex/">vertex()</a>でカスタム形状に
  追加される頂点の法線ベクトルを変更します。<code>normal()</code>は、
  <a href="/reference/p5/vertex/">vertex()</a>と同様に、
  <a href="/reference/p5/beginShape/">beginShape()</a>と
  <a href="/reference/p5/endShape/">endShape()</a>の間で呼び出す必要があります。
  <code>normal()</code>を呼び出すことで設定された法線ベクトルは、
  再び<code>normal()</code>が呼び出されるまで、以降のすべての頂点に影響します：</p>

  <pre><code class="language-js">beginShape();


  // 頂点の法線を設定します。

  normal(-0.4, -0.4, 0.8);


  // 頂点を追加します。

  vertex(-30, -30, 0);


  // 頂点の法線を設定します。

  normal(0, 0, 1);


  // 頂点を追加します。

  vertex(30, -30, 0);

  vertex(30, 30, 0);


  // 頂点の法線を設定します。

  normal(0.4, -0.4, 0.8);


  // 頂点を追加します。

  vertex(-30, 30, 0);


  endShape();

  </code></pre>
