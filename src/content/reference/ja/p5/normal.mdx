description: >
  <p>カスタム3D形状の頂点の法線ベクトルを設定します。</p>

  <p><a href="/reference/p5/beginShape/">beginShape()</a>と

  <a href="/reference/p5/endShape/">endShape()</a>で作成される3D形状は、

  頂点と呼ばれる一連の点を接続して作られます。

  <a href="/reference/p5/vertex/">vertex()</a>で追加される各頂点には、

  その頂点から離れる方向を指す法線ベクトルがあります。法線ベクトルは、

  形状に光がどのように反射するかを制御します。</p>

  <p><code>normal()</code>は、法線ベクトルのコンポーネントを定義するために

  異なるパラメータを持つ2つの方法で呼び出すことができます。</p>

  <p><code>normal()</code>を呼び出す最初の方法は、3つのパラメータ<code>x</code>、<code>y</code>、<code>z</code>を持ちます。

  <code>normal(1, 2, 3)</code>のように数値が渡された場合、それらは法線ベクトルの

  x、y、z成分を設定します。</p>

  <p><code>normal()</code>を呼び出す2番目の方法は、1つのパラメータ<code>vector</code>を持ちます。

  <code>normal(myVector)</code>のように<a href="/reference/p5/p5.Vector">p5.Vector</a>オブジェクトが

  渡された場合、そのコンポーネントが法線ベクトルの設定に使用されます。</p>

  <p><code>normal()</code>は、<a href="/reference/p5/vertex/">vertex()</a>でカスタム形状に

  追加される頂点の法線ベクトルを変更します。<code>normal()</code>は

  <a href="/reference/p5/beginShape/">beginShape()</a>と

  <a href="/reference/p5/endShape/">endShape()</a>の間で、

  <a href="/reference/p5/vertex/">vertex()</a>と同様に呼び出す必要があります。

  <code>normal()</code>を呼び出すことで設定される法線ベクトルは、

  <code>normal()</code>が再び呼び出されるまで、それ以降のすべての頂点に影響します。</p>

      createCanvas(100, 100, WEBGL);

    function draw() {
      background(200);

      // マウスでの軌道制御を有効にします。
      orbitControl();

      // 形状のスタイルを設定します。
      normalMaterial();
      noStroke();

      // p5.Vectorオブジェクトを作成します。
      let n1 = createVector(-0.4, -0.4, 0.8);
      let n2 = createVector(0, 0, 1);
      let n3 = createVector(0.4, -0.4, 0.8);

      // 形状を描画します。
      // normal()を使用して頂点の法線を設定します。
      beginShape();
      normal(n1);
      vertex(-30, -30, 0);

      normal(n2);
      vertex(30, -30, 0);
      vertex(30, 30, 0);

      normal(n3);
      vertex(-30, 30, 0);
      endShape();
    }
