description: >
  <p>マウスポインタを現在の位置にロックし、非表示にします。</p>

  <p><code>requestPointerLock()</code>を使用すると、マウスは画面から離れることなく
  永遠に動き続けることができます。<code>requestPointerLock()</code>を呼び出すと、
  <a href="/reference/p5/mouseX/">mouseX</a>、<a href="/reference/p5/mouseY/">mouseY</a>、
  <a href="/reference/p5/pmouseX/">pmouseX</a>、<a href="/reference/p5/pmouseY/">pmouseY</a>
  の値がロックされます。
  <a href="/reference/p5/movedX/">movedX</a>と<a href="/reference/p5/movedY/">movedY</a>
  は更新を続け、最後のフレームが描画されてからマウスが移動した距離を取得するのに
  使用できます。
  <a href="/reference/p5/exitPointerLock/">exitPointerLock()</a>を呼び出すと、
  マウスシステム変数の更新が再開されます。</p>

  <p>注意：ほとんどのブラウザでは、<code>requestPointerLock()</code>を呼び出す前に
  クリックなどの入力が必要です。<code>requestPointerLock()</code>は
  <a href="/reference/p5/doubleClicked/">doubleClicked()</a>のような
  イベント関数内で呼び出すことをお勧めします。</p>

      textAlign(CENTER);

      text(`Score: ${score}`, 50, 50);
    }

    // ユーザーがダブルクリックしたときにポインタをロック
    function doubleClicked() {
      requestPointerLock();
    }
