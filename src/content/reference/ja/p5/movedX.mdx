description: >
  <p><code>Number</code>型のシステム変数で、マウスの水平方向の動きを追跡します。</p>

  <p><code>movedX</code>は、フレーム間でマウスが左右に動いたピクセル数を追跡します。
  <code>movedX</code>は、フレーム間でマウスが左に動いた場合は負の値を、
  右に動いた場合は正の値を持ちます。<code>movedX</code>は
  <code>mouseX - pmouseX</code>として計算できます。</p>

  <p>注意：<code>movedX</code>は
  <a href="/reference/p5/requestPointerLock/">requestPointerLock()</a>が
  アクティブな場合でも更新し続けます。</p>

    function setup() {
      createCanvas(100, 100);

      describe(
        'グレーの正方形。ユーザーがマウスを右に動かすと ">>" というテキストが表示されます。ユーザーがマウスを左に動かすと "<<" というテキストが表示されます。'
      );
    }

    function draw() {
      background(200);

      // テキストのスタイルを設定します。
      textAlign(CENTER);
      textSize(16);

      // movedXが正の場合は >> を表示し、
      // 負の場合は << を表示します。
      if (movedX > 0) {
        text('>>', 50, 50);
      } else if (movedX < 0) {
        text('<<', 50, 50);
      }
    }
