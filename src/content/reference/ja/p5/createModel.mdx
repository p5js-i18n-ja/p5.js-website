description: >
  <p>OBJまたはSTL文字列から3Dモデルを読み込みます。</p>

  <p>OBJとSTLファイルには組み込みのスケール感がないため、異なるプログラムからエクスポートされたモデルのサイズが異なる場合があります。

  モデルが正しく表示されない場合は、<code>loadModel()</code>を<code>normalize</code>を<code>true</code>に設定して使用し、
  サイズを標準化することを検討してください。

  さらなる調整は<code>scale()</code>関数を使用して行うことができます。</p>

  <p>また、カラーSTLファイルのサポートは存在しません。色付きのSTLファイルは

  色のプロパティなしでレンダリングされます。</p>

  <ul>

  <li>オプションには以下が含まれます：</li>

  </ul>

  <ul>

  <li><code>modelString</code>：読み込むSTLまたはOBJファイルのプレーンテキスト文字列を指定します。</li>

  <li><code>fileType</code>：モデルのファイル拡張子を定義します。</li>

  <li><code>normalize</code>：trueに設定すると、読み込み中に標準化されたサイズスケーリングを有効にします。</li>

  <li><code>successCallback</code>：3Dモデルオブジェクトを使用して読み込み後のアクションを行うコールバック。</li>

  <li><code>failureCallback</code>：モデルの読み込みに失敗した場合のエラーを処理し、イベントエラーを受け取ります。</li>

  <li><code>flipU</code>：モデルのUテクスチャ座標を反転します。</li>

  <li><code>flipV</code>：モデルのVテクスチャ座標を反転します。</li>

  </ul>

    function setup() {

    }

    function draw() {
      background(200);
      rotateX(frameCount * 0.01);
      rotateY(frameCount * 0.01);
