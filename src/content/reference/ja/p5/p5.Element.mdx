  <p><a

  target="_blank">HTML要素</a>を記述するクラスです。</p>

  <p>スケッチは多くの要素を使用できます。一般的な要素には、描画キャンバス、
  ボタン、スライダー、ウェブカメラフィードなどがあります。</p>

  <p>すべての要素は<code>p5.Element</code>クラスのメソッドを共有しています。
  これらは<a href="/reference/p5/createCanvas/">createCanvas()</a>や
  <a href="/reference/p5/createButton/">createButton()</a>などの関数で
  作成されます。</p>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // ボタン要素を作成し、
      // キャンバスの下に配置します。
      let btn = createButton('change');
      btn.position(0, 100);

      // ボタンが押されたときに
      // randomColor()を呼び出します。
      btn.mousePressed(randomColor);

      describe('下に"change"と書かれたボタンがある灰色の正方形。ユーザーがボタンを押すと正方形の色が変わります。');
    }

    // 背景を赤、黄、青、緑の
    // いずれかの色で塗ります。
    function randomColor() {
      let c = random(['red', 'yellow', 'blue', 'green']);
      background(c);
    }
    </code>

      <p>例えば、<code><div></div></code>要素は、ヘッダーと段落という2つのテキストを

      最後に、<code>parent</code>は<code>HTMLElement</code>
      オブジェクトにすることもでき、<code>myElement.parent(anotherElement)</code>のようになります。</p>

      <p>引数なしで<code>myElement.parent()</code>を呼び出すと、要素の
      親を返します。</p>

    description: >
      <p>要素の幅と高さを設定します。</p>

      <p>引数なしで<code>myElement.size()</code>を呼び出すと、要素のサイズを
      <code>width</code>と<code>height</code>プロパティを持つオブジェクトとして返します。
      例えば、<code>{ width: 20, height: 10 }</code>のようになります。</p>
      <p>最初のパラメータ<code>width</code>はオプションです。要素の幅を設定するために
      使用される数値です。<code>myElement.size(10)</code>を呼び出すと</p>

      <p>2番目のパラメータ<code>height</code>もオプションです。要素の高さを設定するために
      使用される数値です。例えば、<code>myElement.size(20, 10)</code>を呼び出すと、
      要素の幅を20ピクセル、高さを10ピクセルに設定します。</p>

      <p>定数<code>AUTO</code>を使用すると、アスペクト比（<code>width / height</code>）を
      維持しながら、一度に1つの次元を調整できます。例えば、
      幅が200ピクセル、高さが100ピクセルの要素を考えてみましょう。
      <code>myElement.size(20, AUTO)</code>を呼び出すと、幅を20ピクセル、高さを10ピクセルに
      設定します。</p>

      <p>注意：画像などのデータをロードする必要がある要素の場合、
      データがロードされた後に<code>myElement.size()</code>を呼び出すまで待ってください。</p>
