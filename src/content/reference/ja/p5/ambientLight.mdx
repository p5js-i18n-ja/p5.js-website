description: >
  <p>あらゆる方向から照らす光を作成します。</p>

  <p>環境光は一方向からは来ません。代わりに、3Dの形状は

  すべての側面から均等に照らされます。環境光はほぼ常に

  他の種類の光と組み合わせて使用されます。</p>

  <p>光の色を設定するオプションのパラメータを使用して、<code>ambientLight()</code>を呼び出す方法は3つあります。</p>

  <p><code>ambientLight()</code>を呼び出す最初の方法には、<code>gray</code>と

  <code>alpha</code>の2つのパラメータがあります。<code>alpha</code>はオプションです。0から255の間の

  グレースケールと透明度の値を渡して環境光の色を設定できます。例えば、
  <code>ambientLight(50)</code>や

  <code>ambientLight(50, 30)</code>のようになります。</p>

  <p><code>ambientLight()</code>を呼び出す2番目の方法には、colorという1つのパラメータがあります。

  <a href="/reference/p5/p5.Color">p5.Color</a>オブジェクト、色の値の配列、または

  CSSの色文字列を渡して環境光の色を設定できます。例えば、<code>ambientLight('magenta')</code>のようになります。</p>

  <p><code>ambientLight()</code>を呼び出す3番目の方法には、<code>v1</code>、<code>v2</code>、

  <code>v3</code>、および<code>alpha</code>の4つのパラメータがあります。<code>alpha</code>はオプションです。RGBA、

  HSBA、またはHSLAの値を渡して環境光の色を設定できます。例えば、<code>ambientLight(255, 0, 0)</code>

  や<code>ambientLight(255, 0, 0, 30)</code>のようになります。色の値は現在の

  <a href="/reference/p5/colorMode/">colorMode()</a>を使用して解釈されます。</p>

    // クリックしてドラッグすると、異なる角度からシーンを見ることができます。
    // キャンバスをダブルクリックすると、光が点灯します。

    let isLit = false;

    function setup() {
      createCanvas(100, 100, WEBGL);

      describe('灰色の背景に対して描かれた球体。ユーザーがダブルクリックすると、球体の色が変わるように見えます。');
    }

    function draw() {
      background(200);

      // マウスでの回転を有効にします。
      orbitControl();

      // 光を制御します。
      if (isLit === true) {
        // グレースケール値80を使用します。
        ambientLight(80);
      }

      // 球体を描画します。
      sphere(30);
    }

    // ユーザーがダブルクリックすると環境光を点灯します。
    function doubleClicked() {
      isLit = true;
    }
    </code>
