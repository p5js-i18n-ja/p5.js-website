description: >
  <p>テキストのフォントを設定します。</p>

  <p>最初のパラメータ<code>font</code>は、<a href="/reference/p5/loadFont">loadFont()</a>で
  作成された<a href="/reference/p5/p5.Font">p5.Font</a>オブジェクト、または文字列でなければなりません。</p>

  <p><code>font</code>が文字列の場合、システムフォントの名前を指定します。
  例えば、'Arial'や'Helvetica'などです。使用可能なシステムフォントは、
  オペレーティングシステムによって異なります。</p>

  <p>2番目のパラメータ<code>size</code>はオプションです。これは、フォントサイズをピクセル単位で
  設定します。<code>size</code>が指定されない場合、現在のテキストサイズが使用されます。</p>

  <p><code>textFont()</code>を引数なしで呼び出すと、現在のフォントの名前を返します。</p>
line: 12
isConstructor: false
itemtype: method
example:
  - |-

    <div>
    <code>
    let myFont;

    function preload() {
      myFont = loadFont('/assets/inconsolata.otf');
    }

    function setup() {
      createCanvas(100, 100);
      background(200);
      textFont(myFont);
      textSize(20);
      text('こんにちは', 10, 30);
      describe('灰色の背景に黒で「こんにちは」と書かれています。テキストはInconsolataフォントで表示されています。');
    }
    </code>
    </div>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);
      background(200);
      textFont('Courier New');
      textSize(20);
      text('こんにちは', 10, 30);
      describe('灰色の背景に黒で「こんにちは」と書かれています。テキストはCourier Newフォントで表示されています。');
    }
    </code>
    </div>
