description: |
  <p>テキストの水平方向と垂直方向の配置を設定します。
  水平方向の配置には
  LEFT、CENTER、または RIGHT を使用します。垂直方向の配置には
  TOP、BOTTOM、CENTER、または BASELINE を使用します。</p>

  <p>textAlign() の2番目のパラメータは省略可能です。両方のパラメータが
  指定されていない場合、textAlign(LEFT) と同じになります。</p>

  <p>textAlign() の現在の設定は、text() 関数で描画されるすべてのテキストに
  影響します。配置は、text() に渡される x と y パラメータを基準に
  設定されます。</p>
line: 13
isConstructor: false
itemtype: method
example:
  - |-

    <div>
    <code>
    textSize(16);
    textAlign(RIGHT);
    text('ABCD', 50, 30);
    textAlign(CENTER);
    text('EFGH', 50, 50);
    textAlign(LEFT);
    text('IJKL', 50, 70);
    describe('文字列"ABCD"が右揃え、"EFGH"が中央揃え、"IJKL"が左揃えで表示されています。');
    </code>
    </div>

    <div>
    <code>
    textSize(16);
    strokeWeight(0.5);

    // 中央揃え
    textAlign(CENTER);
    const centerX = 50;
    line(centerX, 0, centerX, 100);
    text('center', centerX, 30);

    // 中央揃え（水平）、トップ揃え（垂直）
    textAlign(CENTER, TOP);
    text('center/top', centerX, 60);

    // 中央揃え（水平）、ボトム揃え（垂直）
    textAlign(CENTER, BOTTOM);
    text('center/bottom', centerX, 90);
    describe('3つの文字列が中央に垂直線で揃えられています。最初の文字列は中央揃え、2番目は上揃え、3番目は下揃えです。');
    </code>
    </div>
