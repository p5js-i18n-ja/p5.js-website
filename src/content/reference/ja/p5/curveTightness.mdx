  <p><a href="/reference/p5/curve/">curve()</a>と

  <a href="/reference/p5/curveVertex/">curveVertex()</a>の描画方法を調整します。</p>

  <p>スプライン曲線は、一連の点に取り付けられたケーブルのようなものです。

  <code>curveTightness()</code>は、ケーブルが点にどれだけ密着しているかを調整します。</p>

  <p>パラメータ<code>tightness</code>は、曲線が頂点にどのようにフィットするかを決定します。

  デフォルトでは、<code>tightness</code>は0に設定されています。<code>curveTightness(1)</code>のように

  タイトネスを1に設定すると、曲線の点が直線で接続されます。-5から5の範囲の値は、

  曲線を認識可能な状態に保ちながら変形させます。</p>

example:
  - |-

    <div>
    <code>
    // マウスを左右に動かして曲線の変化を確認してください。

    function setup() {
      createCanvas(100, 100);

      describe('黒い曲線が横向きのU字形を形成します。ユーザーがマウスを左右に動かすと曲線が変形します。');
    }

    function draw() {
      background(200);

      // マウスを使用して曲線のタイトネスを設定します。
      let t = map(mouseX, 0, 100, -5, 5, true);
      curveTightness(t);

      // 曲線を描画します。
      noFill();
      beginShape();
      curveVertex(10, 26);
      curveVertex(10, 26);
      curveVertex(83, 24);
      curveVertex(83, 61);
      curveVertex(25, 65);
      curveVertex(25, 65);
      endShape();
    }
    </code>
