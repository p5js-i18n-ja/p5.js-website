description: >
  <p>フルスクリーンモードを切り替えるか、現在のモードを返します。</p>

  <p><code>fullscreen(true)</code>を呼び出すとスケッチをフルスクリーンにします。

  <code>fullscreen(false)</code>を呼び出すとスケッチを元のサイズに戻します。</p>

  <p>引数なしで<code>fullscreen()</code>を呼び出すと、スケッチが

  フルスクリーンモードの場合は<code>true</code>を、そうでない場合は<code>false</code>を返します。</p>

  <p>注意：ブラウザの制限により、<code>fullscreen()</code>はマウスクリックなどの

  ユーザー入力でのみ呼び出すことができます。</p>

isConstructor: false

example:
  - |-

    <div>
    <code>
    function setup() {
      background(200);

      describe('クリックするとデフォルト表示とフルスクリーン表示を切り替えるグレーのキャンバス。');
    }

    // マウスが押されたら、
    // フルスクリーンモードを切り替えます。
    function mousePressed() {
      if (mouseX > 0 && mouseX < width && mouseY > 0 && mouseY < height) {
        let fs = fullscreen();
        fullscreen(!fs);
      }
    }
    </code>
    </div>
