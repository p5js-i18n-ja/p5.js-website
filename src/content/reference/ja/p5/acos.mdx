  <p>数値のアークコサインを計算します。</p>

  <p><code>acos()</code>は<a href="/reference/p5/cos/">cos()</a>の逆関数です。-1から1の範囲の
  引数を期待します。デフォルトでは、<code>acos()</code>は0からπ（約3.14）の範囲の値を
  返します。<a href="/reference/p5/angleMode/">angleMode()</a>が<code>DEGREES</code>の場合、
  値は0から180の範囲で返されます。</p>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // cos()とacos()の値を計算します。
      let a = PI;
      let c = cos(a);
      let ac = acos(c);

      // 値を表示します。
      text(`${round(a, 3)}`, 35, 25);
      text(`${round(c, 3)}`, 35, 50);
      text(`${round(ac, 3)}`, 35, 75);

      describe('3.142、-1、3.142の数字が別々の行に書かれています。');
    }
