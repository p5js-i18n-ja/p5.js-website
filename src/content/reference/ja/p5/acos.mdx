  <p>The inverse of <a href="#/p5/cos">cos()</a>, returns the arc cosine of a
  value. This function expects arguments in the range -1 to 1. By default,
  <code>acos()</code> returns values in the range 0 to π (about 3.14). If the <a
  href="#/p5/angleMode">angleMode()</a> is <code>DEGREES</code>, then values are
  returned in the range 0 to 180.</p>

  <p>Calculates the arc cosine of a number.</p>

  <p><code>acos()</code> is the inverse of <a

  arguments in the range -1 to 1. By default, <code>acos()</code> returns values
  in the

  range 0 to π (about 3.14). If the

  <a href="/reference/p5/angleMode/">angleMode()</a> is <code>DEGREES</code>,
  then values are

  returned in the range 0 to 180.</p>
