  <p>ブラウザウィンドウがリサイズされたときに呼び出される関数です。</p>

  <p><code>windowResized()</code>の本体に配置されたコードは、ブラウザウィンドウのサイズが

  変更されたときに実行されます。新しいウィンドウサイズに対応するために

  <a href="/reference/p5/resizeCanvas/">resizeCanvas()</a>を呼び出したり、

  その他の調整を行うのに適した場所です。</p>

  <p><code>event</code>パラメータはオプションです。関数宣言に追加すると、

  デバッグやその他の目的で使用できます。</p>

    function setup() {
