description: >
  <p>クリッピング領域を定義します。</p>

  <p><code>clip()</code>の後に描画されたすべてのものは、指定された形状内にクリップされます。

  クリッピング領域は、<code>clip()</code>に渡された形状によって定義されます。</p>

  <p>クリッピング領域は、<code>noClip()</code>が呼び出されるまで有効です。</p>

  <p><code>clip()</code>は、<code>push()</code>と<code>pop()</code>の間で使用する必要があります。</p>

example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);
    }

    function draw() {
      background(220);

      push();
      clip(50, 50, 40);

      // この長方形は円の内側にのみ表示されます
      fill('red');
      rect(0, 0, 100, 100);
      pop();

      describe('赤い円。円は長方形でクリップされています。');
    }
    </code>
    </div>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);
    }

    function draw() {
      background(220);

      push();
      clip(25, 25, 50, 50);

      // この円は正方形の内側にのみ表示されます
      fill('blue');
      circle(50, 50, 80);
      pop();

      describe('青い正方形。正方形は円でクリップされています。');
    }
    </code>
    </div>
