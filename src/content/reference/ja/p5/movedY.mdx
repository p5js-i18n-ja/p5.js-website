description: >
  <p><code>Number</code>型のシステム変数で、マウスの垂直方向の動きを追跡します。</p>

  <p><code>movedY</code>は、フレーム間でマウスが上下に動いたピクセル数を追跡します。
  <code>movedY</code>は、フレーム間でマウスが上に動いた場合は負の値を、
  下に動いた場合は正の値を持ちます。<code>movedY</code>は
  <code>mouseY - pmouseY</code>として計算できます。</p>

  <p>注意：<code>movedY</code>は
  <a href="/reference/p5/requestPointerLock/">requestPointerLock()</a>が
  アクティブな場合でも更新し続けます。</p>

    function setup() {
      createCanvas(100, 100);

      describe(

      }
