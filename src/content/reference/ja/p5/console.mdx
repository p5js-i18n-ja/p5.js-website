  <p>ウェブブラウザのコンソールにメッセージを出力します。</p>

  <p><a href="https://developer.mozilla.org/en-US/docs/Web/API/console/"
  target="_blank">console</a>

  オブジェクトは、デバッグ中にメッセージを出力するのに役立ちます。例えば、

  コードのセクションがどのように動作するかを調べる際に、<code>console.log()</code>文を

  追加するのが一般的です：</p>

  <pre><code class="language-js">if (isPlaying === true) {
    // このコードブロックが実行されることを確認するために、console.log()文を追加します。
    console.log('ここまで来ました！');

    // ゲームのロジック
  }

  </code></pre>

  <p><code>console.error()</code>は、フォーマットされたメッセージを出力するため、

  エラーの追跡に役立ちます。例えば、メディアアセットの読み込み時にエラーに

  遭遇することがよくあります：</p>

  <pre><code class="language-js">// 特別なフォーマットでエラーメッセージを記録します。

  function handleFailure(error) {
    console.error('おっと！', error);
  }


  // 画像の読み込みを試み、失敗した場合はhandleError()を呼び出します。

  loadImage('https://example.com/cat.jpg', handleImage, handleError);

  </code></pre>
