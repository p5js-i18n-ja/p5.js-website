  <p>現在のアニメーションを gif として生成し、コンピューターにダウンロードします！。</p>

  <p>duration 引数は、アニメーションから録画したい秒数を指定します。この値は、units
  の値に応じて、生成する必要があるフレーム数に変換されます。次の段落で詳しく説明します。</p>

  <p>さらに2つの引数を含むオプションのオブジェクトです:delay（number）および units（string）。</p>

  <p><code>delay</code> は、録画を開始する前に待つ時間を指定します。</p>

  <p><code>units</code>
  は、'seconds'または'frames'のいずれかの文字列です。デフォルトでは'seconds'です。</p>

  <p><code>units</code> は、duration と delay
  引数の動作方法を指定します。'seconds'の場合、これらの引数は秒に対応し、3秒間のアニメーションが作成されることを意味します。'frames'の場合、引数はアニメーションのフレーム数に対応し、この数値が非常に確実な場合に限ります。</p>

  <p>これは、setup で呼び出すことも、以下の例のように、keyPressed や mousePressed
  などのイベント関数内で呼び出すこともできます。</p>

  <p>Generates a gif from a sketch and saves it to a file.
  <code>saveGif()</code> may be called in <a href="#/p5/setup">setup()</a> or at
  any point while a sketch is running.</p>

  <p>The first parameter, <code>fileName</code>, sets the gif's file name. The
  second parameter, <code>duration</code>, sets the gif's duration in
  seconds.</p>

  <p>The third parameter, <code>options</code>, is optional. If an object is
  passed, <code>saveGif()</code> will use its properties to customize the gif.
  <code>saveGif()</code> recognizes the properties <code>delay</code>,
  <code>units</code>, <code>silent</code>, <code>notificationDuration</code>,
  and <code>notificationID</code>.</p>
