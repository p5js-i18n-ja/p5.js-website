description: >
  <p>キャンバス上の図形のスクリーンリーダーでアクセス可能な説明を作成します。</p>

  <p><code>textOutput()</code>は、一般的な説明、図形のリスト、および

  図形の表をウェブページに追加します。一般的な説明には、

  キャンバスのサイズ、キャンバスの色、図形の数が含まれます。例えば、

  <code>出力は、100 x 100ピクセルの、グレーのキャンバスで、以下の2つの図形を含んでいます：</code>。</p>

  <p>図形のリストが一般的な説明に続きます。リストは各図形の

  色、位置、面積を説明します。例えば、

  <code>キャンバスの3%を占める中央の赤い円</code>。各図形を

  選択してさらに詳細を得ることができます。</p>

  <p><code>textOutput()</code>は図形の表をリストとして使用します。表は

  図形、色、位置、座標、面積を説明します。例えば、

  <code>赤い円 位置 = 中央 面積 = 3%</code>。これは

  <a href="/reference/p5/gridOutput/">gridOutput()</a>とは異なり、gridOutputは表をグリッドとして使用します。</p>

  <p><code>display</code>パラメータはオプションです。これは説明の

  表示方法を決定します。<code>LABEL</code>が渡された場合、例えば<code>textOutput(LABEL)</code>のように、

  説明はキャンバスの隣のdiv要素に表示されます。<code>LABEL</code>を使用すると

  スクリーンリーダーにとって役に立たない重複が作成されます。<code>LABEL</code>は開発中にのみ

  使用してください。<code>FALLBACK</code>が渡された場合、例えば<code>textOutput(FALLBACK)</code>のように、

  説明はスクリーンリーダーにのみ表示されます。これがデフォルトの

  モードです。</p>

  <p>

  <a href="/learn/accessible-labels.html">アクセシブルなキャンバスの説明を書く</a>

  を読んで、スケッチをアクセシブルにする方法についてさらに学んでください。</p>
