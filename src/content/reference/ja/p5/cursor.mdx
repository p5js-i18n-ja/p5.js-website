  <p>Changes the cursor's appearance.</p>

  <p>The first parameter, <code>type</code>, sets the type of cursor to display.
  The built-in options are <code>ARROW</code>, <code>CROSS</code>,
  <code>HAND</code>, <code>MOVE</code>, <code>TEXT</code>, and
  <code>WAIT</code>. <code>cursor()</code> also recognizes standard CSS cursor
  properties passed as strings: <code>'help'</code>, <code>'wait'</code>,
  <code>'crosshair'</code>, <code>'not-allowed'</code>, <code>'zoom-in'</code>,
  and <code>'grab'</code>. If the path to an image is passed, as in
  <code>cursor('assets/target.png')</code>, then the image will be used as the
  cursor. Images must be in .cur, .gif, .jpg, .jpeg, or .png format.</p>

  <p>The parameters <code>x</code> and <code>y</code> are optional. If an image
  is used for the cursor, <code>x</code> and <code>y</code> set the location
  pointed to within the image. They are both 0 by default, so the cursor points
  to the image's top-left corner. <code>x</code> and <code>y</code> must be less
  than the image's width and height, respectively.</p>

      background(200);

      // Divide the canvas into quadrants.
      line(50, 0, 50, 100);
      line(0, 50, 100, 50);

      // Change cursor based on mouse position.
      if (mouseX < 50 && mouseY < 50) {
        cursor(CROSS);
      } else if (mouseX > 50 && mouseY < 50) {
        cursor('progress');
      } else if (mouseX > 50 && mouseY > 50) {
        cursor('https://avatars0.githubusercontent.com/u/1617169?s=16');
      } else {
        cursor('grab');
      }
    }
