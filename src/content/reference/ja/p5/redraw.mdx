description: >
  <p><a href="/reference/p5/draw/">draw()</a>内のコードを1回実行します。</p>

  <p>デフォルトでは、<a href="/reference/p5/draw/">draw()</a>は1秒間に60回実行しようとします。

  <a href="/reference/p5/noLoop/">noLoop()</a>を呼び出すと、

  <a href="/reference/p5/draw/">draw()</a>の繰り返しが停止します。<code>redraw()</code>を呼び出すと、

  <a href="/reference/p5/draw/">draw()</a>関数内のコードが指定された

  回数だけ実行されます。</p>

  <p>パラメータ<code>n</code>はオプションです。<code>redraw(5)</code>のように数値が渡された場合、

  drawループは指定された回数だけ実行されます。デフォルトでは、<code>n</code>は

  1です。</p>
