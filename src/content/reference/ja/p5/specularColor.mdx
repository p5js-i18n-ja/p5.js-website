description: >
  <p>ライトの鏡面反射色を設定します。</p>

  <p><code>specularColor()</code>は、表面から特定の方向に反射するライトに影響します。
  これらのライトには
  <a href="/reference/p5/directionalLight/">directionalLight()</a>、
  <a href="/reference/p5/pointLight/">pointLight()</a>、
  <a href="/reference/p5/spotLight/">spotLight()</a>が含まれます。
  この関数は、<a href="/reference/p5/specularMaterial/">specularMaterial()</a>でスタイリングされた
  <a href="/reference/p5/p5.Geometry">p5.Geometry</a>オブジェクトにハイライトを作成するのに役立ちます。
  ジオメトリが<a href="/reference/p5/specularMaterial/">specularMaterial()</a>を使用していない場合、
  <code>specularColor()</code>は効果がありません。</p>

  <p>注意：<code>specularColor()</code>は、<a href="/reference/p5/ambientLight/">ambientLight()</a>や
  <a href="/reference/p5/imageLight/">imageLight()</a>など、
  あらゆる方向に反射するライトには影響しません。</p>

  <p><code>specularColor()</code>を呼び出すには、オプションのパラメータを使用して
  鏡面反射ハイライトの色を設定する3つの方法があります。</p>

  <p>最初の方法は、<code>gray</code>と<code>alpha</code>という2つのオプションパラメータを持つ
  <code>specularColor()</code>の呼び出しです。
  <code>specularColor(50)</code>や<code>specularColor(50, 80)</code>のように、
  0から255の間のグレースケールと透明度の値を渡して、鏡面反射ハイライトの色を設定できます。</p>

  <p>2つ目の方法は、<code>color</code>という1つのオプションパラメータを持つ
  <code>specularColor()</code>の呼び出しです。
  <a href="/reference/p5/p5.Color">p5.Color</a>オブジェクト、色の値の配列、
  またはCSS色文字列を渡して、鏡面反射ハイライトの色を設定できます。</p>

  <p>3つ目の方法は、<code>v1</code>、<code>v2</code>、<code>v3</code>、<code>alpha</code>という
  4つのオプションパラメータを持つ<code>specularColor()</code>の呼び出しです。
  <code>specularColor(255, 0, 0, 80)</code>のように、RGBA、HSBA、またはHSLAの値を渡して、
  鏡面反射ハイライトの色を設定できます。色の値は、現在の
  <a href="/reference/p5/colorMode/">colorMode()</a>を使用して解釈されます。</p>
