description: >
  <p><code>Boolean</code>または<code>Number</code>を<code>String</code>に変換します。</p>

  <p><code>str()</code>は値を文字列に変換します。テンプレートリテラルの使用に関するガイダンスについては、

  <a href="/reference/p5/String/">String</a>のリファレンスページを参照してください。</p>

  <p>パラメータ<code>n</code>は変換する値です。<code>n</code>がBooleanの場合、例えば

  <code>str(false)</code>または<code>str(true)</code>のように、値は文字列として返されます。

  つまり<code>'false'</code>または<code>'true'</code>となります。<code>n</code>が数値の場合、例えば<code>str(123)</code>のように、その

  値は文字列として返されます。つまり<code>'123'</code>となります。配列が渡された場合、

  例えば<code>str([12.34, 56.78])</code>のように、文字列の配列が返されます。</p>

    function setup() {
      createCanvas(100, 100);

      background(200);

      // 数値の配列を作成します。
      let original = [12, 34, 56];

      // 数値を文字列に変換します。
      let strings = str(original);

      // 空の文字列変数を作成します。
      let final = '';

      // すべての文字列を連結します。
      for (let s of strings) {
        final += s;
      }

      // テキストのスタイルを設定します。
      textAlign(CENTER, CENTER);
      textSize(16);

      // 連結された文字列を表示します。
      text(final, 50, 50);

      describe('グレーの背景に黒で "123456" というテキストが書かれています。');
    }
    </code>
