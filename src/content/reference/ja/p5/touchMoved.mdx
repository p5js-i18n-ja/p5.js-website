  <p>ユーザーが画面に触れて動かしたときに呼び出される関数です。</p>

  <p><code>touchMoved()</code>関数を宣言すると、ユーザーがタッチスクリーンデバイスに触れて動かしたときに

  自動的に実行されるコードブロックを設定できます：</p>

  <pre><code class="language-js">function touchMoved() {
    // 実行するコード
  }

  </code></pre>

  <p><code>touchMoved()</code>がp5.jsによって呼び出されると、<a href="/reference/p5/touches/">touches</a>配列が

  最新のタッチポイントで更新されます：</p>

  <pre><code class="language-js">function touchMoved() {
    // 背景を塗りつぶす
    background(200);

    // ユーザーが動かしている間、各タッチポイントをマークする
    for (let touch of touches) {
      circle(touch.x, touch.y, 40);
    }
  }

  </code></pre>

  <p>パラメータのeventはオプションです。<code>touchMoved()</code>には

  タッチイベントを説明するプロパティを持つ<a href="https://developer.mozilla.org/en-US/docs/Web/API/TouchEvent/"
  target="_blank">TouchEvent</a>

  オブジェクトが渡されます：</p>

  <pre><code class="language-js">function touchMoved(event) {
    // イベントを使用するコード
    console.log(event);
  }

  </code></pre>

  <p>タッチスクリーンデバイスでは、<code>touchMoved()</code>が宣言されていない場合、

  ユーザーのタッチポイントが移動すると<a href="/reference/p5/mouseDragged/">mouseDragged()</a>が

  実行されます。<code>touchMoved()</code>が宣言されている場合、ユーザーの

  タッチポイントが移動すると<code>touchMoved()</code>が実行され、<a href="/reference/p5/mouseDragged/">mouseDragged()</a>

  は実行されません。</p>

  <p>注意：<a href="/reference/p5/touchStarted/">touchStarted()</a>、

  <a href="/reference/p5/touchEnded/">touchEnded()</a>、

  <code>touchMoved()</code>はすべて関連しています。

  <a href="/reference/p5/touchStarted/">touchStarted()</a>はユーザーが

  タッチスクリーンデバイスに触れた瞬間に実行されます。<a href="/reference/p5/touchEnded/">touchEnded()</a>

  はユーザーがタッチを終了した瞬間に実行されます。<code>touchMoved()</code>は

  ユーザーがタッチポイントを動かしている間、繰り返し実行されます。</p>
