description: >
  <p>マウスボタンが離されたときに一度呼び出される関数です。</p>

  <p><code>mouseReleased()</code>関数を宣言すると、ユーザーがマウスボタンを押した後に

  離したときに自動的に実行されるコードブロックを設定します：</p>

  <pre><code class="language-js">function mouseReleased() {
    // 実行するコード
  }

  </code></pre>

  <p><a href="/reference/p5/mouseX/">mouseX</a>や

  <a href="/reference/p5/mouseY/">mouseY</a>などのマウスシステム変数は、

  p5.jsによって<code>mouseReleased()</code>が呼び出されたときに

  最新の値で更新されます：</p>

  <pre><code class="language-js">function mouseReleased() {
    if (mouseX < 50) {
      // マウスが左側にある場合に実行するコード
    }

    if (mouseY > 50) {
      // マウスが下部近くにある場合に実行するコード
    }
  }

  </code></pre>

  <p>パラメータ<code>event</code>はオプションです。
  <code>mouseReleased()</code>には常に

  マウスリリースイベントを説明するプロパティを持つ

  <a href="https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent/"
  target="_blank">MouseEvent</a>

  オブジェクトが渡されます：</p>

  <pre><code class="language-js">function mouseReleased(event) {
    // イベントを使用して実行するコード
    console.log(event);
  }

  </code></pre>

  <p>タッチスクリーンデバイスでは、<a href="/reference/p5/touchEnded/">touchEnded()</a>が

  宣言されていない場合、ユーザーのタッチが終了したときに<code>mouseReleased()</code>が

  実行されます。<a href="/reference/p5/touchEnded/">touchEnded()</a>が宣言されている場合、

  ユーザーのタッチが終了したときに<a href="/reference/p5/touchEnded/">touchEnded()</a>が

  実行され、<code>mouseReleased()</code>は実行されません。</p>

  <p>ブラウザは様々なマウスイベントにデフォルトの動作を関連付けている場合があります。

  例えば、一部のブラウザでは、ユーザーがマウスボタンを押しながらマウスを動かすと

  テキストがハイライトされます。このイベントのデフォルトの動作を防ぐには、

  関数の最後に<code>return false;</code>を追加してください。</p>

  <p>注意：<a href="/reference/p5/mousePressed/">mousePressed()</a>、
  <code>mouseReleased()</code>、

  <a href="/reference/p5/mouseClicked/">mouseClicked()</a>はすべて関連しています。

  <a href="/reference/p5/mousePressed/">mousePressed()</a>はユーザーがマウスをクリックした

  瞬間に実行されます。<code>mouseReleased()</code>はユーザーがマウスクリックを

  離した瞬間に実行されます。<a href="/reference/p5/mouseClicked/">mouseClicked()</a>は

  <code>mouseReleased()</code>の直後に実行されます。</p>
