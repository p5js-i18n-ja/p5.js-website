  <p>既存の配列から要素の配列を抽出します。
  listパラメータは
  要素がコピーされる元の配列を定義し、start
  とcountパラメータは抽出する要素を指定します。countが
  指定されていない場合、要素は配列の先頭から末尾まで抽出されます。
  startを指定する際は、最初の配列要素が0であることを
  忘れないでください。この関数は元の配列を変更しません。</p>

    function setup() {
      let myArray = [1, 2, 3, 4, 5];
      print(myArray); // [1, 2, 3, 4, 5]

      let sub1 = subset(myArray, 0, 3);
      let sub2 = subset(myArray, 2, 2);
      print(sub1); // [1,2,3]
      print(sub2); // [3,4]
    }
    </code></div>
