description: >
  <p>WebGLモードでスプライン曲線を描画するのに使用するセグメントの数を設定します。</p>

  <p>WebGLモードでは、滑らかな形状は多くの平らなセグメントを使用して描画されます。平らな

  セグメントを増やすことで、形状がより滑らかに見えます。</p>

  <p>パラメータ<code>detail</code>は、スプライン曲線を描画する際に使用するセグメントの数です。

  例えば、<code>curveDetail(5)</code>を呼び出すと、<a href="/reference/p5/curve/">curve()</a>関数で

  曲線を描画する際に5つのセグメントを使用します。デフォルトでは、<code>detail</code>は20です。</p>

  <p>注意：<code>curveDetail()</code>は2Dモードでは効果がありません。</p>

    function setup() {
      createCanvas(100, 100);

      background(200);

      noFill();

      curve(5, 26, 5, 26, 73, 24, 73, 61);

      stroke(0);
      point(73, 24);
      point(73, 61);

      // 制御点を赤で描画します。
      stroke(255, 0, 0);
      point(5, 26);
      point(15, 65);

      describe(

    function setup() {
      createCanvas(100, 100, WEBGL);

      background(200);

      // curveDetail()を3に設定します。
      curveDetail(3);

      // 黒いスプライン曲線を描画します。
      noFill();
      strokeWeight(1);
      stroke(0);
      curve(-45, -24, 0, 23, -26, 0, 23, 11, 0, -35, 15, 0);

      // アンカーポイントから制御点への赤いスプライン曲線を描画します。
      stroke(255, 0, 0);
      curve(-45, -24, 0, -45, -24, 0, 23, -26, 0, 23, 11, 0);
      curve(23, -26, 0, 23, 11, 0, -35, 15, 0, -35, 15, 0);

      // アンカーポイントを黒で描画します。
      strokeWeight(5);
      stroke(0);
      point(23, -26);
      point(23, 11);

      // 制御点を赤で描画します。
      stroke(255, 0, 0);
      point(-45, -24);
      point(-35, 15);

      describe(
        '3つのセグメントで描かれたギザギザの曲線のある灰色の正方形。曲線は横向きのU字形で、上下に赤いセグメント、右側に黒いセグメントがあります。すべてのセグメントの端点は点で表示されています。'
      );
    }
