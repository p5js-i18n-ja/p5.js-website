description: >
  <p>文字列の配列をテキストファイルに書き込みます。</p>

  <p>ひとつの文字列ごとに1行。ファイルの保存プロセスと保存されたファイルの場所は、ウェブブラウザー間で異なります。</p>


  <hr/>

  <p>Saves an <code>Array</code> of <code>String</code>s to a file, one per
  line.</p>

  <p>The first parameter, <code>list</code>, is an array with the strings to
  save.</p>

  <p>The second parameter, <code>filename</code>, is a string that sets the
  file's name.

  For example, calling <code>saveStrings(['0', '01', '011'], 'data.txt')</code>
  saves

  the array <code>['0', '01', '011']</code> to a file called
  <code>data.txt</code> on the user's

  computer.</p>

  <p>The third parameter, <code>extension</code>, is optional. If a string is
  passed, as in

  <code>saveStrings(['0', '01', '0</code>1'], 'data', 'txt')`, the second
  parameter will

  be interpreted as the file name and the third parameter as the extension.</p>

  <p>The fourth parameter, <code>isCRLF</code>, is also optional, If
  <code>true</code> is passed, as

  in <code>saveStrings(['0', '01', '011'], 'data', 'txt', true)</code>, then two

  characters, <code>\r\n</code> , will be added to the end of each string to
  create new

  lines in the saved file. <code>\r</code> is a carriage return (CR) and
  <code>\n</code> is a line

  feed (LF). By default, only <code>\n</code> (line feed) is added to each
  string in

  order to create new lines.</p>

  <p>Note: The browser will either save the file immediately or prompt the user

  with a dialogue window.</p>
