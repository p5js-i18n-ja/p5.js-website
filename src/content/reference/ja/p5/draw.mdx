description: >
  <p>スケッチの実行中に繰り返し呼び出される関数です。</p>

  <p><code>draw()</code>関数を宣言すると、スケッチが開始されてから

  繰り返し実行されるコードブロックを設定します。これはアニメーションの作成や

  ユーザー入力への応答に使用されます：</p>

  <pre><code class="language-js">function draw() {
    // 繰り返し実行されるコード。
  }

  </code></pre>

  <p>これは「ドローループ」と呼ばれることが多く、p5.jsが裏で

  <code>draw()</code>内のコードをループで呼び出すためです。デフォルトでは、
  <code>draw()</code>は

  1秒間に60回実行しようとします。実際のレートは多くの要因に依存します。

  描画レート（「フレームレート」と呼ばれる）は、

  <a href="/reference/p5/frameRate/">frameRate()</a>を呼び出すことで制御できます。<code>draw()</code>が

  実行された回数はシステム変数

  <a href="/reference/p5/frameCount/">frameCount()</a>に格納されます。</p>

  <p><code>draw()</code>内に配置されたコードは、

  <a href="/reference/p5/setup/">setup()</a>が実行された後にループを開始します。<code>draw()</code>は

  ユーザーがスケッチを閉じるまで実行され続けます。<code>draw()</code>は

  <a href="/reference/p5/noLoop/">noLoop()</a>関数を呼び出すことで停止できます。<code>draw()</code>は

  <a href="/reference/p5/loop/">loop()</a>関数を呼び出すことで再開できます。</p>
