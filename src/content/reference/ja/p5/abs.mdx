  <p>数値の絶対値を計算します。</p>

  <p>数値の絶対値とは、数直線上でその数値と0との距離です。

  -5と5はどちらも0から5単位離れているので、<code>abs(-5)</code>と

  <code>abs(5)</code>はどちらも5を返します。数値の絶対値は常に
  正の値です。</p>

    function setup() {
      createCanvas(100, 100);

      describe('灰色の正方形の中央に垂直の黒い線があります。ユーザーがマウスを線から遠ざけると、白い長方形が高くなります。');
    }

    function draw() {
      background(200);

      // キャンバスを分割します。
      line(50, 0, 50, 100);

      // マウスの中央からの距離を計算します。
      let h = abs(mouseX - 50);

      // マウスの中央からの距離に基づいて
      // 長方形を描画します。
      rect(0, 100 - h, 100, h);
    }
