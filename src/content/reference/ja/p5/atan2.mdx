  <p>指定した点から座標原点までの角度を、正の x 軸から測定したラジアン単位で計算します。</p>

  <p>Calculates the angle formed by a specified point, the origin, and the
  positive x-axis. By default, <code>atan2()</code> returns values in the range
  -π (about -3.14) to π (3.14). If the <a href="#/p5/angleMode">angleMode()</a>
  is <code>DEGREES</code>, then values are returned in the range -180 to 180.
  The <code>atan2()</code> function is most often used for orienting geometry to
  the mouse's position.</p>

  <p>Note: The y-coordinate of the point is the first parameter and the
  x-coordinate is the second parameter.</p>

    function setup() {
      createCanvas(100, 100);

      describe('A rectangle at the center of the canvas rotates with mouse movements.');
    }

    function draw() {
      background(200);

      // Translate the origin to the center.
      translate(50, 50);

      // Get the mouse's coordinates relative to the origin.
      let x = mouseX - 50;
      let y = mouseY - 50;

      // Calculate the angle between the mouse and the origin.
      let a = atan2(y, x);

      // Rotate.
      rotate(a);

      // Draw the shape.
      rect(-30, -5, 60, 10);
    }
    </code>
