description: >
  <p>数値に最も近い整数を計算します。</p>

  <p>例えば、<code>round(133.8)</code>は値134を返します。</p>

  <p>2番目のパラメータ<code>decimals</code>はオプションです。これは四捨五入する際の

  小数点以下の桁数を設定します。例えば、<code>round(12.34, 1)</code>は

  12.3を返します。<code>decimals</code>のデフォルト値は0です。</p>

      // テキストのスタイルを設定
      textAlign(CENTER);
      textSize(16);

      // 四捨五入された数値を表示
