description: >
  <p>画像をロードし、<a href="/reference/p5/p5.Image">p5.Image</a>オブジェクトを返します。</p>

  <p><code>loadImage()</code>は非同期関数です。通常、<code>preload()</code>内で呼び出されます。
  これにより、画像が<code>setup()</code>や<code>draw()</code>で使用可能になる前に
  完全にロードされることが保証されます。</p>

  <p><code>loadImage()</code>を<code>preload()</code>の外で呼び出す場合、
  コールバック関数を渡す必要があります。コールバックは、画像が完全にロードされた後に
  実行されます。</p>

  <p>サポートされている画像形式は、.gif、.jpg、.jpeg、.png、.svg、.webp、.avif、.apngです。
  ブラウザのサポート状況については、
  <a href="https://caniuse.com/webp" target="_blank">WebP</a>と
  <a href="https://caniuse.com/avif" target="_blank">AVIF</a>を参照してください。</p>

  <p>画像をロードする際、クロスオリジンの問題を避けるために、
  ローカルサーバーを使用することをお勧めします。</p>

example:
  - |-

    <div>
    <code>
    let img;

    function preload() {
      // 画像をロードします。
      img = loadImage('/assets/laDefense.jpg');
    }

    function setup() {
      createCanvas(100, 100);

      // 画像を描画します。
      image(img, 0, 0);

      describe('格子状の天井の下にある白い傘の裏側の画像。');
    }
    </code>
    </div>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // 画像をロードします。
      loadImage('/assets/laDefense.jpg', (img) => {
        // 画像を描画します。
        image(img, 0, 0);
      });

      describe('格子状の天井の下にある白い傘の裏側の画像。');
    }
    </code>
    </div>
