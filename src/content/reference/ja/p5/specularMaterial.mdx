description: >
  <p>形状の表面材質の鏡面反射色を設定します。</p>

  <p><code>specularMaterial()</code>の色は、形状の光沢のある表面が反射する光の色成分を
  設定します。例えば、<code>specularMaterial(255, 255, 0)</code>を呼び出すと、
  形状は赤と緑の光を反射しますが、青い光は反射しません。</p>

  <p><a href="/reference/p5/ambientMaterial/">ambientMaterial()</a>とは異なり、
  <code>specularMaterial()</code>は<a href="/reference/p5/directionalLight/">directionalLight()</a>、
  <a href="/reference/p5/pointLight/">pointLight()</a>、
  <a href="/reference/p5/spotLight/">spotLight()</a>を含む光源の全色を反射します。
  これにより形状に「光沢のある」外観が与えられます。材質の光沢は
  <a href="/reference/p5/shininess/">shininess()</a>関数で制御できます。</p>

  <p><code>specularMaterial()</code>は、材質の色を設定するために異なるパラメータで
  3つの方法で呼び出すことができます。</p>

  <p><code>specularMaterial()</code>を呼び出す最初の方法は、1つのパラメータ<code>gray</code>を
  使用します。<code>specularMaterial(50)</code>のように、0から255の間のグレースケール値を
  渡して材質の色を設定できます。グレースケール値が高いほど、形状はより明るく
  見えます。</p>

  <p><code>specularMaterial()</code>を呼び出す2つ目の方法は、1つのパラメータ<code>color</code>を
  使用します。<a href="/reference/p5/p5.Color">p5.Color</a>オブジェクト、色値の配列、または
  <code>specularMaterial('magenta')</code>のようなCSS色文字列を渡して、材質の色を
  設定できます。</p>

  <p><code>specularMaterial()</code>を呼び出す3つ目の方法は、4つのパラメータ<code>v1</code>、<code>v2</code>、
  <code>v3</code>、<code>alpha</code>を使用します。<code>alpha</code>はオプションです。RGBA、HSBA、またはHSLA値を
  渡して材質の色を設定できます。例えば、<code>specularMaterial(255, 0, 0)</code>や
  <code>specularMaterial(255, 0, 0, 30)</code>のようにします。色の値は現在の
  <a href="/reference/p5/colorMode/">colorMode()</a>を使用して解釈されます。</p>

    <div>
    <code>
    // クリックしてドラッグすると、異なる角度からシーンを見ることができます。
    // キャンバスをダブルクリックすると、鏡面反射材質が適用されます。

    let isGlossy = false;

    function setup() {
      createCanvas(100, 100, WEBGL);

      describe('灰色の背景に赤いトーラスが描かれています。ユーザーがダブルクリックすると光沢が出ます。');
    }

    function draw() {
      background(200);

      // マウスで軌道制御を有効にします。
      orbitControl();

      // 右上に白い点光源を配置します。
      pointLight(255, 255, 255, 30, -40, 30);

      // ユーザーがダブルクリックした場合、光沢のあるコートを追加します。
      if (isGlossy === true) {
        specularMaterial(255);
        shininess(50);
      }

      // トーラスのスタイルを設定します。
      noStroke();
      fill(255, 0, 0);

      // トーラスを描画します。
      torus(30);
    }

    // ユーザーがダブルクリックしたときにトーラスを光沢のある状態にします。
    function doubleClicked() {
      isGlossy = true;
    }
    </code>
