  <p>新しい<a href="/reference/p5/p5.Element">p5.Element</a>オブジェクトを作成します。</p>

  <p>最初のパラメータ<code>tag</code>は、<code>'h5'</code>のようなHTML要素を表す文字列です。</p>

  <p>2番目のパラメータ<code>content</code>はオプションです。新しい要素に挿入するHTML
  コンテンツを設定する文字列です。デフォルトでは、新しい要素にはコンテンツがありません。</p>

example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // 何も入っていないh5要素を作成します。
      createElement('h5');

      describe('灰色の正方形。');
    }
    </code>
    </div>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // "p5*js"というコンテンツを持つh5要素を作成します。
      let h5 = createElement('h5', 'p5*js');

      // 要素のスタイルと位置を設定します。
      h5.style('color', 'deeppink');
      h5.position(30, 15);

      describe('灰色の正方形の中央にピンク色で書かれた"p5*js"というテキスト。');
    }
    </code>
    </div>
