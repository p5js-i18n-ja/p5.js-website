description: >
  <p>線の端の描画スタイルを設定します。</p>

  <p>線の端のキャップは、丸型（<code>ROUND</code>）、四角型（<code>SQUARE</code>）、
  または延長型（<code>PROJECT</code>）のいずれかです。デフォルトのキャップは
  <code>ROUND</code>です。</p>

  <p><code>strokeCap()</code>に渡す引数は、すべて大文字で書く必要があります。
  これは、<code>ROUND</code>、<code>SQUARE</code>、<code>PROJECT</code>という定数が
  このように定義されているためです。JavaScriptは大文字と小文字を区別する言語です。</p>
line: 387
isConstructor: false
itemtype: method
example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      strokeWeight(12);

      // 上部
      strokeCap(ROUND);
      line(20, 30, 80, 30);

      // 中央
      strokeCap(SQUARE);
      line(20, 50, 80, 50);

      // 下部
      strokeCap(PROJECT);
      line(20, 70, 80, 70);

      describe(
        '3本の水平線。上の線は丸い端、中央の線は四角い端、下の線はより長い四角い端を持っています。'
      );
    }
    </code>
    </div>

    description: |
      <p>ROUND、SQUARE、またはPROJECTのいずれか</p>
    type: Constant
