description: >
  <p>3Dスケッチの方向を明確にするためのグリッドと軸アイコンを追加します。</p>

  <p><code>debugMode()</code>は、スケッチの「地面」がどこにあるかを示すグリッドを追加します。

  デフォルトでは、グリッドはスケッチの原点<code>(0, 0, 0)</code>を通り、

  XZ平面に沿って走ります。<code>debugMode()</code>はまた、正のx軸、y軸、z軸に沿って

  指す軸アイコンも追加します。<code>debugMode()</code>を呼び出すと、デフォルトのサイズと

  位置でグリッドと軸アイコンが表示されます。</p>

  <p><code>debugMode()</code>を呼び出す方法には、デバッグ環境をカスタマイズするための

  オプションパラメータを持つ4つの方法があります。</p>

  <p>最初の<code>debugMode()</code>の呼び出し方は、1つのパラメータ<code>mode</code>を持ちます。

  システム定数<code>GRID</code>が渡された場合、例えば<code>debugMode(GRID)</code>のように、

  グリッドが表示され、軸アイコンは非表示になります。定数<code>AXES</code>が

  渡された場合、例えば<code>debugMode(AXES)</code>のように、軸アイコンが表示され、

  グリッドは非表示になります。</p>

  <p>2つ目の<code>debugMode()</code>の呼び出し方は、6つのパラメータを持ちます。最初の

  パラメータ<code>mode</code>は、表示する<code>GRID</code>または<code>AXES</code>を選択します。

  次の5つのパラメータ、<code>gridSize</code>、<code>gridDivisions</code>、<code>xOff</code>、<code>yOff</code>、

  <code>zOff</code>はオプションです。これらは、グリッドの外観（<code>gridSize</code>と

  <code>gridDivisions</code>）と軸アイコンの配置（<code>xOff</code>、<code>yOff</code>、<code>zOff</code>）を

  設定する数値です。例えば、<code>debugMode(20, 5, 10, 10, 10)</code>を呼び出すと、

  <code>gridSize</code>を20ピクセルに、<code>gridDivisions</code>の数を5に設定し、

  x軸、y軸、z軸に沿って軸アイコンを10ピクセルずつオフセットします。</p>

  <p>3つ目の<code>debugMode()</code>の呼び出し方は、5つのパラメータを持ちます。最初の

  パラメータ<code>mode</code>は、表示する<code>GRID</code>または<code>AXES</code>を選択します。

  次の4つのパラメータ、<code>axesSize</code>、<code>xOff</code>、<code>yOff</code>、<code>zOff</code>はオプションです。

  これらは、軸アイコンのサイズ（<code>axesSize</code>）とその配置（<code>xOff</code>、<code>yOff</code>、

  <code>zOff</code>）を設定する数値です。</p>

  <p>4つ目の<code>debugMode()</code>の呼び出し方は、9つのオプションパラメータを持ちます。

  最初の5つのパラメータ、<code>gridSize</code>、<code>gridDivisions</code>、<code>gridXOff</code>、<code>gridYOff</code>、

  <code>gridZOff</code>は、グリッドの外観を設定する数値です。例えば、

  <code>debugMode(100, 5, 0, 0, 0)</code>を呼び出すと、<code>gridSize</code>を100に、

  <code>gridDivisions</code>の数を5に設定し、すべてのオフセットを0に設定して

  グリッドを原点に中心を置きます。次の4つのパラメータ、<code>axesSize</code>、

  <code>xOff</code>、<code>yOff</code>、<code>zOff</code>は、軸アイコンのサイズ（<code>axesSize</code>）と

  その配置（<code>axesXOff</code>、<code>axesYOff</code>、<code>axesZOff</code>）を設定する数値です。

  例えば、<code>debugMode(100, 5, 0, 0, 0, 50, 10, 10, 10)</code>を呼び出すと、

  <code>gridSize</code>を100に、<code>gridDivisions</code>の数を5に設定し、すべてのオフセットを0に

  設定してグリッドを原点に中心を置きます。そして、<code>axesSize</code>を50に設定し、

  各軸に沿って10ピクセルずつアイコンをオフセットします。</p>
