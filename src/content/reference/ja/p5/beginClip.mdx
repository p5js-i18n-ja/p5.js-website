description: >
  <p>クリッピング領域を開始します。</p>

  <p><code>beginClip()</code>と<code>endClip()</code>の間に描画されたすべてのものは、

  指定された形状内にクリップされます。クリッピング領域は、

  <code>beginClip()</code>と<code>endClip()</code>の間に描画された最初の形状によって定義されます。</p>

  <p>クリッピング領域は、<code>endClip()</code>が呼び出されるまで有効です。</p>

  <p><code>beginClip()</code>は、<code>push()</code>と<code>pop()</code>の間で使用する必要があります。</p>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);
    }

    function draw() {
      background(220);

      push();
      beginClip();
      circle(50, 50, 40);
      endClip();

      // この長方形は円の内側にのみ表示されます
      fill('red');
      rect(0, 0, 100, 100);
      pop();

      describe('赤い円。円は長方形でクリップされています。');
    }
    </code>
