description: >
  <p>線分を接続する接合部のスタイルを設定します。</p>

  <p>接合部は、マイター接合（<code>MITER</code>）、ベベル接合（<code>BEVEL</code>）、
  または丸接合（<code>ROUND</code>）のいずれかです。デフォルトの接合は2Dモードでは
  <code>MITER</code>、WebGLモードでは<code>ROUND</code>です。</p>

  <p><code>strokeJoin()</code>に渡される引数は、すべて大文字で書く必要があります。
  これは、<code>MITER</code>、<code>BEVEL</code>、<code>ROUND</code>という定数が
  このように定義されているためです。JavaScriptは大文字と小文字を区別する言語です。</p>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // 線のスタイルを設定します。
      noFill();
      strokeWeight(10);
      strokeJoin(MITER);

      // 線を描画します。
      beginShape();
      vertex(35, 20);
      vertex(65, 50);
      vertex(35, 80);
      endShape();

      describe('キャンバスの中央に尖った先端を持つ右向きの矢印の形。');
    }
