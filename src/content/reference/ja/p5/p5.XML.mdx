description: >
  <p>XMLオブジェクトを記述するクラスです。</p>

  <p>各<code>p5.XML</code>オブジェクトは、XMLデータと簡単に対話する方法を提供します。
  拡張可能なマークアップ言語
  （<a href="https://developer.mozilla.org/en-US/docs/Web/XML/XML_introduction/"
  target="_blank">XML</a>）
  は、アプリケーション間でデータを送信するための標準フォーマットです。HTMLと同様に、
  XMLフォーマットはタグと属性に基づいています。例えば、
  <code><time units="s">1234</time></code>のようになります。</p>

  <p>注意：外部XMLファイルを読み込むには<a href="/reference/p5/loadXML/">loadXML()</a>を使用してください。</p>
line: 9
isConstructor: true
example:
  - |-

    <div>
    <code>
    let myXML;

    // XMLを読み込み、p5.XMLオブジェクトを作成します。
    function preload() {
      myXML = loadXML('/assets/animals.xml');
    }

    function setup() {
      createCanvas(100, 100);

      background(200);

      // すべてのmammalタグを含む配列を取得します。
      let mammals = myXML.getChildren('mammal');

      // テキストのスタイルを設定します。
      textAlign(LEFT, CENTER);
      textFont('Courier New');
      textSize(14);

      // mammals配列を反復処理します。
      for (let i = 0; i < mammals.length; i += 1) {

        // y座標を計算します。
        let y = (i + 1) * 25;

        // 哺乳類の一般名を取得します。
        let name = mammals[i].getContent();

        // 哺乳類の名前を表示します。
        text(name, 20, y);
      }

      describe(
        '3行に分かれて書かれた「Goat」、「Leopard」、「Zebra」という単語。テキストは灰色の背景に黒で書かれています。'
      );
    }
    </code>
    </div>

  getParent:
    description: >
      <p>要素の親要素を新しい<a href="/reference/p5/p5.XML">p5.XML</a>
      オブジェクトとして返します。</p>
    path: p5.XML/getParent
  getName:
    description: >
      <p>要素の名前を<code>String</code>として返します。</p>

      <p>XML要素の名前はそのタグによって与えられます。例えば、
      <code><language>JavaScript</language></code>という要素の名前は
      <code>language</code>です。</p>
    path: p5.XML/getName
  setName:
    description: >
      <p>要素のタグ名を設定します。</p>

      <p>XML要素の名前はそのタグによって与えられます。例えば、
      <code><language>JavaScript</language></code>という要素の名前は
      <code>language</code>です。</p>

      <p>パラメータ<code>name</code>は、要素の新しい名前を文字列で指定します。例えば、
      <code>myXML.setName('planet')</code>を呼び出すと、要素の新しいタグ名が
      <code><planet></planet></code>になります。</p>
    path: p5.XML/setName
  hasChildren:
    description: >
      <p>要素に子要素がある場合は<code>true</code>を、ない場合は<code>false</code>を返します。</p>
    path: p5.XML/hasChildren
  listChildren:
    description: |
      <p>要素の子要素の名前を<code>String</code>の配列として返します。</p>
    path: p5.XML/listChildren
  getChildren:
    description: >
      <p>要素の子要素を新しい<a href="/reference/p5/p5.XML">p5.XML</a>オブジェクトの配列として返します。</p>

      <p>パラメータ<code>name</code>はオプションです。文字列が渡された場合、
      例えば<code>myXML.getChildren('cat')</code>のように、
      <code><cat></code>タグを持つ子要素のみを返します。</p>
    path: p5.XML/getChildren
  getChild:
    description: >
      <p>最初に一致する子要素を新しい<a href="/reference/p5/p5.XML">p5.XML</a>オブジェクトとして返します。</p>

      <p>パラメータ<code>name</code>はオプションです。文字列が渡された場合、
      例えば<code>myXML.getChild('cat')</code>のように、
      <code><cat></code>タグを持つ最初の子要素が返されます。数値が渡された場合、
      例えば<code>myXML.getChild(1)</code>のように、そのインデックスの子要素が返されます。</p>
    path: p5.XML/getChild
  addChild:
    description: >
      <p>新しい子要素を追加し、その参照を返します。</p>

      <p>パラメータ<code>child</code>は、子要素として追加する<a href="/reference/p5/p5.XML">p5.XML</a>オブジェクトです。
      例えば、<code>myXML.addChild(otherXML)</code>を呼び出すと、
      <code>otherXML</code>が<code>myXML</code>の子要素として挿入されます。</p>
    path: p5.XML/addChild
  removeChild:
    description: >
      <p>最初に一致する子要素を削除します。</p>

      <p>パラメータ<code>name</code>は、削除する子要素です。文字列が渡された場合、
      例えば<code>myXML.removeChild('cat')</code>のように、
      <code><cat></code>タグを持つ最初の子要素が削除されます。数値が渡された場合、
      例えば<code>myXML.removeChild(1)</code>のように、そのインデックスの子要素が
      削除されます。</p>
    path: p5.XML/removeChild
  getAttributeCount:
    description: |
      <p>要素が持つ属性の数を返します。</p>
    path: p5.XML/getAttributeCount
  listAttributes:
    description: >
      <p>要素の属性名を<code>Array</code>として返します。</p>

      <p>注意：属性の値を返すには
      <a href="/reference/p5.XML/getString/">myXML.getString()</a>または
      <a href="/reference/p5.XML/getNum/">myXML.getNum()</a>を使用してください。</p>
    path: p5.XML/listAttributes
  hasAttribute:
    description: >
      <p>要素が指定された属性を持っている場合は<code>true</code>を、持っていない場合は<code>false</code>を返します。</p>

      <p>パラメータ<code>name</code>は、チェックする属性の名前を文字列で指定します。</p>

      <p>注意：属性の値を返すには
      <a href="/reference/p5.XML/getString/">myXML.getString()</a>または
      <a href="/reference/p5.XML/getNum/">myXML.getNum()</a>を使用してください。</p>
    path: p5.XML/hasAttribute
  getNum:
    description: >
      <p>属性の値を<code>Number</code>として返します。</p>

      <p>最初のパラメータ<code>name</code>は、チェックする属性の名前を文字列で指定します。
      例えば、<code>myXML.getNum('id')</code>を呼び出すと、要素の<code>id</code>属性を
      数値として返します。</p>

      <p>2番目のパラメータ<code>defaultValue</code>はオプションです。数値が渡された場合、
      例えば<code>myXML.getNum('id', -1)</code>のように、属性が存在しないか
      数値に変換できない場合に返されます。</p>

      <p>注意：属性の値を返すには
      <a href="/reference/p5.XML/getString/">myXML.getString()</a>または
      <a href="/reference/p5.XML/getNum/">myXML.getNum()</a>を使用してください。</p>
    path: p5.XML/getNum
  getString:
    description: >
      <p>属性の値を文字列として返します。</p>

      <p>最初のパラメータ<code>name</code>は、チェックする属性の名前を文字列で指定します。
      例えば、<code>myXML.getString('color')</code>を呼び出すと、要素の<code>color</code>属性を
      文字列として返します。</p>

      <p>2番目のパラメータ<code>defaultValue</code>はオプションです。文字列が渡された場合、
      例えば<code>myXML.getString('color', 'deeppink')</code>のように、属性が存在しない場合に
      返されます。</p>

      <p>注意：属性の値を返すには
      <a href="/reference/p5.XML/getString/">myXML.getString()</a>または
      <a href="/reference/p5.XML/getNum/">myXML.getNum()</a>を使用してください。</p>
    path: p5.XML/getString
  setAttribute:
    description: >
      <p>属性に指定された値を設定します。</p>

      <p>最初のパラメータ<code>name</code>は、設定する属性の名前を文字列で指定します。</p>

      <p>2番目のパラメータ<code>value</code>は、属性の新しい値です。例えば、
      <code>myXML.setAttribute('id', 123)</code>を呼び出すと、<code>id</code>属性の値を
      123に設定します。</p>
    path: p5.XML/setAttribute
  getContent:
    description: >
      <p>要素のコンテンツを<code>String</code>として返します。</p>

      <p>パラメータ<code>defaultValue</code>はオプションです。文字列が渡された場合、
      例えば<code>myXML.getContent('???')</code>のように、要素にコンテンツがない場合に
      返されます。</p>
    path: p5.XML/getContent
  setContent:
    description: >
      <p>要素のコンテンツを設定します。</p>

      <p>要素のコンテンツは、そのタグの間のテキストです。例えば、
      <code><language>JavaScript</language></code>という要素のコンテンツは
      <code>JavaScript</code>です。</p>

      <p>パラメータ<code>content</code>は、要素の新しいコンテンツを文字列で指定します。</p>
    path: p5.XML/setContent
  serialize:
    description: >
      <p>要素を<code>String</code>として返します。</p>

      <p><code>myXML.serialize()</code>は、要素をネットワーク経由で送信したり
      ファイルに保存したりする際に便利です。</p>
    path: p5.XML/serialize
