  <p>指定された色と位置で点光源を作成します。</p>

  <p></p>

  <p>点光源は、単一の点からすべての方向に光を放射します。光は特定の点（位置）から放射されるため、オブジェクトに対して遠くまたは近くに配置すると効果が異なります。</p>

  <p>一度にアクティブにできる点光源は <strong>5</strong> つまでです。</p>

  <p>注: ループプログラム内でライトを持続させるためには、draw() 内でライトを呼び出す必要があります（直接または間接的に）。setup()
  に配置すると、ループの最初の通過でのみ効果があります。</p>


  <hr/>

  <p>Creates a light that shines from a point in all directions.</p>

  <p>Point lights are like light bulbs that shine in all directions. They can be

  placed at different positions to achieve different lighting effects. A

  maximum of 5 point lights can be active at once.</p>

  <p>There are four ways to call <code>pointLight()</code> with parameters to
  set the

  light’s color and position.</p>

  <p>The first way to call <code>pointLight()</code> has six parameters. The
  first three

  parameters, <code>v1</code>, <code>v2</code>, and <code>v3</code>, set the
  light’s color using the current

  <a href="/reference/p5/colorMode">colorMode()</a>. The last three parameters,
  <code>x</code>,

  <code>y</code>, and <code>z</code>, set the light’s position. For example,

  <code>pointLight(255, 0, 0, 50, 0, 0)</code> creates a red <code>(255, 0,
  0)</code> light that

  shines from the coordinates <code>(50, 0, 0)</code>.</p>

  <p>The second way to call <code>pointLight()</code> has four parameters. The
  first three

  parameters, <code>v1</code>, <code>v2</code>, and <code>v3</code>, set the
  light’s color using the current

  <a href="/reference/p5/colorMode">colorMode()</a>. The last parameter,
  position sets

  the light’s position using a <a href="/reference/p5/p5.Vector">p5.Vector</a>
  object.

  For example, <code>pointLight(255, 0, 0, lightPos)</code> creates a red
  <code>(255, 0, 0)</code>

  light that shines from the position set by the <code>lightPos</code>
  vector.</p>

  <p>The third way to call <code>pointLight()</code> has four parameters. The
  first

  parameter, <code>color</code>, sets the light’s color using a

  <a href="/reference/p5/p5.Color">p5.Color</a> object or an array of color
  values. The

  last three parameters, <code>x</code>, <code>y</code>, and <code>z</code>, set
  the light’s position. For

  example, <code>directionalLight(myColor, 50, 0, 0)</code> creates a light that
  shines

  from the coordinates <code>(50, 0, 0)</code> with the color value of
  <code>myColor</code>.</p>

  <p>The fourth way to call <code>pointLight()</code> has two parameters. The
  first

  parameter, <code>color</code>, sets the light’s color using a

  <a href="/reference/p5/p5.Color">p5.Color</a> object or an array of color
  values. The

  second parameter, <code>position</code>, sets the light’s position using a

  <a href="/reference/p5/p5.Vector">p5.Vector</a> object. For example,

  <code>directionalLight(myColor, lightPos)</code> creates a light that shines
  from the

  position set by the <code>lightPos</code> vector with the color value of
  <code>myColor</code>.</p>

    // Click and drag the mouse to view the scene from different angles.
    // Double-click to turn on the point light.

    let isLit = false;

    function setup() {
      createCanvas(100, 100, WEBGL);

      describe('A sphere drawn on a gray background. A red light starts shining from above when the user double-clicks.');
    }

    function draw() {
      background(200);

      // Enable orbiting with the mouse.
      orbitControl();

      // Control the light.
      if (isLit === true) {
        // Add a red point light from above.
        // Use RGB values and XYZ coordinates.
        pointLight(255, 0, 0, 0, -150, 0);
      }

      // Style the sphere.
      noStroke();

      // Draw the sphere.
      sphere(30);
    }

    // Turn on the point light when the user double-clicks.
    function doubleClicked() {
      isLit = true;
    }
