description: >
  <p>全方向に光を放つ点光源を作成します。</p>

  <p>点光源は、あらゆる方向に光を放つ電球のようなものです。異なる位置に配置することで、

  さまざまな照明効果を得ることができます。一度に最大5つの点光源を

  アクティブにできます。</p>

  <p><code>pointLight()</code>を呼び出す方法には、光の色と位置を設定するパラメータを

  使用する4つの方法があります。</p>

  <p><code>pointLight()</code>を呼び出す最初の方法は、6つのパラメータを使用します。最初の3つの

  パラメータ、<code>v1</code>、<code>v2</code>、<code>v3</code>は、現在の

  <a href="/reference/p5/colorMode/">colorMode()</a>を使用して光の色を設定します。残りの3つのパラメータ、

  <code>x</code>、<code>y</code>、<code>z</code>は、光の位置を設定します。例えば、

  <code>pointLight(255, 0, 0, 50, 0, 0)</code>は、座標<code>(50, 0, 0)</code>から

  赤色<code>(255, 0, 0)</code>の光を放つ点光源を作成します。</p>

  <p><code>pointLight()</code>を呼び出す2番目の方法は、4つのパラメータを使用します。最初の3つの

  パラメータ、<code>v1</code>、<code>v2</code>、<code>v3</code>は、現在の

  <a href="/reference/p5/colorMode/">colorMode()</a>を使用して光の色を設定します。最後のパラメータ、position は

  <a href="/reference/p5/p5.Vector">p5.Vector</a>オブジェクトを使用して光の位置を設定します。

  例えば、<code>pointLight(255, 0, 0, lightPos)</code>は、<code>lightPos</code>ベクトルで

  設定された位置から赤色<code>(255, 0, 0)</code>の光を放つ点光源を作成します。</p>

  <p><code>pointLight()</code>を呼び出す3番目の方法は、4つのパラメータを使用します。最初の

  パラメータ、<code>color</code>は、<a href="/reference/p5/p5.Color">p5.Color</a>オブジェクトまたは

  色値の配列を使用して光の色を設定します。残りの3つのパラメータ、<code>x</code>、<code>y</code>、<code>z</code>は、

  光の位置を設定します。例えば、<code>directionalLight(myColor, 50, 0, 0)</code>は、

  座標<code>(50, 0, 0)</code>から<code>myColor</code>の色値で光を放つ点光源を作成します。</p>

  <p><code>pointLight()</code>を呼び出す4番目の方法は、2つのパラメータを使用します。最初の

  パラメータ、<code>color</code>は、<a href="/reference/p5/p5.Color">p5.Color</a>オブジェクトまたは

  色値の配列を使用して光の色を設定します。2番目のパラメータ、<code>position</code>は、

  <a href="/reference/p5/p5.Vector">p5.Vector</a>オブジェクトを使用して光の位置を設定します。

  例えば、<code>directionalLight(myColor, lightPos)</code>は、<code>lightPos</code>ベクトルで

  設定された位置から<code>myColor</code>の色値で光を放つ点光源を作成します。</p>

      createCanvas(100, 100, WEBGL);

        // 上から赤い点光源を追加します。

        pointLight(255, 0, 0, 0, -150, 0);
      }

      // 球体のスタイルを設定します。
      noStroke();

      // 球体を描画します。
      sphere(30);
    }

    // ユーザーがダブルクリックすると点光源をオンにします。
    function doubleClicked() {
      isLit = true;
    }
