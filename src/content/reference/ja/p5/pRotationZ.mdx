  <p>システム変数pRotationZは、常に現在のフレームの前のフレームにおける

  デバイスのz軸周りの回転を含んでいます。

  スケッチの<a href="/reference/p5/angleMode/">angleMode()</a>がDEGREESに設定されている場合、

  値は0から360になります。RADIANSに設定されている場合、値は

  0から2*PIになります。</p>

  <p>pRotationZはrotationZと共に使用して、デバイスのZ軸周りの回転

  方向を判断することもできます。</p>

    <div class='norender'>
    <code>
    // rotationZ - pRotationZ < 0 が真かどうかを
    // 単純にif文で確認するだけで、ほとんどの場合
    // 回転方向を判断するのに十分です。

    // 角度が一周する場合に対応するために
    // 追加のロジックが必要です。
    let rotateDirection = 'clockwise';

    if (
      (rotationZ - pRotationZ > 0 && rotationZ - pRotationZ < 270) ||
      rotationZ - pRotationZ < -270
    ) {
      rotateDirection = 'clockwise';
    } else if (rotationZ - pRotationZ < 0 || rotationZ - pRotationZ > 270) {
      rotateDirection = 'counter-clockwise';
    }
    print(rotateDirection);
