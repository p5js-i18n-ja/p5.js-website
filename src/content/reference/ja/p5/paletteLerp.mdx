description: >
  <p>複数の色を混ぜ合わせて、その間の色を見つけます。</p>

  <p><code>amt</code>パラメータは、色の停止点間で補間する量を指定します。
  色の停止点は、各<code>amt</code>値の「位置」にある色で、
  2つの色の停止点の間にある<code>amt</code>値は、
  両方からの相対的な距離に基づいてそれらの間を補間します。</p>

  <p>色の補間方法は、現在の
  <a href="/reference/p5/colorMode/">colorMode()</a>に依存します。</p>

    function setup() {
      createCanvas(400, 400);

    function draw() {
      // 背景は白から赤、緑、青へと変化します
      background(paletteLerp([
        ['white', 0],
        ['red', 0.05],
        ['green', 0.25],
        ['blue', 1]
      ], millis() / 10000 % 1));
