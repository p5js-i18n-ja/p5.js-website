description: >
  <p>スケッチの実行が開始されたときに一度だけ呼び出される関数です。</p>

  <p><code>setup()</code>関数を宣言すると、スケッチの開始時に自動的に一度だけ実行される

  コードブロックを設定します。これは、キャンバスの作成や変数の初期化など、

  セットアップタスクを実行するために使用されます：</p>

  <pre><code class="language-js">function setup() {
    // スケッチの開始時に一度だけ実行されるコード。
  }

  </code></pre>

  <p><code>setup()</code>内に配置されたコードは、<a href="/reference/p5/draw/">draw()</a>内に

  配置されたコードがループを開始する前に一度だけ実行されます。

  <a href="/reference/p5/preload/">preload()</a>が宣言されている場合、<code>setup()</code>は

  <a href="/reference/p5/preload/">preload()</a>がアセットの読み込みを

  完了した直後に実行されます。</p>

  <p>注意：<code>setup()</code>を宣言する必要はありませんが、一般的にはそうすることが慣例です。</p>
