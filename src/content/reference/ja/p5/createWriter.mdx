  <p>新しい<a href="/reference/p5/p5.PrintWriter">p5.PrintWriter</a>オブジェクトを作成します。</p>

  <p><a href="/reference/p5/p5.PrintWriter">p5.PrintWriter</a>オブジェクトは、
  <em>プリントストリーム</em>と呼ばれるテキストデータの連続をユーザーの
  コンピューターに保存する方法を提供します。これらは低レベルのオブジェクトで、
  テキスト出力の精密な制御を可能にします。
  <a href="/reference/p5/saveStrings/">saveStrings()</a>や
  <a href="/reference/p5/saveJSON/">saveJSON()</a>などの関数は、
  単純なファイル保存にはより使いやすいです。</p>

  <p>最初のパラメータ<code>filename</code>は、書き込むファイルの名前です。
  <code>createWriter('words.txt')</code>のように文字列が渡された場合、
  <code>words.txt</code>という名前のファイルに書き込む新しい
  <a href="/reference/p5/p5.PrintWriter">p5.PrintWriter</a>オブジェクトが
  作成されます。</p>

  <p>2番目のパラメータ<code>extension</code>はオプションです。
  <code>createWriter('words', 'csv')</code>のように文字列が渡された場合、
  最初のパラメータはファイル名として、2番目のパラメータは拡張子として
  解釈されます。</p>
