  <p>Creates an <code>&lt;img&gt;</code> element that can appear outside of the
  canvas.</p>

  <p>The first parameter, <code>src</code>, is a string with the path to the
  image file. <code>src</code> should be a relative path, as in
  <code>'assets/image.png'</code>, or a URL, as in
  <code>'https://example.com/image.png'</code>.</p>

  <p>The second parameter, <code>alt</code>, is a string with the <a
  href="https://developer.mozilla.org/en-US/docs/Web/API/HTMLImageElement/alt#usage_notes"
  target="_blank">alternate text</a> for the image. An empty string
  <code>''</code> can be used for images that aren't displayed.</p>

  <p>The third parameter, <code>crossOrigin</code>, is optional. It's a string
  that sets the <a
  href="https://developer.mozilla.org/en-US/docs/Web/HTML/CORS_settings_attributes"
  target="_blank">crossOrigin property</a> of the image. Use
  <code>'anonymous'</code> or <code>'use-credentials'</code> to fetch the image
  with cross-origin access.</p>

  <p>The fourth parameter, <code>callback</code>, is also optional. It sets a
  function to call after the image loads. The new image is passed to the
  callback function as a <a href="#/p5.Element">p5.Element</a> object.</p>
