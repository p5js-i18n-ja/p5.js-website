description: >
  <p>印字可能な文字のキーが押されたときに一度呼び出される関数です。</p>

  <p><code>keyTyped()</code>関数を宣言すると、ユーザーが<code>a</code>や1のような
  印字可能な文字のキーを押したときに自動的に一度実行されるコードブロックを
  設定します。<code>SHIFT</code>、<code>CONTROL</code>、矢印キーなどの
  修飾キーは無視されます：</p>

  <pre><code class="language-js">function keyTyped() {
    // 実行するコード
  }

  </code></pre>

  <p><a href="/reference/p5/key/">key</a>変数と<a
  href="/reference/p5/keyCode/">keyCode</a>変数は、
  p5.jsによって<code>keyTyped()</code>が呼び出されたときに、
  最後に離されたキーの値で更新されます：</p>

  <pre><code class="language-js">function keyTyped() {
    // keyを使用して "c" 文字をチェック
    if (key === 'c') {
      // 実行するコード
    }

    // keyCodeを使用して "c" をチェック
    if (keyCode === 67) {
      // 実行するコード
    }
  }

  </code></pre>

  <p>パラメータ<code>event</code>はオプションです。<code>keyTyped()</code>には常に
  キー押下イベントを説明するプロパティを持つ
  <a href="https://developer.mozilla.org/en-US/docs/Web/API/KeyboardEvent/"
  target="_blank">KeyboardEvent</a>
  オブジェクトが渡されます：</p>

  <pre><code class="language-js">function keyReleased(event) {
    // イベントを使用するコード
    console.log(event);
  }

  </code></pre>

  <p>注意：<code>ALT</code>などの修飾キーに反応するには、
  <a href="/reference/p5/keyPressed/">keyPressed()</a>関数と
  <a href="/reference/p5/keyCode/">keyCode</a>システム変数を使用してください。</p>

  <p>ブラウザは様々なキーイベントにデフォルトの動作を関連付けている場合があります。
  このイベントのデフォルトの動作を防ぐには、関数の最後に<code>return false;</code>を
  追加してください。</p>
