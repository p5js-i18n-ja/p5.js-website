  <p>キャンバスの幅をピクセル単位で格納する<code>Number</code>変数です。</p>

  <p><code>width</code>のデフォルト値は100です。

  <a href="/reference/p5/createCanvas/">createCanvas()</a>または

  <a href="/reference/p5/resizeCanvas/">resizeCanvas()</a>を呼び出すと

  <code>width</code>の値が変更されます。<a href="/reference/p5/noCanvas/">noCanvas()</a>を呼び出すと

  その値が0に設定されます。</p>

    function setup() {
      createCanvas(100, 100);

      background(200);

      // キャンバスの幅を表示します。
      text(width, 42, 54);

      describe('灰色の正方形の上に黒で書かれた数字100。マウスが押されると、正方形が長方形になり、数字が50になります。');
    }

    // マウスが押されたら、キャンバスのサイズを
    // 変更し、新しい幅を表示します。
    function mousePressed() {
      if (mouseX > 0 && mouseX < width && mouseY > 0 && mouseY < height) {
        resizeCanvas(50, 100);
        background(200);
        text(width, 21, 54);
      }
    }
    </code>
