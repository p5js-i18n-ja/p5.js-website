  <p><code class="language-javascript">describe(text, FALLBACK)</code> は、<a

  <p>Creates a screen reader-accessible description for the canvas.</p>

  <p>The first parameter, <code>text</code>, is the description of the
  canvas.</p>

  <p>The second parameter, <code>display</code>, is optional. It determines how
  the description is displayed. If <code>LABEL</code> is passed, as in
  <code>describe('A description.', LABEL)</code>, the description will be
  visible in a div element next to the canvas. If <code>FALLBACK</code> is
  passed, as in <code>describe('A description.', FALLBACK)</code>, the
  description will only be visible to screen readers. This is the default
  mode.</p>

  <p>Read <a href="/learn/labeling-canvases.html">How to label your p5.js
  code</a> to learn more about making sketches accessible.</p>
