  <p>フォントオブジェクトを表します。</p>

  または<a href="/reference/p5/textFont">textFont()</a>を使用して作成されます。</p>

  <p><code>p5.Font</code>オブジェクトは、テキストのレンダリングに使用されます。
  これには、フォントのメトリクス（文字の高さなど）を取得するためのメソッドも含まれています。</p>

methods:
  textBounds:
    description: >
      <p>指定されたテキスト文字列の境界ボックスを計算します。</p>

      <p>境界ボックスは、テキストを含む最小の長方形です。
      これは、テキストの配置や衝突検出に役立ちます。</p>

      <p>返されるオブジェクトには以下のプロパティがあります：</p>

      <ul>

      <li><code>x</code>：境界ボックスの左上隅のx座標。</li>

      <li><code>y</code>：境界ボックスの左上隅のy座標。</li>

      <li><code>w</code>：境界ボックスの幅。</li>

      <li><code>h</code>：境界ボックスの高さ。</li>

      </ul>
    path: p5.Font/textBounds
  textToPoints:
    description: >
      <p>テキスト文字列のアウトラインを点の配列として返します。</p>

      <p>各点は、<code>x</code>座標、<code>y</code>座標、および文字のインデックスを含むオブジェクトです。</p>

      <p>返される点の数は、<code>sampleFactor</code>パラメータによって制御されます。
      値が小さいほど、より多くの点が返されます。</p>

      <p>オプションのオブジェクトパラメータには、以下のプロパティを含めることができます：</p>

      <ul>

      <li><code>sampleFactor</code>：カーブのサンプリング密度を設定します。デフォルトは0.1です。</li>

      <li><code>simplifyThreshold</code>：単純化のしきい値（0-1）を設定します。デフォルトは0です。</li>

      </ul>

      <p>テキストのアウトラインを描画したり、テキストに基づいてアニメーションを作成したりするのに便利です。</p>
    path: p5.Font/textToPoints
chainable: false
