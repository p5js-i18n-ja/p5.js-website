description: >
  <p>WebGLモードでベジエ曲線を描画する際に使用するセグメント数を設定します。</p>

  <p>WebGLモードでは、滑らかな形状は多くの平らなセグメントを使用して描画されます。平らなセグメントを増やすことで、形状がより滑らかに見えます。</p>

  <p>パラメータ<code>detail</code>は、ベジエ曲線を描画する際に使用するセグメント数です。例えば、<code>bezierDetail(5)</code>を呼び出すと、<a href="/reference/p5/bezier/">bezier()</a>関数で曲線を描画する際に5つのセグメントを使用します。デフォルトでは、<code>detail</code>は20です。</p>

  <p>注意：<code>bezierDetail()</code>は2Dモードでは効果がありません。</p>

example:
  - |-

    <div>
    <code>
    // オリジナルの曲線を描画する

    function setup() {
      createCanvas(100, 100);

      background(200);

      // アンカーポイントを黒で描画
      stroke(0);
      strokeWeight(5);
      point(85, 20);
      point(15, 80);

      // コントロールポイントを赤で描画
      stroke(255, 0, 0);
      point(10, 10);
      point(90, 90);

      // 黒のベジエ曲線を描画
      noFill();
      stroke(0);
      strokeWeight(1);
      bezier(85, 20, 10, 10, 90, 90, 15, 80);

      // アンカーポイントからコントロールポイントへの赤い線を描画
      stroke(255, 0, 0);
      line(85, 20, 10, 10);
      line(15, 80, 90, 90);

      describe(
        '3つの曲線がある灰色の正方形。黒のS字曲線の端から2本の赤い直線が伸びている。すべての曲線の端点は点で示されている。'
      );
    }
    </code>

    <div>
    <code>
    // より少ない詳細度で曲線を描画する

    function setup() {
      createCanvas(100, 100, WEBGL);

      background(200);

      // curveDetail()を5に設定
      bezierDetail(5);

      // アンカーポイントを黒で描画
      stroke(0);
      strokeWeight(5);
      point(35, -30, 0);
      point(-35, 30, 0);

      // コントロールポイントを赤で描画
      stroke(255, 0, 0);
      point(-40, -40, 0);
      point(40, 40, 0);

      // 黒のベジエ曲線を描画
      noFill();
      stroke(0);
      strokeWeight(1);
      bezier(35, -30, 0, -40, -40, 0, 40, 40, 0, -35, 30, 0);

      // アンカーポイントからコントロールポイントへの赤い線を描画
      stroke(255, 0, 0);
      line(35, -30, -40, -40);
      line(-35, 30, 40, 40);

      describe(
        '3つの曲線がある灰色の正方形。黒のS字曲線がギザギザのセグメントで描かれている。端から2本の赤い直線が伸びている。すべての曲線の端点は点で示されている。'
      );
    }
    </code>
