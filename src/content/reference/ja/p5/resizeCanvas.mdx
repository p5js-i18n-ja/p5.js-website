description: >
  <p>キャンバスを指定された幅と高さにリサイズします。</p>

  <p><code>resizeCanvas()</code>は即座にキャンバスをクリアし、

  <a href="/reference/p5/redraw/">redraw()</a>を呼び出します。
  <code>resizeCanvas()</code>は

  一般的に<a href="/reference/p5/windowResized/">windowResized()</a>の本体内で
  
  次のように呼び出されます：</p>

  <pre><code class="language-js">function windowResized() {
    resizeCanvas(windowWidth, windowHeight);
  }

  </code></pre>

  <p>最初の2つのパラメータ、<code>width</code>と<code>height</code>は、キャンバスの

  寸法を設定します。これらは同時に<a href="/reference/p5/width/">width</a>と

  <a href="/reference/p5/height/">height</a>システム変数の値も設定します。例えば、

  <code>resizeCanvas(300, 500)</code>を呼び出すと、キャンバスを300×500ピクセルにリサイズし、

  <a href="/reference/p5/width/">width</a>を300に、

  <a href="/reference/p5/height/">height</a>を500に設定します。</p>

  <p>3番目のパラメータ<code>noRedraw</code>はオプションです。<code>true</code>が渡された場合、

  例えば<code>resizeCanvas(300, 500, true)</code>のように、キャンバスは300×500

  ピクセルにリサイズされますが、<a href="/reference/p5/redraw/">redraw()</a>関数は

  即座に呼び出されません。デフォルトでは、<code>resizeCanvas()</code>の実行が

  完了すると直ちに<a href="/reference/p5/redraw/">redraw()</a>が

  呼び出されます。</p>

    // ダブルクリックしてキャンバスをリサイズします。

    function setup() {
      createCanvas(100, 100);

      describe(
        'グレーの背景に白い円が描かれています。ユーザーが初めてダブルクリックすると、キャンバスは半分のサイズに縮小します。'
      );
    }

    function draw() {
      background(180);

      // キャンバスの中心に円を描きます。
      circle(width / 2, height / 2, 20);
    }

    // ユーザーがダブルクリックしたときにキャンバスをリサイズします。
    function doubleClicked() {
      resizeCanvas(50, 50);
    }
    </code>
    </div>

    <code>
    // Webブラウザのサイズを変更してキャンバスのサイズを変更します。

    function setup() {
      createCanvas(windowWidth, windowHeight);

      describe('グレーの背景に白い円が描かれています。');
    }

    function draw() {
      background(180);

      // キャンバスの中心に円を描きます。
      circle(width / 2, height / 2, 20);
    }

    // 常にブラウザウィンドウを埋めるようにキャンバスをリサイズします。
    function windowResized() {
      resizeCanvas(windowWidth, windowHeight);
    }
    </code>
    </div>
