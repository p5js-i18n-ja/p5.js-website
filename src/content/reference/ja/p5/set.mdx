description: >
  <p>ピクセルの色を設定したり、キャンバスに画像を描画したりします。</p>

  <p><code>set()</code>は使いやすいですが、<a href="/reference/p5/pixels/">pixels</a>
  ほど高速ではありません。多くのピクセル値を設定するには
  <a href="/reference/p5/pixels/">pixels</a>を使用してください。</p>

  <p><code>set()</code>は最初の2つのパラメータをx座標とy座標として解釈します。
  最後のパラメータはグレースケール値、<code>[R, G, B, A]</code>ピクセル配列、
  <a href="/reference/p5/p5.Color">p5.Color</a>オブジェクト、または
  <a href="/reference/p5/p5.Image">p5.Image</a>オブジェクトとして解釈されます。
  画像が渡された場合、最初の2つのパラメータは、現在の
  <a href="/reference/p5/imageMode/">imageMode()</a>に関係なく、
  画像の左上隅の座標を設定します。</p>

  <p><code>set()</code>を使用した後、変更を表示するには
  <a href="/reference/p5/updatePixels/">updatePixels()</a>を呼び出す必要があります。</p>

      createCanvas(100, 100);

      background(255);

      // 水平方向のカラーグラデーションを描画します。
      for (let x = 0; x < 100; x += 1) {
        for (let y = 0; y < 100; y += 1) {
          // グレースケール値を計算します。
          let c = map(x, 0, 100, 0, 255);

          // グレースケール値を使用してピクセルを設定します。
          set(x, y, c);
        }
      }

      // キャンバスを更新します。
      updatePixels();

      describe('黒から白への水平方向のカラーグラデーション。');
    }
