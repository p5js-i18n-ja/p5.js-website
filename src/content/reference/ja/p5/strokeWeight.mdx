description: >
  <p>点、線、および図形の輪郭に使用されるストロークの幅を設定します。</p>

  <p>注意：<code>strokeWeight()</code>は変換の影響を受けます。特に
  <a href="/reference/p5/scale/">scale()</a>の呼び出しの影響を受けます。</p>
line: 540
isConstructor: false
itemtype: method
example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // 上部
      line(20, 20, 80, 20);

      // 中央
      strokeWeight(4);
      line(20, 40, 80, 40);

      // 下部
      strokeWeight(10);
      line(20, 70, 80, 70);

      describe('3本の水平な黒い線。上の線は細く、中央は中程度、下は太い。');
    }
    </code>
    </div>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // 上部
      line(20, 20, 80, 20);

      // 5倍にスケーリング
      scale(5);

      // 下部。座標はスケーリングに合わせて調整されています。
      line(4, 8, 16, 8);

      describe('2本の水平な黒い線。上の線は細く、下の線は上の線の5倍の太さ。');
    }
    </code>
    </div>
