description: >
  <p>新しい変数を宣言します。</p>

  <p>変数は値を格納するコンテナです。例えば、変数は生き物のx座標を<code>Number</code>として、

  または名前を<code>String</code>として格納することができます。変数は以下のように再代入することで

  値を変更できます：</p>

  <pre><code class="language-js">// 変数xを宣言し、値10を代入します。

  let x = 10;


  // xを50に再代入します。

  x = 50;

  </code></pre>

  <p>変数はブロックスコープを持ちます。変数が中括弧<code>{}</code>の間で宣言されると、

  それらの中括弧で定義されたブロック内でのみ存在します。

  例えば、以下のコードは<code>ReferenceError</code>をスローします。なぜなら<code>x</code>は

  <code>setup()</code>関数のブロック内で宣言されているからです：</p>

  <pre><code class="language-js">function setup() {
    createCanvas(100, 100);

    let x = 50;
  }


  function draw() {
    background(200);

    // xはsetup()内で宣言されたため、ここで参照することはできません。
    circle(x, 50, 20);
  }

  </code></pre>

  <p>すべての中括弧<code>{}</code>の外で宣言された変数はグローバルスコープにあります。

  グローバルスコープにある変数は、スケッチのどこでも使用および変更することができます：</p>

  <pre><code class="language-js">let x = 50;


  function setup() {
    createCanvas(100, 100);
  }


  function draw() {
    background(200);

    // xの値を変更します。
    x += 10;

    circle(x, 50, 20);
  }

  </code></pre>
