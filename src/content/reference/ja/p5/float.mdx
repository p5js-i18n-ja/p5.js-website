description: >-
  <p>文字列を浮動小数点表現に変換します。</p>

  <p>文字列の内容は数値に似ていなければならず、そうでない場合は
  NaN（数値ではない）が返されます。たとえば、float("1234.56")は1234.56に評価されますが、float("キリン")は NaN
  が返されます。</p>

  <p>値の配列が渡された場合、同じ長さの浮動小数点の配列が返されます。</p>


  <hr/>

  <p>Converts a <code>String</code> to a floating point (decimal)
  <code>Number</code>.</p>

  <p><code>float()</code> converts strings that resemble numbers, such as
  <code>'12.34'</code>, into

  numbers.</p>

  <p>The parameter, <code>str</code>, is the string value to convert. For
  example, calling

  <code>float('12.34')</code> returns the number <code>12.34</code>.  If an
  array of strings is

  passed, as in <code>float(['12.34', '56.78'])</code>, then an array of numbers
  will be

  returned.</p>

  <p>Note: If a string can't be converted to a number, as in
  <code>float('giraffe')</code>,

  then the value <code>NaN</code> (not a number) will be returned.</p>

    function setup() {
      createCanvas(100, 100);

      background(200);

      // Create a string variable.
      let original = '12.3';

      // Convert the string to a number.
      let converted = float(original);

      // Double the converted value.
      let twice = converted * 2;

      textAlign(CENTER, CENTER);
