description: >
  <p><code>Number</code>型のシステム変数で、最後に入力されたキーのコードを含みます。</p>

  <p>すべてのキーには<code>keyCode</code>があります。例えば、<code>a</code>キーの
  <code>keyCode</code>は65です。

  <code>keyCode</code>変数は、特殊キーが入力されたかどうかを確認するのに
  役立ちます。例えば、以下の条件文はエンターキーが押されたかどうかを
  チェックします：</p>

  <pre><code class="language-js">if (keyCode === 13) {
    // エンターキーが押された場合に実行するコード。
  }

  </code></pre>

  <p>同じコードは、値が13のシステム変数<code>ENTER</code>を使用して
  より明確に書くことができます：</p>

  <pre><code class="language-js">if (keyCode === ENTER) {
    // エンターキーが押された場合に実行するコード。
  }

  </code></pre>

  <p>システム変数<code>BACKSPACE</code>、<code>DELETE</code>、
  <code>ENTER</code>、<code>RETURN</code>、<code>TAB</code>、

  <code>ESCAPE</code>、<code>SHIFT</code>、<code>CONTROL</code>、
  <code>OPTION</code>、<code>ALT</code>、<code>UP_ARROW</code>、<code>DOWN_ARROW</code>、

  <code>LEFT_ARROW</code>、<code>RIGHT_ARROW</code>はすべて、特殊キーのキーコードの
  便利な省略形です。キーコードは

  <a href="http://keycode.info/">keycode.info</a>などのウェブサイトで確認できます。</p>

        } else if (keyCode === LEFT_ARROW) {
