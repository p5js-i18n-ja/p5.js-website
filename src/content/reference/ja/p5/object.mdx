description: "<p>データをキーと値のペアとして格納するコンテナです。</p>\n<p>オブジェクトは、他のオブジェクトを含む任意の型の関連データを整理するのに役立ちます。\nオブジェクトに格納された値は、キーと呼ばれる名前でアクセスできます。各\nキーと値のペアは「プロパティ」と呼ばれます。オブジェクトはPythonの辞書や\nJavaやRubyのマップに似ています。</p>\n<p>例えば、オブジェクトは犬の位置、サイズ、外見を含むことができます：</p>\n<pre><code class=\"language-js\">// dog変数を宣言し、オブジェクトを割り当てる。\nlet dog = { x: 50, y: 50, size: 20, emoji: '\U0001F436' };\n\n// テキストのスタイルを設定する。\ntextAlign(CENTER, CENTER);\ntextSize(dog.size);\n\n// 犬を描く。\ntext(dog.emoji, dog.x, dog.y);\n</code></pre>\n<p><code>dog</code>変数には4つのプロパティを持つオブジェクトが割り当てられています。オブジェクト\nは中括弧<code>{}</code>で宣言されます。値にはドット演算子を使ってアクセスできます\n（<code>dog.size</code>のように）。上の例では、キー<code>size</code>は\n値<code>20</code>に対応しています。オブジェクトは最初は空にすることもできます：</p>\n<pre><code class=\"language-js\">// cat変数を宣言し、空のオブジェクトを割り当てる。\nlet cat = {};\n\n// オブジェクトにプロパティを追加する。\ncat.x = 50;\ncat.y = 50;\ncat.size = 20;\ncat.emoji = '\U0001F431';\n\n// テキストのスタイルを設定する。\ntextAlign(CENTER, CENTER);\ntextSize(cat.size);\n\n// 猫を描く。\ntext(cat.emoji, cat.x, cat.y);\n</code></pre>\n<p>オブジェクトのデータはスケッチの実行中に更新できます。例えば、<code>cat</code>は\n<code>dog</code>から逃げるために位置を更新することができます：</p>\n<pre><code class=\"language-js\">// 右に走る。\ncat.x += 5;\n</code></pre>\n<p>必要に応じて、オブジェクトの値には角括弧<code>[]</code>と文字列を使用して\nドット記法の代わりにアクセスすることもできます：</p>\n<pre><code class=\"language-js\">// 右に走る。\ncat[\"x\"] += 5;\n</code></pre>\n<p>この構文は、<code>cat['height (m)']</code>のように、キーの名前にスペースが含まれている場合に\n役立ちます。</p>\n"
