description: >
  <p>キャンバスの背景に使用される色を設定します。</p>

  <p>デフォルトでは、背景は透明です。<code>background()</code>は通常、

  <a href="/reference/p5/draw/">draw()</a>内で使用され、各フレームの

  開始時にディスプレイウィンドウをクリアします。また、

  <a href="/reference/p5/setup/">setup()</a>内で使用して、アニメーションの

  最初のフレームで背景を設定することもできます。</p>

  <p>1つのパラメータを持つ<code>background()</code>のバージョンは、値を4つの方法で

  解釈します。パラメータが<code>Number</code>の場合、グレースケール値として

  解釈されます。パラメータが<code>String</code>の場合、CSS色文字列として

  解釈されます。RGB、RGBA、HSL、HSLA、16進数、および名前付きの色文字列が

  サポートされています。パラメータが<a href="/reference/p5/p5.Color">p5.Color</a>

  オブジェクトの場合、背景色として使用されます。パラメータが

  <a href="/reference/p5/p5.Image">p5.Image</a>オブジェクトの場合、背景画像として

  使用されます。</p>

  <p>2つのパラメータを持つ<code>background()</code>のバージョンは、最初のパラメータを

  グレースケール値として解釈します。2番目のパラメータはアルファ（透明度）

  値を設定します。</p>

  <p>3つのパラメータを持つ<code>background()</code>のバージョンは、現在の

  <a href="/reference/p5/colorMode/">colorMode()</a>に応じて、それらをRGB、

  HSB、またはHSL色として解釈します。デフォルトでは、色はRGB値で

  指定されます。<code>background(255, 204, 0)</code>を呼び出すと、背景が

  明るい黄色に設定されます。</p>
