description: >
  <p>シンプルな音声/動画再生のための <code><video></code> 要素を作成します。</p>

  <p><code>createVideo()</code> は新しい

  <a href="/reference/p5/p5.MediaElement">p5.MediaElement</a> オブジェクトを返します。
  動画はデフォルトで表示されます。

  <code>video.hide()</code> を呼び出すことで非表示にし、

  <a href="/reference/p5/image/">image()</a> を使用してキャンバスに描画することができます。</p>

  <p>最初のパラメータ <code>src</code> は動画へのパスです。単一の文字列が

  渡された場合（例：<code>'/assets/topsecret.mp4'</code>）、1つの動画が読み込まれます。

  文字列の配列を使用して、同じ動画を異なる形式で読み込むことができます。

  例えば、<code>['/assets/topsecret.mp4', '/assets/topsecret.ogv',
  '/assets/topsecret.webm']</code>

  のようにします。これは、異なる機能を持つ様々なブラウザで動画を

  再生できるようにするのに役立ちます。サポートされている形式の詳細については、

  <a
  href="https://developer.mozilla.org/en-US/docs/Web/HTML/Supported_media_formats/">MDN</a>

  を参照してください。</p>

  <p>2番目のパラメータ <code>callback</code> はオプションです。これは動画が

  再生可能になったときに呼び出される関数です。</p>
