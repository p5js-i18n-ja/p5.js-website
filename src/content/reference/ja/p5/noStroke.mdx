description: >
  <p>図形の輪郭を無効にします。</p>

  <p><code>noStroke()</code>を呼び出すと、その後に描画されるすべての図形が

  輪郭なしで描画されます。輪郭を再び有効にするには、<code>stroke()</code>を使用します。</p>

  <p>この関数は、<code>fill()</code>と組み合わせて使用すると、

  塗りつぶしのみの図形を作成できます。</p>
line: '1637'

itemtype: method
example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);
      background(200);

      // 輪郭なしの正方形
      noStroke();
      fill(0, 0, 255);
      square(20, 20, 60);

      describe('青く塗りつぶされた正方形（輪郭なし）');
    }
    </code>
    </div>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);
      background(200);

      // 輪郭ありの円
      stroke(255, 0, 0);
      circle(33, 50, 40);

      // 輪郭なしの円
      noStroke();
      circle(67, 50, 40);

      describe('赤い輪郭の円と、輪郭のない黒く塗りつぶされた円');
    }
    </code>
    </div>
