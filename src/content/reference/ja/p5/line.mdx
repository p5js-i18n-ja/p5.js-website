description: >
  <p>2点間に直線を描きます。</p>

  <p>線のデフォルトの幅は1ピクセルです。4つのパラメータを持つ
  <code>line()</code>のバージョンは2Dで線を描きます。線の色を変更するには、
  <a href="/reference/p5/stroke/">stroke()</a>関数を使用します。線の幅を変更するには、
  <a href="/reference/p5/strokeWeight/">strokeWeight()</a>関数を使用します。線は
  塗りつぶすことができないため、<a href="/reference/p5/fill/">fill()</a>関数は
  線の色に影響を与えません。</p>

  <p>6つのパラメータを持つ<code>line()</code>のバージョンは、3D空間に線を描くことができます。
  これを行うには、<a href="/reference/p5/createCanvas/">createCanvas()</a>に
  <code>WEBGL</code>引数を追加する必要があります。</p>

    function draw() {
      background(200);

      // y軸を中心に回転します。
      rotateY(frameCount * 0.01);

      // 線を描きます。
      line(0, 0, 0, 30, 20, -10);

      // 中心の球を描きます。
      sphere(10);

      // 2つ目の点に移動します。
      translate(30, 20, -10);

      // 右下の球を描きます。
      sphere(10);
    }
