  <p><a href="#/p5/if-else">if-else</a> 文は、コードのフローを制御するのに役立ちます。</p>

  <p></p>

  <p>「if」に続く括弧の間に条件が置かれ、その条件が <a
  href="https://developer.mozilla.org/ja/docs/Glossary/truthy">truthy（真値）</a>
  に評価されると、次の中括弧の間のコードが実行されます。逆に、条件が <a
  href="https://developer.mozilla.org/ja/docs/Glossary/Falsy">falsy（偽値）</a>
  に評価されると、「else」ブロックの中括弧の間のコードが代わりに実行されます。else ブロックの記述はオプションです。</p>

  <p><a
  href="https://developer.mozilla.org/ja/docs/Web/JavaScript/Reference/Statements/if...else">MDN
  エントリ</a> より: if...else 文は、指定された条件が真値ならば文を実行します。条件が偽値なら、オプションの else
  節にあるもう一方の文を実行します。</p>


  <hr/>

  <p>The <a href="/reference/p5/if-else">if-else</a> statement helps control the
  flow of your code.</p>

  <p>A condition is placed between the parenthesis following 'if',

  when that condition evalues to <a
  href="https://developer.mozilla.org/en-US/docs/Glossary/truthy">truthy</a>,

  the code between the following curly braces is run.

  Alternatively, when the condition evaluates to <a
  href="https://developer.mozilla.org/en-US/docs/Glossary/Falsy">falsy</a>,

  the code between the curly braces of 'else' block is run instead. Writing an

  else block is optional.</p>

  <p>From <a
  href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/if...else">the
  MDN entry</a>:

  The 'if' statement executes a statement if a specified condition is truthy.

  If the condition is falsy, another statement can be executed</p>
