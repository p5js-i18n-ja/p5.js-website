  <p>Tracks the amount of time, in milliseconds, it took for <a
  href="#/p5/draw">draw</a> to draw the previous frame. <code>deltaTime</code>
  is useful for simulating physics.</p>


  <hr/>

  <p>A <code>Number</code> variable that tracks the number of milliseconds it
  took to draw

  the last frame.</p>

  <p><code>deltaTime</code> contains the amount of time it took

  <a href="/reference/p5/draw">draw()</a> to execute during the previous frame.
  It's

  useful for simulating physics.</p>

      // Use deltaTime to calculate
      // a change in position.
      let deltaX = speed * deltaTime;

      // Update the x variable.
      x += deltaX;

      // Reset x to 0 if it's
      // greater than 100.
