description: >
  <p>配列の要素をシャッフルします。</p>

  <p>最初のパラメータ<code>array</code>は、シャッフルする配列です。例えば、

  <code>shuffle(myArray)</code>を呼び出すと、<code>myArray</code>の要素が

  シャッフルされます。デフォルトでは、元の配列は変更されません。代わりに、

  コピーが作成され、シャッフルされて返されます。</p>

  <p>2番目のパラメータ<code>modify</code>はオプションです。<code>true</code>を渡すと、

  <code>shuffle(myArray, true)</code>のように、配列はコピーを作成せずに

  その場でシャッフルされます。</p>

    function setup() {

      let colors = ['red', 'orange', 'yellow', 'green', 'blue', 'indigo', 'violet'];

      let shuffledColors = shuffle(colors);

        let x = (i + 1) * 12.5;

        // 円のスタイルを設定します。
        let c = shuffledColors[i];
