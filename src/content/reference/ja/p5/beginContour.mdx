description: >
  <p>平面形状内に穴を作り始めます。</p>

  <p><code>beginContour()</code>と<a
  href="/reference/p5/endContour/">endContour()</a>

  関数は、平面的なカスタム形状内に負の空間を作成できます。

  <code>beginContour()</code>は負の空間に頂点の追加を開始し、

  <a href="/reference/p5/endContour/">endContour()</a>はその追加を停止します。

  <code>beginContour()</code>と<a
  href="/reference/p5/endContour/">endContour()</a>は

  <a href="/reference/p5/beginShape/">beginShape()</a>と

  <a href="/reference/p5/endShape/">endShape()</a>の間で呼び出す必要があります。</p>

  <p><a href="/reference/p5/translate/">translate()</a>、

  <a href="/reference/p5/rotate/">rotate()</a>、<a
  href="/reference/p5/scale/">scale()</a>

  などの変換は<code>beginContour()</code>と

  <a href="/reference/p5/endContour/">endContour()</a>の間では機能しません。また、

  <a href="/reference/p5/ellipse/">ellipse()</a>や

  <a href="/reference/p5/rect/">rect()</a>などの他の形状を<code>beginContour()</code>と

  <a href="/reference/p5/endContour/">endContour()</a>の間で使用することもできません。</p>

  <p>注意：負の空間を定義する頂点は、外側の形状とは逆の方向に「巻く」必要があります。

  まず、外側の形状の頂点を時計回りの順序で描画します。

  次に、負の空間の頂点を反時計回りの順序で描画します。</p>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // 形状の描画を開始します。
      beginShape();

      // 外側の頂点、時計回りの巻き方。
      vertex(10, 10);
      vertex(90, 10);
      vertex(90, 90);
      vertex(10, 90);

      // 内側の頂点、反時計回りの巻き方。
      beginContour();
      vertex(30, 30);
      vertex(30, 70);
      vertex(70, 70);
      vertex(70, 30);
      endContour();

      // 形状の描画を終了します。
      endShape(CLOSE);

      describe('灰色の背景に中央に正方形の穴がある白い正方形が描かれています。');
    }
    </code>
