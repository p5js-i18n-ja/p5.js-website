description: >
  <p>3Dスケッチの現在のカメラに対して正投影を設定します。</p>

  <p>正投影では、同じサイズの形状は、カメラからの距離に関係なく、

  常に同じサイズに見えます。</p>

  <p><code>ortho()</code>は、カメラの視錐台を切頂四角錐から

  直方体に変更することで、カメラの視点を変更します。カメラは

  視錐台の前に配置され、視錐台のニアプレーンと

  ファープレーンの間にあるすべてを表示します。<code>ortho()</code>には、

  視錐台を定義するための6つのオプションパラメータがあります。</p>

  <p>最初の4つのパラメータ、<code>left</code>、<code>right</code>、<code>bottom</code>、<code>top</code>は、

  視錐台の側面、底面、上面の座標を設定します。例えば、

  <code>ortho(-100, 100, 200, -200)</code>を呼び出すと、幅200ピクセル、

  高さ400ピクセルの視錐台が作成されます。デフォルトでは、これらの座標は

  スケッチの幅と高さに基づいて設定され、

  <code>ortho(-width / 2, width / 2, -height / 2, height / 2)</code>のようになります。</p>

  <p>最後の2つのパラメータ、<code>near</code>と<code>far</code>は、

  カメラからの視錐台のニアプレーンとファープレーンの距離を設定します。例えば、

  <code>ortho(-100, 100, 200, 200, 50, 1000)</code>を呼び出すと、幅200ピクセル、

  高さ400ピクセル、カメラから50ピクセルの位置から始まり、1,000

  ピクセルの位置で終わる視錐台が作成されます。デフォルトでは、<code>near</code>と<code>far</code>は

  それぞれ0と<code>max(width, height) + 800</code>に設定されます。</p>

  <p>注意：<code>ortho()</code>はWebGLモードでのみ使用できます。</p>

      ortho();

      translate(-10, 10, 600);

      // 座標系を回転させる。
      rotateY(-0.1);
