  <p>システム変数pRotationXは、常に現在のフレームの前のフレームにおける

  デバイスのx軸に沿った回転を含んでいます。

  スケッチの<a href="/reference/p5/angleMode/">angleMode()</a>がDEGREESに設定されている場合、

  値は-180から180になります。RADIANSに設定されている場合、値は

  -PIからPIになります。</p>

  <p>pRotationXは、rotationXと共に使用して、デバイスのX軸に沿った

  回転方向を判断することもできます。</p>

    // rotationX - pRotationX < 0が真かどうかを
    // 単純にif文で確認するだけで、ほとんどの場合
    // 回転方向を判断するのに十分です。

    // 角度が一周する場合に対応するために
    // 追加のロジックが必要です。
    let rotateDirection = 'clockwise';

    // 物事を単純化するための簡単な範囲変換。
    // これは絶対に必要ではありませんが、
    // そうでない場合はロジックが異なります。

    let rX = rotationX + 180;
    let pRX = pRotationX + 180;

    if ((rX - pRX > 0 && rX - pRX < 270) || rX - pRX < -270) {
      rotateDirection = 'clockwise';
    } else if (rX - pRX < 0 || rX - pRX > 270) {
      rotateDirection = 'counter-clockwise';
    }

    print(rotateDirection);
