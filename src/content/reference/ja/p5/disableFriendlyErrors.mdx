  <hr/>

  <p>Turns off the parts of the Friendly Error System (FES) that impact
  performance.</p>

  <p>The <a
  href="https://github.com/processing/p5.js/blob/main/contributor_docs/friendly_error_system.md"
  target="_blank">FES</a>

  can cause sketches to draw slowly because it does extra work behind the

  scenes. For example, the FES checks the arguments passed to functions,

  which takes time to process. Disabling the FES can significantly improve

  performance by turning off these checks.</p>
