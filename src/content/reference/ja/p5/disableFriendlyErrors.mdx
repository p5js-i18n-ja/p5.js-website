  <p>パフォーマンスに影響を与えるフレンドリーエラーシステム（FES）の部分をオフにします。</p>

  <p><a
  href="https://github.com/processing/p5.js/blob/main/contributor_docs/friendly_error_system.md"
  target="_blank">FES</a>は、

  バックグラウンドで追加の作業を行うため、スケッチの描画速度を遅くする可能性があります。

  例えば、FESは関数に渡される引数をチェックしますが、これには処理時間がかかります。

  FESを無効にすることで、これらのチェックをオフにし、パフォーマンスを大幅に

  向上させることができます。</p>

    // FESを無効にする。
    p5.disableFriendlyErrors = true;

      createCanvas(100, 100);

      // 表示されるはずです。代わりに、何も起こらず、
      // 静かに失敗します。
      circle(50, 50);
