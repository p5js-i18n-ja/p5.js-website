description: >
  <p>文字列に正規表現を適用し、マッチした結果の配列を返します。</p>

  <p><code>match()</code>は正規表現（regex）を使用してテキスト内のパターンを照合します。

  例えば、正規表現<code>abc</code>は文字列内の正確な

  文字列<code>abc</code>を検索するのに使用できます。正規表現についての

  詳細は<a
  href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Regular_Expressions#tools"
  target="_blank">MDN</a>を参照してください。

  <code>matchAll()</code>は<a href="/reference/p5/match/">match()</a>とは異なり、

  最初のマッチだけでなく、すべてのマッチを返します。</p>

  <p>最初のパラメータ<code>str</code>は、検索対象の文字列です。</p>

  <p>2番目のパラメータ<code>regex</code>は、適用する正規表現を含む文字列です。

  例えば、<code>matchAll('p5*js is easier than abc123', '[a-z][0-9]')</code>を呼び出すと、

  2次元配列<code>[['p5'], ['c1']]</code>が返されます。</p>

  <p>注意：マッチが見つからない場合、空の配列<code>[]</code>が返されます。</p>

    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // 文字列変数を作成します。
      let string = 'p5*js is easier than abc123';

      // 小文字の後に数字が続く文字列を
      // マッチさせます。
      let matches = matchAll(string, '[a-z][0-9]');

      // マッチした配列をコンソールに表示します：
      // [['p5'], ['c1']]
      print(matches);

      // テキストのスタイルを設定します。
      textAlign(CENTER, CENTER);
      textSize(16);

      // マッチした配列をイテレートします。
      for (let i = 0; i < matches.length; i += 1) {

        // y座標を計算します。
        let y = (i + 1) * 33;

        // マッチした結果を表示します。
        text(matches[i], 50, y);
      }

      describe(
        '別々の行に書かれた"p5"と"c1"のテキスト。テキストは灰色の背景に黒で表示されています。'
      );
    }
