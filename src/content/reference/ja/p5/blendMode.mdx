description: >
  <p>キャンバスに追加されたときの色のブレンド方法を設定します。</p>

  <p>デフォルトでは、ソリッドカラーで描画すると、キャンバス上の現在のピクセル値が上書きされます。

  <code>blendMode()</code>は、色をブレンドするための多くのオプションを提供します。</p>

  <p>形状、画像、テキストは、キャンバスに描画するためのソースとして使用できます。

  ソースピクセルは、描画される場所のキャンバスピクセルの色を変更します。最終的な色は、

  ソースピクセルの色とキャンバスピクセルの色をブレンドすることによって得られます。ソースとキャンバスの

  ピクセルのRGBカラー値は、比較、加算、減算、乗算、除算されて、さまざまな効果を生み出します。赤の値は赤の値と、緑は緑と、青は青と比較されます。</p>

  <p>パラメータ<code>mode</code>は、ブレンドモードを設定します。たとえば、<code>blendMode(ADD)</code>を呼び出すと、ブレンドモードが<code>ADD</code>に設定されます。次のブレンドモードは、

  2DおよびWebGLモードの両方で利用可能です：</p>

  <ul>

  <li><code>BLEND</code>: ソースからの色値がキャンバスを上書きします。これがデフォルトモードです。</li>

  <li><code>ADD</code>: ソースからの色値がキャンバスからの値に加算されます。</li>

  <li><code>DARKEST</code>: もっとも暗い色値を保持します。</li>

  <li><code>LIGHTEST</code>: もっとも明るい色値を保持します。</li>

  <li><code>EXCLUSION</code>: <code>DIFFERENCE</code>に似ていますが、コントラストが少なくなります。</li>

  <li><code>MULTIPLY</code>: ソースからの色値がキャンバスからの値と乗算されます。結果は常に暗くなります。</li>

  <li><code>SCREEN</code>: すべての色値が反転され、次に乗算され、再び反転されます。結果は常に明るくなります。（<code>MULTIPLY</code>の反対）</li>

  <li><code>REPLACE</code>: 最後に描画されたソースがキャンバスの残りを完全に置き換えます。</li>

  <li><code>REMOVE</code>: 重なり合うピクセルは、完全に透明にすることによって削除されます。</li>

  </ul>

  <p>次のブレンドモードは、2Dモードでのみ利用可能です：</p>

  <ul>

  <li><code>DIFFERENCE</code>: ソースからの色値がキャンバスからの値から引かれます。差が負の数の場合は、正の数に変換されます。</li>

  <li><code>OVERLAY</code>: <code>MULTIPLY</code>と<code>SCREEN</code>を組み合わせます。キャンバスの暗い値は暗くなり、明るい値は明るくなります。</li>

  <li><code>HARD_LIGHT</code>: <code>MULTIPLY</code>と<code>SCREEN</code>を組み合わせます。ソースの暗い値は暗くなり、明るい値は明るくなります。</li>

  <li><code>SOFT_LIGHT</code>: <code>HARD_LIGHT</code>の柔らかいバージョンです。</li>

  <li><code>DODGE</code>: 明るいトーンを明るくし、コントラストを増加させます。ソースからの反転色値でキャンバスの色値を割ります。</li>

  <li><code>BURN</code>: 暗いトーンを暗くし、コントラストを増加させます。ソースの色値をキャンバスの反転色値で割り、その結果を反転させます。</li>

  </ul>

  <p>次のブレンドモードは、WebGLモードでのみ利用可能です：</p>

  <ul>

  <li><code>SUBTRACT</code>: ソースからのRGB値がキャンバスからの値から引かれます。差が負の数の場合は、正の数に変換されます。ソースとキャンバスからのアルファ（透明度）値は加算されます。</li>

  </ul>
