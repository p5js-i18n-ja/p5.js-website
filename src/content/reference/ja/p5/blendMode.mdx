description: >
  <p>キャンバスに色を追加する際の混合方法を設定します。</p>

  <p>デフォルトでは、単色で描画すると、キャンバス上の現在のピクセル値の上に

  塗り重ねられます。<code>blendMode()</code>は色を混合するための多くのオプションを提供します。</p>

  <p>形状、画像、テキストをキャンバスに描画するためのソースとして使用できます。

  ソースピクセルは、描画される場所のキャンバスピクセルの色を変更します。

  最終的な色は、ソースピクセルの色とキャンバスピクセルの色を

  ブレンドした結果です。ソースとキャンバスのピクセルのRGB色値が

  比較され、加算、減算、乗算、除算されて異なる効果を

  生み出します。赤は赤と、緑は緑と、青は青と比較されます。</p>

  <p>パラメータ<code>mode</code>はブレンドモードを設定します。例えば、

  <code>blendMode(ADD)</code>を呼び出すとブレンドモードを<code>ADD</code>に設定します。
  以下のブレンドモードは2DモードとWebGLモードの両方で利用可能です：</p>

  <ul>

  <li><code>BLEND</code>: ソースの色値がキャンバスの値を上書きします。これがデフォルトモードです。</li>

  <li><code>ADD</code>: ソースの色値がキャンバスの値に加算されます。</li>

  <li><code>DARKEST</code>: 最も暗い色値を保持します。</li>

  <li><code>LIGHTEST</code>: 最も明るい色値を保持します。</li>

  <li><code>EXCLUSION</code>: <code>DIFFERENCE</code>に似ていますが、コントラストが低くなります。</li>

  <li><code>MULTIPLY</code>: ソースの色値がキャンバスの値と乗算されます。結果は常に暗くなります。</li>

  <li><code>SCREEN</code>: すべての色値が反転され、乗算され、再び反転されます。結果は常に明るくなります。（<code>MULTIPLY</code>の反対）</li>

  <li><code>REPLACE</code>: 最後に描画されたソースがキャンバスの残りの部分を完全に置き換えます。</li>

  <li><code>REMOVE</code>: 重なり合うピクセルが完全に透明になることで削除されます。</li>

  </ul>

  <p>以下のブレンドモードは2Dモードでのみ利用可能です：</p>

  <ul>

  <li><code>DIFFERENCE</code>: ソースの色値がキャンバスの値から減算されます。差が負の数になる場合は正の数に変換されます。</li>

  <li><code>OVERLAY</code>: <code>MULTIPLY</code>と<code>SCREEN</code>を組み合わせます。キャンバスの暗い値はより暗くなり、明るい値はより明るくなります。</li>

  <li><code>HARD_LIGHT</code>: <code>MULTIPLY</code>と<code>SCREEN</code>を組み合わせます。ソースの暗い値はより暗くなり、明るい値はより明るくなります。</li>

  <li><code>SOFT_LIGHT</code>: <code>HARD_LIGHT</code>のソフトバージョンです。</li>

  <li><code>DODGE</code>: 明るい色調を明るくし、コントラストを増加させます。キャンバスの色値をソースの反転した色値で除算します。</li>

  <li><code>BURN</code>: 暗い色調を暗くし、コントラストを増加させます。ソースの色値をキャンバスの反転した色値で除算し、結果を反転します。</li>

  </ul>

  <p>以下のブレンドモードはWebGLモードでのみ利用可能です：</p>

  <ul>

  <li><code>SUBTRACT</code>: ソースのRGB値がキャンバスの値から減算されます。差が負の数になる場合は正の数に変換されます。ソースとキャンバスのアルファ（透明度）値は加算されます。</li>

  </ul>

    function setup() {
      createCanvas(100, 100);

      background(200);

      // デフォルトのブレンドモードを使用します。
      blendMode(BLEND);

      // 線のスタイルを設定します。
      strokeWeight(30);

      // 青い線を描画します。
      stroke('blue');
      line(25, 25, 75, 75);

      // 赤い線を描画します。
      stroke('red');
      line(75, 25, 25, 75);
