description: >
  <p>カスタム形状に頂点の追加を開始します。</p>

  <p><code>beginShape()</code>と<a
  href="/reference/p5/endShape/">endShape()</a>関数は、

  2Dまたは3Dでカスタム形状を作成できます。<code>beginShape()</code>は

  カスタム形状への頂点の追加を開始し、<a
  href="/reference/p5/endShape/">endShape()</a>は

  その追加を停止します。</p>

  <p>パラメータ<code>kind</code>は、作成する形状の種類を設定します。デフォルトでは、

  任意の不規則な多角形を描画できます。kindに使用できるモードは以下の通りです：</p>

  <ul>

  <li><code>POINTS</code>：一連の点を描画します。</li>

  <li><code>LINES</code>：一連の接続されていない線分を描画します。</li>

  <li><code>TRIANGLES</code>：一連の個別の三角形を描画します。</li>

  <li><code>TRIANGLE_FAN</code>：最初の頂点を共有する一連の接続された三角形をファン状に描画します。</li>

  <li><code>TRIANGLE_STRIP</code>：一連の接続された三角形をストリップ状に描画します。</li>

  <li><code>QUADS</code>：一連の個別の四辺形（クアッド）を描画します。</li>

  <li><code>QUAD_STRIP</code>：隣接する辺を使用して次のクアッドを形成するクアッドストリップを描画します。</li>

  <li><code>TESS</code>：明示的なテッセレーションによって塗りつぶし曲線を作成します（WebGLのみ）。</li>

  </ul>

  <p><code>beginShape()</code>を呼び出した後、

  <a href="/reference/p5/vertex/">vertex()</a>、

  <a href="/reference/p5/bezierVertex/">bezierVertex()</a>、

  <a href="/reference/p5/quadraticVertex/">quadraticVertex()</a>、および/または

  <a href="/reference/p5/curveVertex/">curveVertex()</a>を呼び出すことで形状を構築できます。

  <a href="/reference/p5/endShape/">endShape()</a>を呼び出すと、形状への頂点の

  追加が停止します。各形状は現在のストロークカラーで輪郭が描かれ、

  現在のフィルカラーで塗りつぶされます。</p>

  <p><a href="/reference/p5/translate/">translate()</a>、

  <a href="/reference/p5/rotate/">rotate()</a>、

  <a href="/reference/p5/scale/">scale()</a>などの変換は、<code>beginShape()</code>と

  <a href="/reference/p5/endShape/">endShape()</a>の間では機能しません。また、

  <a href="/reference/p5/ellipse/">ellipse()</a>や

  <a href="/reference/p5/rect/">rect()</a>などの他の形状を<code>beginShape()</code>と

  <a href="/reference/p5/endShape/">endShape()</a>の間で使用することもできません。</p>
