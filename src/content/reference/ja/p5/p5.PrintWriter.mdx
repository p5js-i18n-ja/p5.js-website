  <p>プリントストリームを記述するクラスです。</p>

  一連のテキストデータをユーザーのコンピュータに保存する方法を提供します。
  これはテキスト出力の精密な制御を可能にする低レベルのオブジェクトです。
  <a href="/reference/p5/saveStrings/">saveStrings()</a> や
  <a href="/reference/p5/saveJSON/">saveJSON()</a> などの関数は、
  単純なファイル保存にはより使いやすいです。</p>

  <p>注意: このクラスのインスタンスを作成するには、
  <a href="/reference/p5/createWriter/">createWriter()</a> を使用することをお勧めします。</p>
