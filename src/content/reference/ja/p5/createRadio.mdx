description: >
  <p>ラジオボタン要素を作成します。</p>

  <p>パラメータはオプションです。文字列を渡した場合、例えば

  <code>let myRadio = createSelect('food')</code>のように、各ラジオオプションは

  <code>"food"</code>を<code>name</code>パラメータとして持ちます：<code><input
  name="food"></code>。

  既存の<code><div></div></code>または<code><span></span></code>

  要素を渡した場合、例えば<code>let myRadio = createSelect(container)</code>のように、
  その要素が

  ラジオボタンの親要素となります。</p>

  <p>ラジオボタンは<a href="/reference/p5/p5.Element">p5.Element</a>クラスを拡張し、

  オプションを管理するための便利なメソッドをいくつか提供します：</p>

  <ul>

  <li><code>myRadio.option(value, [label])</code>はメニューにオプションを追加します。
  最初のパラメータ<code>value</code>は、オプションの値とラベルを設定する文字列です。
  2番目のパラメータ<code>label</code>はオプションです。提供された場合、<code>value</code>
  に対して表示されるラベルを設定します。<code>value</code>を持つオプションが既に存在する場合、
  そのラベルが変更され、その値が返されます。</li>

  <li><code>myRadio.value()</code>は現在選択されているオプションの値を返します。</li>

  <li><code>myRadio.selected()</code>は現在選択されているオプションを返します。</li>

  <li><code>myRadio.selected(value)</code>は指定されたオプションを選択し、
  <a href="https://developer.mozilla.org/en-US/docs/Web/API/HTMLInputElement/"
  target="_blank"><code>HTMLInputElement</code></a>として返します。</li>

  <li><code>myRadio.disable(shouldDisable)</code>は<code>true</code>が渡された場合
  ラジオボタンを無効化し、<code>false</code>が渡された場合は有効化します。</li>

  </ul>

      myRadio.disable(true);
