description: >
  <p>消去モードを開始します。</p>

  <p><code>erase()</code>を呼び出すと、その後のすべての描画が既存のピクセルを消去します。

  <code>noErase()</code>を呼び出すまで、消去モードが継続します。</p>

  <p><code>erase()</code>の動作は、現在のレンダラーによって異なります：</p>

  <p>P2Dレンダラー（デフォルト）：

  消去された領域は透明になります。</p>

  <p>WEBGLレンダラー：

  消去された領域は背景色になります。</p>

  <p>オプションのパラメータを使用して、消去の強度を設定できます。

  値が高いほど、より多くのピクセルが消去されます。</p>
line: '1337'

example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);
      background(235, 145, 15);
      noStroke();
    }

    function draw() {
      background(235, 145, 15);
      circle(33, 50, 60);

      // 円を部分的に消去
      erase();
      circle(67, 50, 60);
      noErase();

      describe('オレンジ色の背景に2つの重なる円。右側の円は部分的に消去されています。');
    }
    </code>
    </div>

      background(235, 145, 15);

      // 円を完全に消去

      circle(67, 50, 60);
      noErase();

      describe('オレンジ色の背景に2つの円。左側の円は黒で、右側の円は完全に消去されています。');
    }
    </code>
