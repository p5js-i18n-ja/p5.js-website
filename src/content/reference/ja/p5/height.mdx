  <p>A numeric variable that stores the height of the drawing canvas. Its
  default value is 100.</p>

  <p>Calling <a href="#/p5/createCanvas">createCanvas()</a> or <a
  href="#/p5/resizeCanvas">resizeCanvas()</a> changes the value of
  <code>height</code>. Calling <a href="#/p5/noCanvas">noCanvas()</a> sets its
  value to 0.</p>


  <hr/>

  <p>A <code>Number</code> variable that stores the height of the canvas in
  pixels.</p>

  <p><code>height</code>'s default value is 100. Calling

  <a href="/reference/p5/createCanvas/">createCanvas()</a> or

  <a href="/reference/p5/resizeCanvas/">resizeCanvas()</a> changes the value of

  <code>height</code>. Calling <a href="/reference/p5/noCanvas/">noCanvas()</a>
  sets its value to

  0.</p>

    // If the mouse is pressed, reisze
    // the canvas and display its new
    // height.
    function mousePressed() {
      if (mouseX > 0 && mouseX < width && mouseY > 0 && mouseY < height) {
        resizeCanvas(100, 50);
        background(200);
        text(height, 42, 27);
