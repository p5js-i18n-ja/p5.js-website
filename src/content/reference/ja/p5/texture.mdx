description: >
  <p>形状に適用されるテクスチャを設定します。</p>

  <p>テクスチャは形状を包む皮のようなものです。<code>texture()</code>は

  <a href="/reference/p5/square/">square()</a>や

  <a href="/reference/p5/sphere/">sphere()</a>などの組み込み形状や、

  <a href="/reference/p5/buildGeometry/">buildGeometry()</a>などの関数で

  作成されたカスタム形状で動作します。

  <a href="/reference/p5/beginShape/">beginShape()</a>で作成されたジオメトリに

  テクスチャを適用するには、各

  <a href="/reference/p5/vertex/">vertex()</a>呼び出しにuvコーディネートを

  渡す必要があります。</p>

  <p>パラメータ<code>tex</code>は適用するテクスチャです。
  <code>texture()</code>は画像、動画、

  <a href="/reference/p5/p5.Graphics">p5.Graphics</a>や

  <a href="/reference/p5/p5.Framebuffer">p5.Framebuffer</a>オブジェクトなどの

  オフスクリーンレンダラーを含む様々なソースを使用できます。</p>

  <p><a href="/reference/p5/beginShape/">beginShape()</a>で作成されたジオメトリに

  テクスチャを適用するには、<a href="/reference/p5/vertex/">vertex()</a>で

  uvコーディネートを指定する必要があります。</p>

  <p>注意：<code>texture()</code>はWebGLモードでのみ使用できます。</p>

    }

    function setup() {

      texture(vid);

      // uvコーディネートを使用してカスタム形状を描画します。

      vertex(-40, -40, 0, 0);
      vertex(40, -40, 1, 0);

      vertex(-40, 40, 0, 1);
