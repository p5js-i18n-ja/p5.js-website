description: >
  <p>ベジエ曲線に接する線上の座標を計算します。</p>

  <p>接線は曲線の表面をなぞります。接線の傾きは、交点での曲線の傾きと等しくなります。</p>

  <p><code>bezierTangent()</code>は、ベジエ曲線のアンカーポイントと制御点を使用して、

  接線上の座標を計算します。<a href="/reference/p5/bezier/">bezier()</a>関数と

  同じ順序でポイントを指定することが期待されます。<code>bezierTangent()</code>は

  一度に1つの軸に対して機能します。アンカーポイントと制御点の

  x座標を渡すと、接線上のポイントのx座標が計算されます。

  y座標を渡すと、接線上のポイントのy座標が計算されます。</p>

  <p>最初のパラメータ<code>a</code>は、最初のアンカーポイントの座標です。</p>

  <p>2番目と3番目のパラメータ<code>b</code>と<code>c</code>は、制御点の座標です。</p>

  <p>4番目のパラメータ<code>d</code>は、最後のアンカーポイントの座標です。</p>

  <p>5番目のパラメータ<code>t</code>は、曲線に沿って補間する量です。0は

  最初のアンカーポイント、1は2番目のアンカーポイント、0.5はその中間点です。</p>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // 曲線のアンカーポイントと制御点の座標を設定します。
      let x1 = 85;
      let x2 = 10;
      let x3 = 90;
      let x4 = 15;
      let y1 = 20;
      let y2 = 10;
      let y3 = 90;
      let y4 = 80;

      // 曲線のスタイルを設定します。
      noFill();

      // 曲線を描画します。
      bezier(x1, y1, x2, y2, x3, y3, x4, y4);

      // 曲線のパスに沿って接線を描画します。
      fill(255);

      // 右上の円。
      stroke(0);
      let x = bezierPoint(x1, x2, x3, x4, 0);
      let y = bezierPoint(y1, y2, y3, y4, 0);
      circle(x, y, 5);

      // 右上の接線。
      // 接点をスケーリングして短い線を描画します。
      stroke(255, 0, 0);
      let tx = 0.1 * bezierTangent(x1, x2, x3, x4, 0);
      let ty = 0.1 * bezierTangent(y1, y2, y3, y4, 0);
      line(x + tx, y + ty, x - tx, y - ty);

      // 中央の円。
      stroke(0);
      x = bezierPoint(x1, x2, x3, x4, 0.5);
      y = bezierPoint(y1, y2, y3, y4, 0.5);
      circle(x, y, 5);

      // 中央の接線。
      // 接点をスケーリングして短い線を描画します。
      stroke(255, 0, 0);
      tx = 0.1 * bezierTangent(x1, x2, x3, x4, 0.5);
      ty = 0.1 * bezierTangent(y1, y2, y3, y4, 0.5);
      line(x + tx, y + ty, x - tx, y - ty);

      // 左下の円。
      stroke(0);
      x = bezierPoint(x1, x2, x3, x4, 1);
      y = bezierPoint(y1, y2, y3, y4, 1);
      circle(x, y, 5);

      // 左下の接線。
      // 接点をスケーリングして短い線を描画します。
      stroke(255, 0, 0);
      tx = 0.1 * bezierTangent(x1, x2, x3, x4, 1);
      ty = 0.1 * bezierTangent(y1, y2, y3, y4, 1);
      line(x + tx, y + ty, x - tx, y - ty);

      describe(
        'グレーの正方形上に黒のS字カーブ。カーブの端点と中心が白い円で示されています。赤い接線が白い円から伸びています。'
      );
    }
    </code>
