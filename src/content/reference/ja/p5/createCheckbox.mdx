  <p>チェックボックス <code><input></input></code> 要素を作成します。</p>

  <p>チェックボックスは <a href="/reference/p5/p5.Element">p5.Element</a>
  クラスを拡張し、

  <code>checked()</code> メソッドを持ちます。<code>myBox.checked()</code> を呼び出すと、
  ボックスが

  チェックされている場合は <code>true</code> を、そうでない場合は <code>false</code> を返します。</p>

  <p>最初のパラメータ <code>label</code> はオプションです。チェックボックスの横に
  表示するラベルを

  設定する文字列です。</p>

  <p>2番目のパラメータ <code>value</code> もオプションです。チェックボックスの値を
  設定する

  ブール値です。</p>

    let checkbox;

    function setup() {
      createCanvas(100, 100);

      // チェックボックスを作成し、キャンバスの下に配置します。
      checkbox = createCheckbox();
      checkbox.position(0, 100);

      describe('黒い正方形とその下にチェックボックスがあります。ボックスがチェックされると正方形が白くなります。');

    function draw() {
      // チェックボックスを使用して背景色を設定します。

      // チェックボックスを使用して背景色を設定します。

        background(255);

        background(0);

    }
