description: >
  <p>スケッチが実行される前にアセットを読み込むために一度だけ呼び出される関数です。</p>

  <p><code>preload()</code>関数を宣言すると、<a href="/reference/p5/setup/">setup()</a>や

  <a href="/reference/p5/draw/">draw()</a>の前に自動的に一度実行されるコードブロックが設定されます。

  マルチメディアファイル、フォント、データ、3Dモデルなどのアセットを読み込むために使用されます：</p>

  <pre><code class="language-js">function preload() {
    // スケッチの残りの部分の前に実行されるコード。
  }

  </code></pre>

  <p><a href="/reference/p5/loadImage/">loadImage()</a>、

  <a href="/reference/p5/loadFont/">loadFont()</a>、

  <a href="/reference/p5/loadJSON/">loadJSON()</a>、

  <a href="/reference/p5/loadModel/">loadModel()</a>などの関数は、<code>preload()</code>内で

  呼び出された場合、読み込みが完了するかエラーが発生することが保証されています。

  これにより、スケッチの実行が開始されるときにアセットが利用可能であることが

  確保されます。</p>
