description: >
  <p>現在の URL パスを配列として取得します。</p>

  <p>注意: p5 エディターを使用している場合、スケッチが iframe
  に埋め込まれているため、この関数は空のオブジェクトを返します。エディターのプレゼント URL やシェア URL
  でスケッチを表示すると、正しく機能します。</p>

  <p>/// todo ///</p>

  <p>Returns the current <a
  href="https://developer.mozilla.org/en-US/docs/Learn/Common_questions/Web_mechanics/What_is_a_URL#path_to_resource"
  target="_blank">URL</a> path as an array of strings.</p>

  <p>For example, consider a sketch hosted at the URL
  <code>https://example.com/sketchbook</code>. Calling <code>getURLPath()</code>
  returns <code>['sketchbook']</code>. For a sketch hosted at the URL
  <code>https://example.com/sketchbook/monday</code>, <code>getURLPath()</code>
  returns <code>['sketchbook', 'monday']</code>.</p>
