description: >
  <p>文字列に正規表現を適用し、最初の一致を含む配列を返します。</p>

  <p><code>match()</code>は正規表現（regex）を使用してテキスト内のパターンを照合します。

  例えば、正規表現<code>abc</code>は文字列内の正確な

  文字列<code>abc</code>を検索するのに使用できます。正規表現についての

  詳細は<a
  href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Regular_Expressions#tools"
  target="_blank">MDN</a>を参照してください。</p>

  <p>最初のパラメータ<code>str</code>は検索対象の文字列です。</p>

  <p>2番目のパラメータ<code>regex</code>は適用する正規表現を含む文字列です。

  例えば、<code>match('Hello, p5*js!', '[a-z][0-9]')</code>を呼び出すと、

  配列<code>['p5']</code>が返されます。</p>

  <p>注意：一致するものが見つからない場合、<code>null</code>が返されます。</p>

    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // 文字列変数を作成します。
      let string = 'Hello, p5*js!';

      // 小文字の後に数字が続く文字を
      // マッチングします。
      let matches = match(string, '[a-z][0-9]');

      // マッチした配列をコンソールに表示します：
      // ['p5']
      print(matches);

      // テキストのスタイルを設定します。
      textAlign(CENTER, CENTER);
      textSize(16);

      // マッチした結果を表示します。
      text(matches, 50, 50);

      describe('灰色のキャンバスに黒で書かれた "p5" というテキスト。');
    }
