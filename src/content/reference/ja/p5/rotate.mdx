  <p>指定された角度パラメーターで図形を回転させます。</p>

  <p>この関数は <a href="#/p5/angleMode">angleMode</a> を考慮しているため、角度は RADIANS または
  DEGREES で入力できます。</p>

  <p>オブジェクトは常に原点に対する相対位置を中心に回転し、正の数値はオブジェクトを時計回りに回転させます。変換はその後のすべての操作に適用され、関数への後続の呼び出しは効果を累積します。たとえば、rotate(HALF_PI)を呼び出した後に
  rotate(HALF_PI)を呼び出すことは、rotate(PI)と同じです。すべての変換は、<a href="#/p5/draw">draw()</a>
  が再度始まるとリセットされます。</p>

  <p>技術的には、<a href="#/p5/rotate">rotate()</a> は現在の変換行列に回転行列を乗算します。この関数は、<a
  href="#/p5/push">push()</a> および <a href="#/p5/pop">pop()</a> でさらに制御できます。</p>


  <hr/>

  <p>Rotates the coordinate system.</p>

  <p>By default, the positive x-axis points to the right and the positive y-axis

  points downward. The <code>rotate()</code> function changes this orientation
  by

  rotating the coordinate system about the origin. Everything drawn after

  <code>rotate()</code> is called will appear to be rotated.</p>

  <p>The first parameter, <code>angle</code>, is the amount to rotate. For
  example, calling

  <code>rotate(1)</code> rotates the coordinate system clockwise 1 radian which
  is

  nearly 57˚. <code>rotate()</code> interprets angle values using the current

  <a href="/reference/p5/angleMode/">angleMode()</a>.</p>

  <p>The second parameter, <code>axis</code>, is optional. It's used to orient
  3D rotations

  in WebGL mode. If a <a href="/reference/p5/p5.Vector">p5.Vector</a> is passed,
  as in

  <code>rotate(QUARTER_PI, myVector)</code>, then the coordinate system will
  rotate

  <code>QUARTER_PI</code> radians about <code>myVector</code>. If an array of
  vector components is

  passed, as in <code>rotate(QUARTER_PI, [1, 0, 0])</code>, then the coordinate
  system

  will rotate <code>QUARTER_PI</code> radians about a vector with the components

  <code>[1, 0, 0]</code>.</p>

  <p>By default, transformations accumulate. For example, calling
  <code>rotate(1)</code>

  twice has the same effect as calling <code>rotate(2)</code> once. The

  <a href="/reference/p5/push/">push()</a> and <a
  href="/reference/p5/pop/">pop()</a> functions

  can be used to isolate transformations within distinct drawing groups.</p>

  <p>Note: Transformations are reset at the beginning of the draw loop. Calling

  <code>rotate(1)</code> inside the <a href="/reference/p5/draw/">draw()</a>
  function won't cause

  shapes to spin.</p>
