description: >
  <p>座標系を回転させます。</p>

  <p>デフォルトでは、正のx軸は右を向き、正のy軸は下を向いています。<code>rotate()</code>関数は

  原点を中心に座標系を回転させることで、この向きを変更します。

  <code>rotate()</code>が呼び出された後に描画されるものはすべて回転して表示されます。</p>

  <p>最初のパラメータ<code>angle</code>は回転量です。例えば、

  <code>rotate(1)</code>を呼び出すと、座標系が時計回りに1ラジアン（約57度）回転します。

  <code>rotate()</code>は現在の<a href="/reference/p5/angleMode/">angleMode()</a>を使用して

  角度値を解釈します。</p>

  <p>2番目のパラメータ<code>axis</code>はオプションです。WebGLモードで3D回転の向きを

  指定するために使用されます。<a href="/reference/p5/p5.Vector">p5.Vector</a>が渡された場合、

  例えば<code>rotate(QUARTER_PI, myVector)</code>のように、座標系は

  <code>myVector</code>を軸として<code>QUARTER_PI</code>ラジアン回転します。ベクトル成分の

  配列が渡された場合、例えば<code>rotate(QUARTER_PI, [1, 0, 0])</code>のように、座標系は

  <code>[1, 0, 0]</code>の成分を持つベクトルを軸として<code>QUARTER_PI</code>ラジアン

  回転します。</p>

  <p>デフォルトでは、変換は累積されます。例えば、<code>rotate(1)</code>を2回呼び出すことは、

  <code>rotate(2)</code>を1回呼び出すのと同じ効果があります。

  <a href="/reference/p5/push/">push()</a>関数と<a href="/reference/p5/pop/">pop()</a>関数を

  使用して、個別の描画グループ内で変換を分離できます。</p>

  <p>注意：変換はdrawループの開始時にリセットされます。<a href="/reference/p5/draw/">draw()</a>関数内で

  <code>rotate(1)</code>を呼び出しても、図形は回転し続けません。</p>
