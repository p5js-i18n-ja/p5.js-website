description: >
  <p>ある範囲の数値を別の範囲に再マッピングします。</p>

  <p>例えば、<code>map(2, 0, 10, 0, 100)</code>を呼び出すと20が返されます。最初の3つの

  引数は元の値を2に設定し、元の範囲を0から10に設定します。

  最後の2つの引数は目標範囲を0から100に設定します。20の目標範囲[0, 100]における

  位置は、2の元の範囲[0, 10]における位置に比例しています。</p>

  <p>6番目のパラメータ<code>withinBounds</code>はオプションです。デフォルトでは、<code>map()</code>は

  目標範囲外の値を返すことができます。例えば、

  <code>map(11, 0, 10, 0, 100)</code>は110を返します。6番目のパラメータとして<code>true</code>を渡すと、

  再マッピングされた値を目標範囲内に制限します。例えば、

  <code>map(11, 0, 10, 0, 100, true)</code>は100を返します。</p>
