description: >
  <p>弧を描画します。</p>

  <p>弧は、<code>x</code>、<code>y</code>、<code>w</code>、<code>h</code>パラメータで定義される楕円の一部です。
  <code>x</code>と<code>y</code>は弧の中心の位置を設定します。<code>w</code>と<code>h</code>は弧の幅と高さを設定します。
  詳細については<a href="/reference/p5/ellipse/">ellipse()</a>と
  <a href="/reference/p5/ellipseMode/">ellipseMode()</a>を参照してください。</p>

  <p>5番目と6番目のパラメータ、<code>start</code>と<code>stop</code>は、
  弧を描画する角度の範囲を設定します。弧は常に<code>start</code>から<code>stop</code>まで
  時計回りに描画されます。角度は常にラジアンで指定します。</p>

  <p>7番目のパラメータ<code>mode</code>はオプションです。これは弧の塗りつぶしスタイルを
  決定します。塗りつぶしモードには、半円（<code>OPEN</code>）、閉じた半円
  （<code>CHORD</code>）、または閉じたパイの切片（<code>PIE</code>）があります。</p>

  <p>8番目のパラメータ<code>detail</code>もオプションです。これはWebGLモードで
  弧を描画するために使用される頂点の数を決定します。デフォルト値は25です。
  50を超える詳細度では輪郭線は描画されません。</p>

    function setup() {
      createCanvas(100, 100);

      describe('黒い背景に黄色い円。円が口を開閉する。');
    }

    function draw() {
      background(0);

      // 弧のスタイルを設定
      noStroke();
      fill(255, 255, 0);

      // 開始角度と終了角度を更新
      let biteSize = PI / 16;
      let startAngle = biteSize * sin(frameCount * 0.1) + biteSize;
      let endAngle = TWO_PI - startAngle;

      // 弧を描画
      arc(50, 50, 80, 80, startAngle, endAngle, PIE);
    }
