  <p><a href="#/p5.Color">p5.Color</a>オブジェクト、色成分の配列、または CSS
  カラー文字列から色相値を抽出します。</p>

  <p></p>

  <p>Hue は HSB と HSL の両方に存在します。これは色相環上の色の位置を表しています。デフォルトでは、この関数は HSL
  に正規化された色相を返します。もし <a href="#/colorMode">colorMode()</a> が HSB に設定されている場合は、 HSB
  に正規化された色相を返します。</p>


  <hr/>

  <p>Gets the hue value of a color.</p>

  <p><code>hue()</code> extracts the hue value from a

  <a href="/reference/p5/p5.Color">p5.Color</a> object, an array of color
  components, or

  a CSS color string.</p>

  <p>Hue describes a color's position on the color wheel. By default,
  <code>hue()</code>

  returns a color's HSL hue in the range 0 to 360. If the

  <a href="/reference/colorMode/">colorMode()</a> is set to HSB or HSL, it
  returns the hue

  value in the given mode.</p>

      let c = color(0, 50, 100);

      // Draw the left rectangle.
      noStroke();
      fill(c);
      rect(15, 20, 35, 60);

      // Set 'hueValue' to 0.
      let hueValue = hue(c);

      // Draw the right rectangle.
      fill(hueValue);
      rect(50, 20, 35, 60);

      describe(
