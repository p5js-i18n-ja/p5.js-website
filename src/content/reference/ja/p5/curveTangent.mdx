description: >
  <p>スプライン曲線の接線上の座標を計算します。</p>

  <p>接線は曲線の表面をかすめます。接線の傾きは、交点での曲線の傾きと等しくなります。</p>

  <p><code>curveTangent()</code>は、スプライン曲線のアンカーポイントと制御点を使用して

  接線上の座標を計算します。<a href="/reference/p5/curve/">curve()</a>関数と同じ順序で

  点を指定することが期待されます。<code>curveTangent()</code>は一度に1つの軸に対して

  機能します。アンカーポイントと制御点のx座標を渡すと、接線上の点のx座標が

  計算されます。アンカーポイントと制御点のy座標を渡すと、接線上の点のy座標が

  計算されます。</p>

  <p>最初のパラメータ<code>a</code>は、最初の制御点の座標です。</p>

  <p>2番目と3番目のパラメータ<code>b</code>と<code>c</code>は、アンカーポイントの座標です。</p>

  <p>4番目のパラメータ<code>d</code>は、最後の制御点の座標です。</p>

  <p>5番目のパラメータ<code>t</code>は、曲線に沿って補間する量です。0は最初の

  アンカーポイント、1は2番目のアンカーポイント、0.5はその中間点です。</p>

    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // 曲線のアンカーポイントと制御点の座標を設定します。
      let x1 = 5;
      let y1 = 26;
      let x2 = 73;
      let y2 = 24;
      let x3 = 73;
      let y3 = 61;
      let x4 = 15;
      let y4 = 65;

      // 曲線を描画します。
      noFill();
      curve(x1, y1, x2, y2, x3, y3, x4, y4);

      // 曲線のパスに沿って接線を描画します。
      fill(255);

      // 上部の円。
      stroke(0);
      let x = curvePoint(x1, x2, x3, x4, 0);
      let y = curvePoint(y1, y2, y3, y4, 0);
      circle(x, y, 5);

      // 上部の接線。
      // より短い線を描画するために接点をスケーリングします。
      stroke(255, 0, 0);
      let tx = 0.2 * curveTangent(x1, x2, x3, x4, 0);
      let ty = 0.2 * curveTangent(y1, y2, y3, y4, 0);
      line(x + tx, y + ty, x - tx, y - ty);

      // 中央の円。
      stroke(0);
      x = curvePoint(x1, x2, x3, x4, 0.5);
      y = curvePoint(y1, y2, y3, y4, 0.5);
      circle(x, y, 5);

      // 中央の接線。
      // より短い線を描画するために接点をスケーリングします。
      stroke(255, 0, 0);
      tx = 0.2 * curveTangent(x1, x2, x3, x4, 0.5);
      ty = 0.2 * curveTangent(y1, y2, y3, y4, 0.5);
      line(x + tx, y + ty, x - tx, y - ty);

      // 下部の円。
      stroke(0);
      x = curvePoint(x1, x2, x3, x4, 1);
      y = curvePoint(y1, y2, y3, y4, 1);
      circle(x, y, 5);

      // 下部の接線。
      // より短い線を描画するために接点をスケーリングします。
      stroke(255, 0, 0);
      tx = 0.2 * curveTangent(x1, x2, x3, x4, 1);
      ty = 0.2 * curveTangent(y1, y2, y3, y4, 1);
      line(x + tx, y + ty, x - tx, y - ty);

      describe(
        '灰色の正方形上の黒い曲線。白い円が曲線に沿って前後に動きます。'
      );
    }
    </code>
