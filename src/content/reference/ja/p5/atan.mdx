  <p>The inverse of <a href="#/p5/tan">tan()</a>, returns the arc tangent of a
  value. This function expects input values in the range of -Infinity to
  Infinity. By default, <code>atan()</code> returns values in the range -π ÷ 2
  (about -1.57) to π ÷ 2 (about 1.57). If the <a
  href="#/p5/angleMode">angleMode()</a> is <code>DEGREES</code> then values are
  returned in the range -90 to 90.</p>


  <hr/>

  <p>Calculates the arc tangent of a number.</p>

  <p><code>atan()</code> is the inverse of <a
  href="/reference/p5/tan">tan()</a>. It expects input

  values in the range of -Infinity to Infinity. By default, <code>atan()</code>
  returns

  values in the range -π ÷ 2 (about -1.57) to π ÷ 2

  (about 1.57). If the <a href="/reference/p5/angleMode">angleMode()</a> is
  <code>DEGREES</code>

  then values are returned in the range -90 to 90.</p>

      text(`${round(a, 3)}`, 35, 25);
