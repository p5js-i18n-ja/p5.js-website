  <p>数値の逆正接（アークタンジェント）を計算します。</p>

  <p><code>atan()</code>は<a href="/reference/p5/tan/">tan()</a>の逆関数です。
  -無限大から無限大までの範囲の入力値を想定しています。デフォルトでは、<code>atan()</code>は
  -π ÷ 2（約-1.57）からπ ÷ 2（約1.57）の範囲の値を返します。
  <a href="/reference/p5/angleMode/">angleMode()</a>が<code>DEGREES</code>の場合、
  値は-90から90の範囲で返されます。</p>

    function setup() {
      createCanvas(100, 100);

      background(200);

      // tan()とatan()の値を計算します。
      let a = PI / 3;
      let t = tan(a);
      let at = atan(t);

      // 値を表示します。
      text(`${round(a, 3)}`, 35, 25);
      text(`${round(t, 3)}`, 35, 50);
      text(`${round(at, 3)}`, 35, 75);
