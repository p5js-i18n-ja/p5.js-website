  <p>正、負、またはゼロの値を取ることができる数値。</p>

  <p><code>Number</code>データ型は、位置、サイズ、色などの値を表現するのに便利です。
  数値は20のような整数や、12.34のような小数点数になります。例えば、円の位置とサイズは
  3つの数値で表現できます：</p>

  <pre><code class="language-js">circle(50, 50, 20);

  </code></pre>

  <pre><code class="language-js">circle(50, 50, 12.34);

  </code></pre>

  <p>数値は基本的な算術演算をサポートし、標準的な演算順序に従います：
  括弧、指数、乗算、除算、加算、減算（PEMDAS）。例えば、p5.jsの数値型のシステム変数と
  算術演算子を組み合わせて使用するのが一般的です：</p>

  <pre><code class="language-js">// 中心に円を描く

  circle(width / 2, height / 2, 20);

  </code></pre>

  <pre><code class="language-js">// 左から右に移動する円を描く

  circle(frameCount * 0.01, 50, 20);

  </code></pre>

  <p>以下は算術演算子の簡単な概要です：</p>

  <pre><code class="language-js">1 + 2 // 加算

  1 - 2 // 減算

  1 * 2 // 乗算

  1 / 2 // 除算

  1 % 2 // 剰余

  1 ** 2 // べき乗

  </code></pre>

  <p>算術を使用して数値変数を更新するのは一般的です。例えば、オブジェクトの位置は
  次のように更新できます：</p>

  <pre><code class="language-js">x = x + 1;

  </code></pre>

  <p>上記の文は<code>+</code>演算子を使用して変数<code>x</code>に1を加えています。
  加算代入演算子<code>+=</code>は同じ意味を表現します：</p>

  <pre><code class="language-js">x += 1;

  </code></pre>

  <p>以下は代入演算子の簡単な概要です：</p>

  <pre><code class="language-js">x += 2 // 加算代入

  x -= 2 // 減算代入
  
  x *= 2 // 乗算代入
  
  x /= 2 // 除算代入
  
  x %= 2 // 剰余代入

  </code></pre>

  <p>数値は
  <a
  href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators#relational_operators"
  target="_blank">関係演算子</a>
  <span class="code">></span>, <span class="code"><</span>, <span class="code">>=</span>, <span class="code"><=</span>,
  <code>===</code>, <code>!==</code>を使用して比較できます。例えば、スケッチの
  <a href="/reference/p5/frameCount/">frameCount</a>をタイマーとして使用できます：</=<></code></<></code></p>

  <pre><code class="language-js">if (frameCount > 1000) {
    text('Game over!', 50, 50);
  }

  </code></pre>

  <p><span class="code">frameCount > 1000</span>のような式は<code>true</code>または<code>false</code>の
  <code>Boolean</code>値に評価されます。関係演算子はすべて<code>Boolean</code>値を生成します：</p>

  <pre><code class="language-js">2 > 1 // true

  2 < 1 // false

  2 >= 2 // true

  2 <= 2 // true
 
  2 === 2 // true
 
  2 !== 2 // false</code></pre>

  <p>比較と条件についての詳細は<a href="/reference/p5/Boolean/">Boolean</a>を参照してください。</p>

  <p>注意：<code>==</code>と<code>!=</code>演算子も存在しますが、<code>=</code>が1つ少ないです。これらは使用しないでください。</p>

  <p>数値を含む式は、何か問題が発生した場合に特別な値を生成することもあります：</p>

  <pre><code class="language-js">sqrt(-1) // NaN
  1 / 0 // Infinity

  </code></pre>

  <p><code>NaN</code>は
  <a
  href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/NaN/"
  target="_blank">Not-A-Number（非数）</a>
  を表します。<code>NaN</code>は計算や変換が機能しない場合に現れます。
  <code>Infinity</code>はどの数よりも大きい値です。特定の計算で現れます。</p>
