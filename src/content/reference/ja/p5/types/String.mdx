  <p>文字の並び。</p>

  <p><code>String</code>データ型はテキストを扱うのに役立ちます。例えば、
  文字列にはウェルカムメッセージを含めることができます：</p>

  <pre><code class="language-js">// 文字列リテラルを使用する。

  text('Hello!', 10, 10);

  </code></pre>

  <pre><code class="language-js">// 文字列変数を作成する。

  let message = 'Hello!';


  // 文字列変数を使用する。

  text(message, 10, 10);

  </code></pre>

  <p>文字列を作成する最も一般的な方法は、以下のようにいくつかの形式の引用符を使用することです：</p>

  <pre><code class="language-js">text("hi", 50, 50);

  </code></pre>

  <pre><code class="language-js">text('hi', 50, 50);

  </code></pre>

  <pre><code class="language-js">text(`hi`, 50, 50);

  </code></pre>

  <p><code>"hi"</code>、<code>'hi'</code>、そして<code>`hi`</code>はすべて文字列リテラルです。「リテラル」とは、
  <code>text('hi', 50, 50)</code>のように実際に値が書かれていることを意味します。対照的に、
  <code>text(message, 50, 50)</code>は変数<code>message</code>を使用しているため、
  文字列リテラルではありません。</p>

  <p>シングルクォート<code>''</code>とダブルクォート<code>""</code>は同じ意味を持ちます。
  文字列に一方の種類の引用符が含まれている場合に選択肢があるのは便利です：</p>

  <pre><code class="language-js">text("What's up?", 50, 50);

  </code></pre>

  <pre><code class="language-js">text('Air quotes make you look "cool."', 50,
  50);

  </code></pre>

  <p>バッククォート<code>``</code>はテンプレートリテラルを作成します。テンプレートリテラルには多くの用途があります。
  例えば、必要に応じてシングルクォートとダブルクォートの両方を含めることができます：</p>

  <pre><code class="language-js">text(`"Don't you forget about me"`,  10, 10);

  </code></pre>

  <p>テンプレートリテラルは、次のように変数から文字列を作成する際に役立ちます：</p>

  <pre><code class="language-js">let size = random(10, 20);

  circle(50, 50, size);


  text(`The circle's diameter is ${size} pixels.`,  10, 10);

  </code></pre>

  <p><code>size</code>変数の値は、文字列が作成されるときに<code>${size}</code>に置き換えられます。
  <code>${}</code>は任意の値のプレースホルダーです。つまり、<code>${round(PI, 3)}</code>のように
  式を使用することができます。以下はすべて有効なテンプレートリテラルです：</p>

  <pre><code class="language-js">text(`π is about ${round(PI, 2)} pixels.`,  10,
  10);

  text(`It's ${mouseX < width / 2} that I'm on the left half of the canvas.`, 
  10, 30);

  </code></pre>

  <p>テンプレートリテラルには複数の変数を含めることができます：</p>

  <pre><code class="language-js">let x = random(0, 100);

  let y = random(0, 100);

  let size = random(10, 20);

  circle(x, y, size);


  text(`The circle at (${x}, ${y}) has a diameter of ${size} pixels.`,  10, 10);

  </code></pre>

  <p>テンプレートリテラルは、次のように複数行のテキストを作成する際にも役立ちます：</p>

  <pre><code class="language-js">let poem = `My sketch doesn't run;

  it waits for me patiently

  while bugs point the way.`;


  text(poem, 10, 10);

  </code></pre>
