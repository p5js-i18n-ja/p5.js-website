description: >
  <p>複数のデータを順序付けて保持するリスト。</p>

  <p>配列は関連するデータを保存するのに役立ちます。どのような型のデータでも含むことができます。例えば、配列には誰かのお気に入りの色のリストを文字列として含むことができます。配列は次のように作成されます：</p>

  <pre><code class="language-js">let myArray = ['deeppink', 'darkorchid',
  'magenta'];

  </code></pre>

  <p>配列内の各データは要素と呼ばれます。各要素には、その配列内のアドレス、つまりインデックスがあります。変数<code>myArray</code>は3つの<a href="/reference/p5/String/">String</a>要素、<code>'deeppink'</code>、<code>'darkorchid'</code>、<code>'magenta'</code>を持つ配列を参照します。配列はゼロから始まるインデックスを持ち、これは<code>'deeppink'</code>がインデックス0、<code>'darkorchid'</code>がインデックス1、'<code>magenta'</code>がインデックス2にあることを意味します。配列の要素には、次のようにインデックスを使ってアクセスできます：</p>

  <pre><code class="language-js">let zeroth = myArray[0]; // 'deeppink'

  let first = myArray[1]; // 'darkorchid'

  let second = myArray[2]; // 'magenta'

  </code></pre>

  <p>要素は、<a
  href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/push/"
  target="_blank">push()</a>メソッドを呼び出すことで配列の末尾に追加できます：</p>

  <pre><code class="language-js">myArray.push('lavender');


  let third = myArray[3]; // 'lavender'

  </code></pre>

  <p>配列についての詳細は<a
  href="https://developer.mozilla.org/en-US/docs/Learn/JavaScript/First_steps/Arrays/"
  target="_blank">MDN</a>を参照してください。</p>

    let xCoordinates = [];


    function setup() {
      createCanvas(100, 100);

      // ループを使用して配列に要素を追加する
      for (let x = 20; x < 100; x += 20) {

      }

      describe('グレーの背景に水平に並んだ4つの白い円。');
    }


    function draw() {
      background(200);

      // インデックスiの要素にアクセスし、それを使って円を描く

        circle(xCoordinates[i], 50, 20);

    let xCoordinates = [];

        xCoordinates[i] += random(-1, 1);

        // インデックスiの要素を使って円を描く
