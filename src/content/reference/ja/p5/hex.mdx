description: >-
  <p>数値をそれに相当する16進数表記の文字列に変換します。</p>

  <p>2つ目のパラメーターが渡された場合、16進数表記で生成する文字数を設定するために使用されます。配列が渡された場合、同じ長さの16進数表記の文字列の配列が返されます。</p>


  <hr/>

  <p>Converts a <code>Number</code> to a <code>String</code> with its
  hexadecimal value.</p>

  <p><code>hex()</code> converts a number to a string with its hexadecimal
  number value.

  Hexadecimal (hex) numbers are base-16, which means there are 16 unique

  digits. Hex extends the numbers 0–9 with the letters A–F. For example, the

  number <code>11</code> (eleven) in base-10 is written as the letter
  <code>B</code> in hex.</p>

  <p>The first parameter, <code>n</code>, is the number to convert. For example,
  <code>hex(20)</code>,

  returns the string <code>'00000014'</code>. If an array is passed, as in

  <code>hex([1, 10, 100])</code>, an array of hexadecimal strings is
  returned.</p>

  <p>The second parameter, <code>digits</code>, is optional. If a number is
  passed, as in

  <code>hex(20, 2)</code>, it sets the number of hexadecimal digits to display.
  For

  example, calling <code>hex(20, 2)</code> returns the string
  <code>'14'</code>.</p>

      for (let i = 0; i < converted.length; i += 1) {

        // Calculate the y-coordinate.
        let y = (i + 1) * 25;

        // Display the original and converted values.
        text(`${ original[i]} = ${converted[i]}`, 75, y);
