description: >
  <p><code>Number</code>を16進数値の<code>String</code>に変換します。</p>

  <p><code>hex()</code>は数値を16進数値の文字列に変換します。

  16進数（hex）は基数16の数字系で、16種類の一意の数字があります。16進数は0〜9の数字をA〜Fの文字で拡張します。例えば、

  10進数の<code>11</code>（11）は16進数では文字<code>B</code>と表記されます。</p>

  <p>最初のパラメータ<code>n</code>は変換する数値です。例えば、<code>hex(20)</code>は

  文字列<code>'00000014'</code>を返します。配列が渡された場合、例えば

  <code>hex([1, 10, 100])</code>のように、16進数文字列の配列が返されます。</p>

  <p>2番目のパラメータ<code>digits</code>はオプションです。数値が渡された場合、

  <code>hex(20, 2)</code>のように、表示する16進数の桁数を設定します。

  例えば、<code>hex(20, 2)</code>を呼び出すと、文字列<code>'14'</code>が返されます。</p>

    function setup() {
      createCanvas(100, 100);

      background(200);

      // 数値の配列を作成します。
      let original = [1, 10, 100];

      // 数値を16進数文字列に変換します。
      // 16進数の2桁のみ使用します。
      let converted = hex(original, 2);

      // テキストのスタイルを設定します。
      textAlign(RIGHT, CENTER);
      textSize(16);

      // 変換された値を繰り返し処理します。
      for (let i = 0; i < converted.length; i += 1) {

        // y座標を計算します。
        let y = (i + 1) * 25;

        // 元の値と変換後の値を表示します。
        text(`${ original[i]} = ${converted[i]}`, 75, y);
      }

      describe(
        '3行に分けて "1 = 01"、"10 = 0A"、"100 = 64" と書かれたテキスト。テキストは灰色の背景に黒で書かれています。'
      );
    }
    </code>
