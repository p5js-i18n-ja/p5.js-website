description: >
  <p>カメラの位置と向きを、他の2つのカメラの中間の値に設定します。</p>

  <p><code>myCamera.slerp()</code>は球面線形補間を使用して、2つの他のカメラの中間の

  位置と向きを計算します。これは、2つの視点間をスムーズに

  遷移させるのに役立ちます。</p>

  <p>最初の2つのパラメータ、<code>cam0</code>と<code>cam1</code>は、現在のカメラの設定に

  使用される<code>p5.Camera</code>オブジェクトです。</p>

  <p>3番目のパラメータ<code>amt</code>は、<code>cam0</code>と<code>cam1</code>の間で補間する

  量です。0.0はカメラの位置と向きを<code>cam0</code>と同じに保ち、

  0.5は<code>cam0</code>と<code>cam1</code>の中間に設定し、1.0は

  位置と向きを<code>cam1</code>と同じに設定します。</p>

  <p>例えば、<code>myCamera.slerp(cam0, cam1, 0.1)</code>を呼び出すと、カメラの位置

  と向きが<code>cam0</code>に非常に近くなります。

  <code>myCamera.slerp(cam0, cam1, 0.9)</code>を呼び出すと、カメラの位置と向きが

  <code>cam1</code>に非常に近くなります。</p>

  <p>注意：すべてのカメラは同じ投影を使用する必要があります。</p>

    let cam;
    let cam0;
    let cam1;

    function setup() {
      createCanvas(100, 100, WEBGL);

      // メインカメラを作成します。
      // デフォルト設定のままにします。
      cam = createCamera();

      // 最初のカメラを作成します。
      // デフォルト設定のままにします。
      cam0 = createCamera();

      // 2番目のカメラを作成します。
      cam1 = createCamera();

      // 右上に配置します。
      cam1.setPosition(400, -400, 800);

      // 原点を向くようにします。
      cam1.lookAt(0, 0, 0);

      // 現在のカメラをcamに設定します。
      setCamera(cam);

      describe('灰色の背景に描かれた白い立方体。カメラがゆっくりと正面図と上空からの図の間を行き来します。');
    }

    function draw() {
      background(200);

      // cam0とcam1の間で補間する量を計算します。
      let amt = 0.5 * sin(frameCount * 0.01) + 0.5;

      // メインカメラの位置と向きを更新します。
      cam.slerp(cam0, cam1, amt);

      box();
    }
